パネル A のモデルでは $\bar{\varepsilon}(\hat{\beta}_p)$ の値が小さいが、その諸統計量は、他のモデルの値とほぼ同一である。$\bar{\varepsilon}(\bar{\gamma}_5 - \bar{R}_{ft})$ の値は他のモデルのものよりかなり小さい。直近の期間 1961-6/68 を除き、パネル A の $\bar{\gamma}_5 - \bar{R}_{ft}$ の値はすべて正であり、概して月あたり 0.4 パーセントを上回る。期間全体 1935-6/68 についての $\bar{\varepsilon}(\bar{\gamma}_5 - \bar{R}_{ft})$ の値は有意であり、各サブ期間についても同様に大きい。

各標本期間について、表3の列3を用いて S-L 仮説を検定できる。

ある標本期間内では、リスクとリターンの確率過程の等価な2つの表現として式 (7) と次式 (13) を用いることができる。

$$\tilde{R}_{it} = \tilde{\gamma}'_{0t} + \tilde{\gamma}'_{1t}\beta_i + \tilde{\gamma}_{2t}(1 - \beta_i)^2 + \tilde{\gamma}_{3t}s_i + \tilde{\eta}_{it}. \tag{13}$$

ここで $\tilde{\gamma}_{1t} = \tilde{\gamma}'_{1t} - 2\tilde{\gamma}_{2t}$、$\tilde{\gamma}_{0t} = \tilde{\gamma}'_{0t} + \tilde{\gamma}_{2t}$ である。式 (13) は単に式 (7) を書き換えたものにすぎないので、$\tilde{\gamma}_{jt}$（$j=0,1,2,3$）の標本平均は、$\tilde{\gamma}_{jt}$ の標本平均とまったく同じ情報をもたらす。

各標本期間における月次のリターン分散をより詳しく説明するために、式 (14) を用いる。

$$R_{pt} = \hat{\gamma}'_{0t} + \hat{\gamma}'_{1t}\hat{\beta}_p + \hat{\gamma}_{2t}(1 - \hat{\beta}_p)^2 + \hat{\gamma}_{3t}\bar{s}_p(\hat{\varepsilon}_i) + \hat{\eta}_{pt}, \tag{14}$$

ここで $\bar{s}_p(\hat{\varepsilon}_i)$ は $\hat{\varepsilon}_i$ の平均である。式 (7) と (14) の検定（いずれも同一に分布する誤差の検定）の結果は、$E(\tilde{\gamma}_{2t}) = 0$ という仮説と整合的である。
