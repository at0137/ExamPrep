リスク・リターン回帰の結果は、効率的市場仮説、すなわち、価格が任意の時点で利用可能な情報を完全に反映している市場という仮説と整合的である。

## 付録（Appendix）

### いくつかの関連する論点（Some Related Issues）

#### A1. 市場モデルと市場効率性の検定（Market Models and Tests of Market Efficiency）

時系列の回帰条件式 (10) は、もちろん、本稿の各種検定の前提となっている。しかしこれらの係数は、ある証券リターンの確率過程に関する一連の仮定にもとづいて導かれたものでもある。ここで Fama et al. (1969) の表記を用いると、これら2変数の確率過程が「市場モデル」（market model）と呼ばれてきたものである。すなわち次式で表される。

$$R_{it} = \hat{a}_i + \hat{\beta}_i R_{mt} + \hat{\varepsilon}_{it}. \tag{15}$$

この回帰式では、項 $\tilde{\varepsilon}_{it}$ は、証券 $i$ のリターンのうち、市場ファクター $\tilde{R}_{Mt}$ の変動によって説明される部分を取り除いた残差を表す。これらの係数が、(10) の時系列回帰で得られた値と整合的であるなら、(15) の各係数 $\tilde{\beta}_i$ は前期と同じ値をとり、効率性を示す結果となる。

しかし、$\tilde{\varepsilon}_{it}$ の期間ごとの変動に独立性があるか否か（$\tilde{\beta}_i$ と $\tilde{\beta}_j$ や、(15) における $\tilde{\alpha}_i$ と $\tilde{\alpha}_j$ のあいだの独立性）という点については、より精緻な仮定が必要となる。なぜなら、効率的な市場では同時点の独立性が成り立つはずだからである。すなわち時系列に沿った独立性の検定（$\tilde{\beta}_i$ や $\tilde{\beta}_j$）は、市場の効率性の検定でもある。

$$R_{pt} = \hat{\gamma}_{0t}(1 - \hat{\beta}_p) + \hat{\beta}_p R_{mt} + \hat{\eta}_{pt}. \tag{16}$$

ここで $\tilde{\beta}_p$ と $\tilde{\varepsilon}_{pt}$ は、それぞれ $\tilde{\beta}_i$ と $\tilde{\varepsilon}_{it}$ のポートフォリオ平均である。

$$R_{pt} = \hat{\gamma}_{0t}(1 - \hat{\beta}_p) + \hat{\beta}_p R_{mt} + \hat{\gamma}_{2t}(\hat{\beta}_p^2 - \hat{\beta}_p\bar{\bar{\hat{\beta}}}^2) + \hat{\gamma}_{3t}[\bar{s}_p(\hat{\varepsilon}_i) - \hat{\beta}_p\bar{\bar{s}}(\hat{\varepsilon}_i)] + \hat{\eta}_{pt}, \tag{17}$$

ここで $\bar{\bar{\hat{\beta}}}^2$ と $\bar{\bar{s}}(\hat{\varepsilon}_i)$ は、それぞれ $\hat{\beta}_p^2$ と $\bar{s}_p(\hat{\varepsilon}_i)$ の $p$ に関する平均である。

Ball, Jaffe, and Mandelker は、(15) の二要素ではなく一要素の市場モデルを用いており、ここではこれを使用していない。最初は、$\tilde{\beta}_i$ と $\tilde{\beta}_j$ が（(15) において）独立であるか否かを問題にした。Mandelker (1972) は、(15) の二要素モデルが、(15) における残差リターンの確率過程についてより精緻な仮定を必要とすることを示した。
