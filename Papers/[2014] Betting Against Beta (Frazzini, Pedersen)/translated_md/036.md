次に、制約付きエージェントの要求収益率が $\psi$ とともに減少することを示す。

$$
\frac{\partial}{\partial \psi} \sum_{i:\, \text{constrained}} P_t^s x^s = \sum_{i:\, \text{constrained}} \left(\frac{\partial P_t^s}{\partial \psi} x^s + P_t^s \frac{\partial x^s}{\partial \psi}\right) < 0
\tag{A8}
$$

$\dfrac{\partial P_t^s}{\partial \psi} x^s < 0$ がすべての証券について成り立つことを示すために、まず次の式を用いる。

$$
\sum_{i:\, \text{constrained}} P_t^s \frac{\partial}{\partial \psi} x^s = \sum_{i:\, \text{constrained}} \left(\frac{E_t\left(P_{t+1}\right) - \gamma \Sigma x^*}{1 + r^f + \psi}\right) \frac{\partial}{\partial \psi} \Sigma^{-1} \, \Omega^{-1}\left[E_t\left(P_{t+1}\right) - \left(1 + r^f + \psi\right) \frac{E_t\left(P_{t+1}\right) - \gamma \Sigma x^*}{1 + r^f + \psi}\right]
$$

$$
= -\left(\frac{E_t\left(P_{t+1}\right) - \gamma \Sigma x^*}{1 + r^f + \psi}\right) \frac{\partial}{\partial \psi} \Omega^{-1} \sum_{i:\, \text{constrained}} \gamma^i \left(1 + r^f + \psi\right) \frac{E_t\left(P_{t+1}\right) - \gamma \Sigma x^*}{1 + r^f + \psi}
$$

$$
= -\left(\frac{E_t\left(P_{t+1}\right) - \gamma \Sigma x^*}{1 + r^f + \psi}\right) \frac{\partial}{\partial \psi} \Omega^{-1} \frac{1}{\gamma}\left[\gamma\left(1 + r^f\right) + \psi\right] \frac{E_t\left(P_{t+1}\right) - \gamma \Sigma x^*}{1 + r^f + \psi}
$$

$$
= -\frac{1}{1 + r^f + \psi} \frac{1}{\gamma} \frac{\partial}{\partial \psi} \frac{q\left(1 + r^f\right) + \psi}{1 + r^f + \psi} \left(E_t\left(P_{t+1}\right) - \gamma \Sigma x^*\right) \Omega^{-1} \left(E_t\left(P_{t+1}\right) - \gamma \Sigma x^*\right)
$$

$$
< 0
$$

ここで $q \equiv \displaystyle\sum_{i:\, \text{constrained}} \frac{\gamma}{\gamma^i} < 1$ かつ $\psi$ について $\displaystyle\sum_{i:\, \text{constrained}} \frac{\gamma}{\gamma^i} \psi^i = \sum_i \frac{\gamma}{\gamma^i} \psi^i = \psi$ と定めた。$\psi^i = 0$ は制約のないエージェントに対応する。これで証明が完了する。$\square$

$$
\frac{\partial P_t^s}{\partial \psi_t} = -\frac{1}{1 + r^f + \psi_t}
\tag{A9}
$$

これはすべての証券 $s$ について同じ値となる。
