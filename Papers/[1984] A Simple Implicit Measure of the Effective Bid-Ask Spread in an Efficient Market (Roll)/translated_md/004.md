次の表は、$\Delta p_{t-1} = -s$(時点 $t-1$ がビッド)である場合、次の価格変化 $\Delta p_t$ が新たな情報なしには生じえないことを示している。すなわち、次の価格変化はゼロ(別の取引もビッドで成立)か $+s$(次がアスク、価格上昇)のいずれかである。同様に、ある時点がアスクである確率はビッドである確率と同じである(価格下落の場合も同様)。

$\Delta p_{t-1} = -s$ がビッドである場合:

|              | $\Delta p_t = -s$ | $\Delta p_t = 0$ | $\Delta p_t = +s$ |
|--------------|:----:|:----:|:----:|
| $\Delta p_{t+1} = -s$ |  0   | 1/4  | 1/4  |
| $\Delta p_{t+1} = 0$  | 1/4  |  0   |  0   |
| $\Delta p_{t+1} = +s$ | 1/4  |  0   |  0   |

$\Delta p_{t-1} = +s$ がアスクである場合:

|              | $\Delta p_t = -s$ | $\Delta p_t = 0$ | $\Delta p_t = +s$ |
|--------------|:----:|:----:|:----:|
| $\Delta p_{t+1} = -s$ |  0   |  0   | 1/4  |
| $\Delta p_{t+1} = 0$  |  0   |  0   | 1/4  |
| $\Delta p_{t+1} = +s$ | 1/4  | 1/4  |  0   |

連続する価格変化の共分散を計算するには、上記2つの結合確率分布を平均すればよい。すなわち、中央の行と中央の列をゼロで埋めて両者を平均する。下記の共分散はこの操作で得られる。

|              | $\Delta p_t = -s$ | $\Delta p_t = 0$ | $\Delta p_t = +s$ |
|--------------|:----:|:----:|:----:|
| $\Delta p_{t+1} = -s$ |  0   | 1/8  | 1/8  |
| $\Delta p_{t+1} = 0$  | 1/8  |  0   | 1/8  |
| $\Delta p_{t+1} = +s$ | 1/8  | 1/8  |  0   |

価格変化どうしの共分散を計算するには、上記の結合確率を用いる。すべての価格変化どうしの積を取り、列ごとに(同様に行ごとに)対称な要素が打ち消し合うことに注意すると、

$$\text{Cov}(\Delta p_t,\, \Delta p_{t+1}) = \tfrac{1}{8}(-s^2 - s^2) = -s^2/4 \tag{1}$$

この式における共分散は、1取引あたりのビッド・アスク・スプレッド $s$ の2乗のみに依存している。同様に、$\Delta p$ の分散は $s^2/2$ となり、自己相関係数は $-\tfrac{1}{2}$ となる。

このスプレッドの大きさは、実効ビッド・アスク・スプレッドを実装可能なかたちで測定するために用いることができる。なぜなら、観測される価格変化の標本共分散は、本論文の主たる関心対象である未知のスプレッドへと直接変換できるからである。価格変化の共分散はマーケットメーカーの利潤の指標と見なせるが、未観測の真の価値変化に支配される。観測される価格変化は2つの要因——真の価値変化とビッド・アスク往復——の和であるが、これらの共分散はゼロでないスプレッドが存在するときに限り非ゼロとなる。したがって、観測される価格変化の標本共分散から、未知のスプレッドが回復される。³

ここで、共分散は連続する2つの価格変化のあいだで測られる点に注意されたい。日次データではなく取引ごとのデータを用いる場合、共分散の推定はより容易で、また小標本においても誤差が小さくなりやすい。ただし、後者のデータは入手が難しいため、本論文の経験分析(第II節を参照)では日次データを用いる。
