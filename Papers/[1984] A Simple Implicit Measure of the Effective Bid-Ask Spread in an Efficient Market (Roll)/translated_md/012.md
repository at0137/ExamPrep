スプレッド $s$ における価値の変化 $t-1$ から $t$ は $\pm s/2$、スプレッド $s'$ における価値の変化 $t$ から $t+1$ は $\pm s'/2$ となる。観測される価格変化は、価値の変化（矢印で示される）に取引コストを加えたものである。ここでも、新情報に起因する価値の変化が独立であるという仮定のもとで、対称性の仮定を用いて結合確率分布を導くことができる。

$$\mathrm{Cov}(\Delta p_{t-1}^*, \, \Delta p_t) = \mathrm{Cov}(\Delta p_t^*, \, \Delta p_{t-1}) = 0$$

この場合も依然として成り立つ。さらに、観測価格変化の一次系列共分散が *観測された* 系列共分散を保証する。$\Delta p_t^*$ と $\Delta p_{t-1}$ の符号が独立であるという対称性の仮定のもとで、観測価格変化 $\Delta p_t$ の結合確率分布は次のようになる。

|  |  | $\Delta p_t$ |  |  |  |
|---|---|:---:|:---:|:---:|:---:|
|  |  | $-s_{t-1} - \Delta s/2$ | $-\Delta s/2$ | $\Delta s/2$ | $s_{t-1} + \Delta s/2$ |
|  | $-s_t$ | 0 | 0 | 1/8 | 1/8 |
| $\Delta p_{t+1}$ | $0$ | 1/8 | 1/8 | 1/8 | 1/8 |
|  | $s_t$ | 1/8 | 1/8 | 0 | 0 |

ここで $\Delta s = s_t - s_{t-1}$ である。しかし、この結合確率分布から、$\mathrm{Cov}(\Delta p_t, \Delta p_{t+1}) = -s_t^2/4$ であることが直接的に検証できる。

驚くべきことに、スプレッドが情報によって変動するという唯一の事実が、本文で導いた公式のもとで成立し、提示スプレッドの分散が拡散していてもなお、提示スプレッドの平均のもとで導かれた仮定のもとで成立する。観測された系列共分散はやはり、スプレッドの公式に関連して、同じ単純な公式で表される。

$t-1$ から $t$、$t$ から $t+1$ について計算されるサンプル・クロス積については、次式が成り立つ。

$$E[\Delta p_t, \, \Delta p_{t+1}] = -s_t^2/4.$$

したがって、サンプル全体についての共分散の期待値は次式となる。

$$E\!\left[\sum_t (\Delta p_t, \Delta p_{t+1}) / \tau\right] = -\bar{s}^2/4, \qquad t = 1, \cdots, \tau,$$

ここで $\bar{s}^2$ はサンプル期間中の二乗スプレッドの平均である。したがって、この公式は任意の変動するスプレッドにも適用できる。例えば、提示スプレッドの非定常な分散を、リターンの非定常な共分散として扱ったとしても、本文の公式は依然として有効である。実際には、リターンの非定常性、すなわちスプレッドの公式が依然として有効であるような単純な公式の例として有効である。
