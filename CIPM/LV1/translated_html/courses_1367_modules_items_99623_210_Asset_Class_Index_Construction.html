2.10 | Asset Class Index Construction

モジュール2：ベンチマーク入門（Introduction to Benchmarks）2.10｜資産クラス・インデックスの構築（Asset Class Index Construction）

学習目標（Learning Outcomes）

The candidate should be able to:

  • 時価総額加重（market-capitalization-weighting）、等加重（equal-weighting）、株価加重（price-weighting）、およびファンダメンタル加重（fundamental-weighting）のインデックス構築手法を、それぞれの利点および欠点を含めて比較すること；
  • 時価総額インデックスにおける浮動株調整（float adjustment）の目的と影響を説明すること；

インデックスは、まず一連のルールを定義し、それを既存の証券集合に適用することで作成される。ただし、すべてのインデックスが同じ方法で構築されるわけではない。インデックス構築には主に3つの選択がある：組入基準（inclusion criteria）、証券の加重方法（security weighting）、およびインデックスの維持管理（index maintenance）である。さらに、インデックス提供者はインデックス値を算出し、そこからリターンが計算される。

第一の選択である組入基準は、インデックスがどの証券集団を代表するかを決定する。証券数が多く、かつ業種や規模で分散されているほど、インデックスは広範な市場パフォーマンスをより適切に測定できる。一方、より狭いユニバースは特定の証券群のパフォーマンス測定に適している。ベンチマークとして機能するためには、インデックスの組入基準はマネジャーのポートフォリオと類似した構成となるべきである。

第二の選択である証券の加重は、通常、株価加重（price weighting）、時価総額加重（value weighting）、または代替加重（alternative weighting）のいずれかから選択される。一部のインデックスでは特定の証券により大きなウェイトを与えるため、リターンとリスクが異なる。スポンサーは、マネジャーのポートフォリオとインデックスを比較する前にこれらの違いを理解しておく必要がある。インデックスの維持およびリターン計算のルールも、ベンチマークとしての適合性に影響を与える。

各インデックス提供者によって詳細は異なるものの、資産クラス・インデックスの構築における一般的なアプローチは、以下のステップに関するルールを定めることである：

適格証券の定義（Define eligible securities）

まず、特定の国におけるすべての普通株（common equity shares）など、出発点となる証券ユニバースを特定する必要がある。通常、インデックスの投資可能性（investability）を高めるために、さまざまな適格性ルール（eligibility rules）が適用される。代表的なものは以下のとおりである：

  • 取引要件（Trading requirements）：適格証券は主要な取引所に上場されていなければならない（例えば店頭取引（over-the-counter）銘柄は対象外）。
  • 最低取引価格（Minimum trading price）：株式は一定の最低価格以上で取引されていなければならない。
  • 最低浮動株数（Minimum available shares（float））：一般投資家に流通している株式が少ない企業は対象外となる。
  • 最低流動性（Minimum liquidity）：一定期間における取引量または取引金額が一定水準以上である必要がある。
  • 企業構造（Company structure）：有限責任組合（limited partnerships）、有限責任会社（limited liability companies）、クローズドエンド型投資会社（closed-end investment companies）などの構造を持つ企業は対象外となる。
  • 株式の種類（Share types）：優先株（preferred stock）や転換優先株（convertible preferred stock）などは普通株インデックスから除外される。
  • 国の割当（Country assignment）：証券がどの国に属するかは、設立国だけでなく他の基準にも依存する場合がある。企業は運営、税制、政治的、またはその他の財務上の理由で特定の地域に登記することがある。例えば、本社所在地や主要な取引市場などが考慮される。

組入基準は証券数を管理可能な規模に減らし、通常はインデックスの投資可能性を向上させるために設定される。インデックス提供者は、すべての適格証券を含めるか、さらに絞り込むかも決定する。銘柄数に制限を設ける場合、市場を適切に代表するよう、セクターや業種の代表性を確保するルールなどが設けられる。

適格性ルールの範囲や詳細はインデックス提供者ごとに異なる。一部の提供者は委員会によって銘柄選定を行う。多くの場合、インデックス構成銘柄は対象市場全体のサンプルとみなされるべきであり、そのサンプルが基礎市場を適切に代表していることを確保することが目標である。

多くのインデックス提供者は、企業の時価総額（market capitalizations）に基づき、ほぼすべてのサイズ帯をカバーする包括的なインデックス・ファミリーを構築している。大型株（large-cap）、中型株（mid-cap）、小型株（small-cap）などを含む複数のインデックスで構成される。適格証券を定義した後、各証券をこれらのサイズ区分に割り当てるルールが適用される。

インデックス加重の定義（Define index weighting）

適格証券が決定された後、それらに対する加重ルールが適用される。証券の加重はインデックスのリスクとリターンに直接影響し、またマネジャーのベンチマークとしての妥当性にも影響するため重要である。

一般的に用いられる加重方式には以下がある：

  • 時価総額加重（Capitalization weighting）（market value weighting、market cap weighting、cap weightingとも呼ばれる）。最も一般的な方式であり、各構成銘柄は価格×浮動株数により算出される時価総額に比例して保有される。価値加重インデックスのパフォーマンスは、各銘柄の発行済み価値全体を保有するポートフォリオのパフォーマンスを表す。この方式は投資専門家に最も広く受け入れられている。
  • 株価加重（Price weighting）：この方式では構成銘柄は株価に比例して加重される。インデックス値は単純に株価の平均として解釈できる。株価加重インデックスのパフォーマンスは、各銘柄を1単位ずつ保有するポートフォリオのパフォーマンスを表す。単純であることが利点だが、投資家の実際のポートフォリオ構成方法との関連性は低い。代表例はDow Jones Averagesであり、長い歴史と広い認知を持つ。日経平均株価（Nikkei Stock Average）も株価加重インデックスである。
  • 等加重（Equal weighting）：純粋な等加重では、すべての構成銘柄が指定されたリバランス時点で同一ウェイトとなる。このインデックスのパフォーマンスは、各銘柄に同額を投資するポートフォリオのパフォーマンスを表す。セクターや業種単位で均等加重するなどのバリエーションもある。個別銘柄のリターン差によりウェイトが乖離するため、定期的（例：四半期ごと）なリバランスが必要である。Value Line Composite Averageは米国株の等加重インデックスである。
  • ファンダメンタル加重（Fundamental weighting）：Research Affiliates LLCが特許を取得した手法であり、売上高（sales）、キャッシュフロー（cash flow）、簿価（book value）、配当（dividends）など、市場価格以外の企業特性に基づいて加重する。評価に重要と考えられる変数に基づいてウェイトを決定することで、市場価格ではなく「本源的価値（true values）」に基づく加重を目指す。他の組織も類似のアプローチを採用し、さまざまな変数の組み合わせを用いたインデックスを開発している。このインデックスのパフォーマンスは、評価指標に基づいて投資するポートフォリオのパフォーマンスを表す。
  • 最適化ベースの加重（optimization-based weighting）（最小分散（minimum-variance）および効率性加重（efficiency weighting））。低ボラティリティ・ポートフォリオ（low-volatility portfolios）と比較して、異常なリスク調整後リターン（abnormal risk-adjusted returns）の存在を示すいくつかの証拠がある。最小分散ポートフォリオ（minimum-variance portfolios）は、この見かけ上の異常を追求するための自然な出発点である。他にも、シャープレシオ（Sharpe ratio）（リスク単位当たりの超過リターン（excess return per unit of risk））を最大化する方法や、すべての資産間でリスクを均等化する方法（リスク・パリティ（risk parity）として知られる）がある。これらのタイプのベンチマークは、インデックス設計と投資戦略の区別を曖昧にし、実質的に投資戦略がインデックス設計に組み込まれている。

これらのインデックスの相対的な利点および欠点については、「インデックス構築のトレードオフ（Index Construction Tradeoffs）」のレッスンで議論する。

ほとんどすべての時価総額加重インデックスは、フリーフロート（free float）で調整されている。フリーフロートとは、企業の発行済株式のうち一般投資家が取引可能な株式数である。この調整は、広く購入可能ではなく投資可能な機会集合（investable opportunity set）の一部ではない企業の時価総額部分を除外することを目的としている。この結果得られるインデックスは、フリーフロート調整時価総額インデックス（free-float-adjusted market capitalization index）、または単にフロート加重インデックス（float-weighted index）と呼ばれる。フロート加重インデックスは、取引可能なすべてのインデックス構成銘柄を保有するポートフォリオのパフォーマンスを表す。

除外される株式の例は以下のとおりである：

  • クロス保有（Cross ownership）：同じインデックスに含まれる他の企業が保有する株式。
  • 大口の企業または個人保有（Large corporate or private holdings）：例えば、発行済株式の10%を超える株式を保有する企業や個人の持分。
  • 従業員保有株式（Employee-owned shares）：例えば、従業員持株制度（employee stock ownership plans）によって保有される株式。
  • IPOロックアップ（IPO lockups）：新規公開株式（initial public offering）において一定期間売却が制限される株式。
  • 政府保有（Government holdings）：例えば、「政府（government of）」として記載される保有分は取引不能とみなされる。

フリーフロート調整は、特に政府、創業家、他企業などが大きな株式割合を保有することが多い発展途上市場（less developed markets）において重要である。これらの株式は理論的には購入可能であっても、実際には高いプレミアムが必要となり、多くの投資家にとって魅力的ではない。このような実質的に購入不可能な株式を除外することで、インデックスはより容易に複製可能となる。もし取引不可能な株式がインデックスに含まれ、多くの投資家がそれを複製しようとした場合、需給の不均衡により大きな価格歪みが生じる可能性がある。すべての主要なグローバル・インデックス（S&P Dow Jones、FTSE、MSCIなど）はフロート調整を行っている。フロート調整によりインデックスの投資可能性は向上するが、その調整は主観的であり、インデックス提供者によって判断が異なる場合がある。

インデックス維持（Index maintenance）

インデックスを継続的に維持するためには、さまざまなルールが必要である。例は以下のとおり：

  • 再構成およびリバランス間隔（Reconstitution and rebalancing intervals）：市場は常に変化しており、企業は合併（mergers）、スピンオフ（spinoffs）、IPO、破産（bankruptcies）、上場廃止（delistings）などによって出入りする。また、小型企業が大型化したり、大型企業が縮小したりする。したがって、インデックスが意図された設計を維持するためには、構成銘柄やウェイトの見直しが必要となる。多くの提供者は年次または四半期などの定期的な再構成を行う。非時価総額加重インデックスでは、加重を維持するためにリバランスが不可欠である。提供者は不要なターンオーバーを避けるため、銘柄が頻繁に入れ替わらないよう慎重にルールを設計する。大規模IPOをいつ、あるいは含めるかどうかの判断は重要な意思決定となる。
  • コーポレート・アクション（Corporate actions）：自社株買い（buybacks）、追加発行（secondary offerings）、スピンオフ（spinoffs）、株式分配（stock distributions）などにより発行済株式数は変化する。インデックス提供者は通常、これらの変化を発生時に定められたルールに従って処理する。
  • 配当（Dividends）：配当は通常、後述するインデックス値の計算に反映される。配当を含むインデックスはトータルリターン・インデックス（total return index）と呼ばれ、配当を含まないものはプライスリターン・インデックス（price return index）と呼ばれる。
  • 取引停止（Suspensions）：取引停止銘柄の価格付けに関するルールを定める必要がある。
