モジュール1：投資パフォーマンス評価（Investment Performance Appraisal）1.11｜アルファ（Alpha）

学習目標（Learning Outcomes）

The candidate should be able to:

  • シャープレシオ（Sharpe ratio）、M²（M-squared）、トレイナーレシオ（Treynor ratio）、ジェンセンのアルファ（Jensen’s alpha）、アルファ（alpha）、インフォメーションレシオ（information ratio）、およびアプレイザルレシオ（appraisal ratio）を計算し、解釈し、比較せよ
  • パフォーマンス評価指標の適切な使用方法を特定し、その妥当性を説明せよ
  • パフォーマンス評価指標を計算し、解釈し、比較せよ
  • 非対称なリターン分布（non-symmetrical return distributions）が評価指標にどのように影響するかを説明せよ

アルファ（alpha）は、ポートフォリオまたは証券のリターンと、そのリスクを踏まえて市場で要求されるリターンとの差を測定する指標である。アルファはしばしば異常リターン（abnormal return）とも呼ばれる。これは期待ベース（ex ante）でも実現ベース（ex post）でも推定できる。

資産の期待リターンと必要リターンとの差は、その資産の期待アルファ（expected alpha）または事前アルファ（ex ante alpha）であり、式11（Equation 11）に示される。

Expected alpha=Expected return−Required return11

ファイナンスにおける根本的な問題の一つは、リスク資産の必要リターンを事前に決定することである。一般的かつ概念的に魅力的な方法は、CAPM（capital asset pricing model、1960年代に開発）、ファマ・フレンチの3ファクターモデル（Fama–French three-factor model、1993年公表）、またはカーハートの4ファクターモデル（Carhart four-factor model、1997年公表）などの資産価格モデルを仮定することである。しかし、これらのモデルは理論的・直感的には魅力的であるものの、実証研究ではいずれも証券リターンを完全に説明できていないことが示されている。したがって、ポートフォリオマネージャーは通常、期待アルファに基づいて証券選択を行うが、実際のリターンが明らかになるまでは意思決定能力を評価することは非常に困難である。

投資戦略の実際の成果を評価するために、アナリストは一定の保有期間における実現アルファ（realized alpha）または事後アルファ（ex post alpha）を検討する（式12）。同時点の必要リターン（contemporaneous required return）とは、保有期間中に同程度のリスクを持つ資産が実際に得たリターンである。

Realized alpha=Actual holding-period return− Contemporaneous required return12

「アルファ」という用語は、多くの実務家によって厳密ではない形で「アクティブリターン（active return）」を指すために用いられることが多い点に注意が必要である。アクティブリターンとはベンチマークリターンを上回るリターンである。この見方では、ベンチマークポートフォリオのリターンが式12における同時点の必要リターンとみなされる。

ベンチマークがアクティブ運用に対する「パッシブ」な投資代替手段を表すと仮定すると、ベンチマークのリターンはアクティブ運用ポートフォリオのパフォーマンスを比較するための妥当な基準となる。しかし、アクティブリターンは一般にアルファの近似に過ぎない。両者が厳密に一致するのは、ポートフォリオのリスクがベンチマークと完全に一致している場合（すなわちベータが1.0である場合）のみである。この仮定が成立しない場合、長期的にシステマティック・リスクに対する報酬が存在するならば、アウトパフォーマンスやアンダーパフォーマンスはベンチマークに対する系統的なリスクの偏りによって説明される可能性がある。例えば、価格上昇局面では、ベンチマークより高いシステマティック・リスクを持つようにポートフォリオを傾けることでアウトパフォーマンスが生じる可能性がある。同様に、価格上昇局面でベンチマークより低いシステマティック・リスクを持つ場合、アンダーパフォーマンスが生じる可能性がある。

しかし、ポートフォリオのシステマティック・リスクをベンチマークから逸脱させること自体がマネージャーの能力不足を意味するわけではない。マーケットタイミング型ファンドは市場見通しの変化に応じてシステマティック・リスクのエクスポージャーを頻繁に変更する。このような調整を成功裏に行う能力もマネージャーのスキルの一部である。したがって、アクティブリターンは、主として銘柄選択能力に基づくスキルを評価する場合で、かつポートフォリオがベンチマークとリスクが一致している場合に最も適している。
