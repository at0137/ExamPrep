1.08 | M2: Risk-Adjusted Performance (RAP)

モジュール1：投資パフォーマンス評価（Investment Performance Appraisal）1.08｜M²：リスク調整後パフォーマンス（Risk-Adjusted Performance, RAP）

学習到達目標（Learning Outcomes）

The candidate should be able to:

  • シャープレシオ（Sharpe ratio）、M²、トレイナーレシオ（Treynor ratio）、ジェンセンのアルファ（Jensen’s alpha）、アルファ（alpha）、インフォメーションレシオ（information ratio）、およびアプレイザルレシオ（appraisal ratio）を計算し、解釈し、比較する
  • パフォーマンス評価指標の適切な利用方法を特定し、その妥当性を説明する
  • パフォーマンス評価指標を計算し、解釈し、比較する
  • 非対称なリターン分布が評価指標にどのように影響するかを説明する

M²は、あるベンチマークに対するポートフォリオの総リスクに基づいて調整されたポートフォリオリターンの指標を提供する。1997年にノーベル賞受賞者フランコ・モディリアーニ（Franco Modigliani）とその孫娘レア・モディリアーニ（Leah Modigliani）は、リスク調整後パフォーマンス指標（risk-adjusted performance measure, RAP）と呼ばれるものを開発した。このRAPは、その後モディリアーニの名にちなんでM²として広く知られるようになった（Modigliani and Modigliani 1997）。これはシャープレシオと関連しており、ポートフォリオの順位付けは同一となるが、ベンチマークと同一の総リスクを仮定した場合のリターンの優位性を、なじみのあるパーセンテージで表せるという利点を有する。

M²は資本市場理論（capital market theory）に基づき、ポートフォリオがレバレッジまたはデレバレッジされ、そのボラティリティ（標準偏差で測定）がベンチマークと一致するまで調整されると仮定する。この調整により、ポートフォリオのリスクをベンチマークと等しくするポートフォリオ固有のレバレッジ比率が得られる。その後、ポートフォリオの超過リターンにこのレバレッジ比率を掛け、無リスク金利を加えた値をベンチマークの実際のリターンと比較することで、リスク調整後ベースでアウトパフォームしているかアンダーパフォームしているかを判断する。

Equation 3 provides a formula for M2, where σB is the standard deviation of the benchmark portfolio and σB/σp is the portfolio-specific leverage ratio. Because the Sharpe ratio is defined as E[rp]−rFσp, Equation 3 also shows that M2 can be thought of as a rescaling of the Sharpe ratio that allows for easier comparisons among different funds and benchmarks. The reason that M2 and Sharpe ratios rank portfolios identically is because, in a given time period—and for any given comparison benchmark B—both the risk-free rate and the benchmark volatility are constant across all comparisons. Only the Sharpe ratio differs, so it determines all rankings. Based on sample estimates, Equation 3 can be expressed as Equation 4.

M2=(E[rp]−rF)σBσp+rF=SR×σB+rF3
M2^=(r¯p−r¯F)σ^Bσ^p+rF=SR^×σ^B+rF4

For example, assume that r¯F = 4.0%, r¯p = 14.0%, σp = 25.0% and σB = 20.0%. The Sharpe ratio is 0.4, SR^=0.14−0.040.25=0.4, and M2^ is 12.0%, M2^ = 0.4(0.2) + 0.04 = 0.12 = 12.0%. If the return on the benchmark was 10%, then the fund outperformed the benchmark on a risk-adjusted basis by 12.0% − 10.0% = 2.0%. This difference between the risk-adjusted performance of the portfolio and the performance of the benchmark is frequently referred to as M2 alpha.

The Sharpe ratio of the benchmark is SR^=0.10−0.040.20=0.3. Comparing the Sharpe ratio of the portfolio with the Sharpe ratio of the benchmark shows that the fund outperformed the benchmark. But the 2.0% difference between M2 and the benchmark’s return tells us the risk-adjusted outperformance as a percentage return.

Exhibit 9はM²パフォーマンス指標を示し、ポートフォリオZをベンチマークであるポートフォリオBと比較している。無リスク金利からポートフォリオの点を通って伸びる直線の傾きは、そのポートフォリオのシャープレシオである。ポートフォリオZのシャープレシオにポートフォリオBの標準偏差を掛けることで、ポートフォリオZのリスクをベンチマークと一致させるレバレッジ調整が得られる。さらに、ポートフォリオZのシャープレシオとポートフォリオBの標準偏差の積に無リスク金利を加えることでM²が得られる。これは、ポートフォリオZがベンチマークと同じ総リスクを持っていた場合に達成していたであろう理論上のリターンである。このリターンとポートフォリオBの実際のリターンとの差が、ポートフォリオZのM²アルファであり、Exhibit 9ではα_Zとして示されている。
