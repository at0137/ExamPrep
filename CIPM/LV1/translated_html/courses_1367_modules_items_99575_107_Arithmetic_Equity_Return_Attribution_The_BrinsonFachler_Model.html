1.07 | Arithmetic Equity Return Attribution: The Brinson–Fachler Model

モジュール1：リターン・アトリビューション（Return Attribution）1.07 | 算術的株式リターン・アトリビューション：ブリンソン＝ファクラー・モデル（Arithmetic Equity Return Attribution: The Brinson–Fachler Model）

 Learning Outcomes

The candidate should be able to:

  • ブリンソン＝フッド＝ビーボワー・モデル（Brinson–Hood–Beebower model）およびブリンソン＝ファクラー・モデル（Brinson–Fachler model）を用いて、ポートフォリオのパフォーマンスの源泉を分析する
  • 算術的アロケーション効果（arithmetic allocation effect）、セレクション効果（selection effect）、インタラクション効果（interaction effect）を計算し解釈する

ブリンソン＝ファクラー（BF）モデル（Brinson–Fachler model）は、個々のセクターのアロケーション効果の計算方法のみがBHBモデルと異なる。

BHBモデルでは、正のリターンを持つセクターにおけるすべてのオーバーウェイトは、全体のベンチマーク・リターンに関係なく正のアロケーション効果を生み出し、一方で負の市場におけるすべてのオーバーウェイトは負のアロケーション効果を生み出す。したがって、セクター i が正のリターン（Bᵢ > 0）を持つ場合、そのセクターのリターンが全体のベンチマーク・リターンより低い場合（すなわち Bᵢ < B）であっても、オーバーウェイトは正のアロケーション効果 Aᵢ ＝ ( wᵢ − Wᵢ )Bᵢ > 0 をもたらす。セクター・リターンが負（0 > Bᵢ）の場合、オーバーウェイトは負のアロケーション効果 Aᵢ ＝ ( wᵢ − Wᵢ )Bᵢ < 0 をもたらす。

明らかに、ポートフォリオ・マネージャーが全体のベンチマークをアウトパフォームしている負の市場をオーバーウェイトしている場合、その効果は正であるべきである。

ブリンソン・モデルの2つのバージョンの唯一の違いは、個々のセクターのアロケーション効果の計算である。BFモデルは、多くのポートフォリオ・マネージャーの意思決定プロセスとより整合的であり、そのためより広く用いられている。

図表3に示されたポートフォリオ・データにおけるセクター配分による算術的超過リターンへの寄与は、Bₛ − B ＝ 6.8% − 8.2% ＝ −1.4% である。修正されたBFセクター・アロケーション効果は、Aᵢ ＝ ( wᵢ − Wᵢ )( Bᵢ − B ) を用いて次のように計算される：

エネルギー（Energy） （50% − 50%）×（10% − 8.2%）＝ 0.0%
ヘルスケア（Health care） （30% − 20%）×（−2.0% − 8.2%）＝ −1.02%
金融（Financials） （20% − 30%）×（12% − 8.2%）＝ −0.38%
Total 0.0% − 1.02% − 0.38% ＝ −1.4%

ヘルスケアにおける影響ははるかに大きい。負の市場においてオーバーウェイトしていることによる−0.2%のコストに加え、ポートフォリオ・マネージャーは全体市場リターン8.2%に投資していない機会費用についても正しく評価される。この結果、追加のコスト10% × −8.2% ＝ −0.82%が生じ、合計で−1.02%となる。別の言い方をすれば、ポートフォリオは全体市場を−10.2%（−2.0% − 8.2%）下回る市場を10%オーバーウェイトしており、−1.02%の損失を生じている。

金融における影響ははるかに小さい。正の市場をアンダーウェイトしていることによる−1.2%のコストに対し、全体市場リターン8.2%に投資していない機会費用を加味すると、−10% × −8.2% ＝ 0.82%の寄与が生じ、合計で−0.38%となる。別の言い方をすれば、ポートフォリオは全体市場を3.8%（12.0% − 8.2%）上回る業種を10%アンダーウェイトしており、−0.38%の損失を生じている。予想どおり、ポートフォリオ全体レベルでは、アロケーション効果−1.4%はBHBモデルで計算されたものと同一である。
