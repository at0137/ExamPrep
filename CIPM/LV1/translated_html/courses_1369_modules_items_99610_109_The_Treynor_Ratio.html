1.09 | The Treynor Ratio

モジュール1：投資パフォーマンス評価（Investment Performance Appraisal）1.09｜トレイナーレシオ（The Treynor Ratio）

学習目標（Learning Outcomes）

The candidate should be able to:

  • シャープレシオ（Sharpe ratio）、M²（M-squared）、トレイナーレシオ（Treynor ratio）、ジェンセンのアルファ（Jensen’s alpha）、アルファ（alpha）、インフォメーションレシオ（information ratio）、およびアプレイザルレシオ（appraisal ratio）を計算し、解釈し、比較せよ
  • パフォーマンス評価指標の適切な使用方法を特定し、その妥当性を説明せよ
  • パフォーマンス評価指標を計算し、解釈し、比較せよ
  • 非対称なリターン分布（non-symmetrical return distributions）が評価指標にどのように影響するかを説明せよ

トレイナーレシオ（Treynor ratio）（Treynor 1965）は、システマティック・リスク（systematic risk）1単位あたりの超過リターン（excess return）を測定する指標である。トレイナーレシオ（TR）およびそれに続くシステマティック・リスクに基づく評価指標では、マネージャーのファンドのシステマティック・リスクを測定するための市場ベンチマークを慎重に選択する必要がある。これに対して、シャープレシオは市場ベンチマークを明示的に選択せずとも異なるファンド間で比較可能である。同様に、M²は原理的にはユーザーが選択した任意のベンチマークに対して計算することができる。

以下では、βpを、英国株式におけるFTSE All Share Indexのような、広範で時価総額加重の市場インデックスに対するファンドのベータ（beta）と仮定する（このような市場ベンチマークの選択は実務上一般的である）。トレイナーレシオは式5（Equation 5）で与えられ、その標本推定値は式6（Equation 6）で与えられる。

TR=E[rp]−rFβp5
TR^=r¯p−r¯Fβ^p6

したがって、ポートフォリオの期待リターンが8%、リスクフリーレートが2%、受容された市場インデックスに対するポートフォリオのベータが1.2である場合、トレイナーレシオは (8% − 2%) / 1.2 = 6% / 1.2 = 5% となり、通常は5と表記される。シャープレシオとは対照的に、トレイナーレシオはパーセンテージリターンの単位で表されるが、多くのアナリストはパーセント記号を省略する。

トレイナーレシオは、リスク1単位あたりの報酬を測定する点でシャープレシオと類似している。両者とも分子はリスクフリーレートを上回る超過リターンを測定する。両者の違いは、シャープレシオがポートフォリオの標準偏差で測定されるトータル・リスクを考慮するのに対し、トレイナーレシオはベータで測定されるシステマティック・リスクを考慮する点にある。シャープレシオと同様に、トレイナーレシオは付加価値（value added）を定量化するものではなく、単なる順位付けの指標である。

トレイナーレシオの有用性は、パフォーマンス評価においてシステマティック・リスクとトータル・リスクのどちらが適切かに依存する。ベータに基づくため、トレイナーレシオはファンド自身のボラティリティではなく、十分に分散されたポートフォリオに組み入れた際にどの程度のボラティリティをもたらすかを示す。したがって、トレイナーレシオに基づくポートフォリオの順位付けは、評価対象のポートフォリオがより広範で完全に分散されたポートフォリオの一部として組み合わされる場合に最も有用である。この指標は、同一のベンチマークインデックスと比較される場合に最も情報価値が高い。

逆に、投資家が資産の大部分を非分散的で集中したポートフォリオに投資する場合、トレイナーレシオの有用性は限定的である。超過リターンが同一であれば、システマティック・リスクが同じポートフォリオは同じトレイナーレシオを持つ。しかし、これらのポートフォリオは非システマティック・リスクに関しては異なる可能性がある。そのため、標準偏差で測定されるトータル・リスクを考慮するシャープレシオやM²の方が、単一ポートフォリオ投資家にはより適している。

図表10（Exhibit 10）はトレイナーレシオを示している。ポートフォリオUとVの2つを仮定し、それぞれの平均超過リターンとベータの組み合わせが図表10に示されている。各ポートフォリオのトレイナーレシオは、リスクフリーレートから当該ポートフォリオの点を通る直線の傾きとして表される。図表10に基づくと、ポートフォリオUはベータで測定されるシステマティック・リスク1単位あたりの超過リターンが高いため、ポートフォリオVより優れている。
