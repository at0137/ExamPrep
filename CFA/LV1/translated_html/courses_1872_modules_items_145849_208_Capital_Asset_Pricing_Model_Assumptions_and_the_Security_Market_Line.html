2.08 | Capital Asset Pricing Model: Assumptions and the Security Market Line

モジュール2：ポートフォリオのリスクとリターン：パートII 2.08 | 資本資産価格モデル（Capital Asset Pricing Model: CAPM）：仮定と証券市場線（Security Market Line: SML）

 Learning Outcome

The candidate should be able to:

  • 資本資産価格モデル（CAPM）を、その仮定および証券市場線（SML）を含めて説明せよ
  • CAPMを用いて資産の期待リターンを計算し、解釈せよ

資本資産価格モデル（CAPM）は、ポートフォリオ理論における最も重要な革新の一つである。このモデルは単純でありながら強力であり、直感的でありながら深い洞察を与える。CAPMはWilliam Sharpe、John Lintner、Jack Treynor、Jan Mossinによって独立に導入され、Harry Markowitzによる分散投資および現代ポートフォリオ理論の先行研究に基づいている。このモデルは、資産のベータに基づいて期待リターンを正確に決定する線形の期待リターン–ベータ関係を提供する。これにより、総リスクからシステマティックリスクへと焦点が移り、後者が期待リターンの主要な決定要因となる。以下の式を思い出そう：

E(Ri) = Rf + βi[E(Rm) – Rf]

CAPMは、資産の期待リターンはベータによって測定されるシステマティックリスクのみによって決まると主張する。同じベータを持つ2つの資産は、それらの性質に関係なく同じ期待リターンを持つ。リスクとリターンの関係に基づき、すべての資産はベータリスクによってのみ定義される。

本節の残りでは、CAPMを導く際に用いられる仮定と、それらがもたらす制約について検討する。次に、証券市場線（SML）を用いてCAPMを適用し、効率的・非効率的を問わずあらゆるポートフォリオや資産の価格付けを行う。最後に、投資、バリュエーション（valuation）、資本予算（capital budgeting）にCAPMをどのように適用できるかを検討する。

CAPMの仮定（Assumptions of the CAPM）

他のすべてのモデルと同様に、CAPMは金融市場の多くの複雑性を単純化のための仮定によって無視している。これらの仮定により、分析を複雑にすることなく資産価格決定の重要な洞察を得ることができる。基本的な関係が確立された後で、仮定を緩和し、洞察がどのように変化するかを検討できる。これらの仮定の一部は制約的であり、他は無害であり、またある仮定は特定の資産にのみ影響するか、仮説的関係に対してわずかな影響しか持たない。

  1. 投資家はリスク回避的で、効用最大化を目指す合理的個人である。リスク回避とは、投資家がリスクを引き受ける対価として補償を期待することを意味する。この仮定は、投資家が同じリスク回避度を持つ必要はなく、単にリスクを嫌うことのみを要求する。効用最大化とは、投資家が低いリターンではなく高いリターンを望み、常により多くの富を求めることを意味する（すなわち満足することがない）。投資家は合理的であり、利用可能な情報を正しく評価・分析して合理的な意思決定に至ると理解される。同じ情報を用いても異なる期待リスクや期待リターンの推定に至ることがあるが、投資家の同質性（仮定4参照）は投資家が合理的であることを前提とする。リスク回避および効用最大化は現実的な前提として一般に受け入れられているが、投資家の合理性については疑問も呈されている。投資家は個人的なバイアスや経験によって意思決定が歪められ、最適でない投資を行う可能性がある。しかし、そのような非合理的行動が価格に大きな影響を与えない限り（すなわち非合理的投資家の取引が互いに相殺されるか、合理的投資家の取引に支配される場合）、モデルの結果には影響しない。
  2. 市場は摩擦がなく、取引コストや税金が存在しない。この仮定により、市場の運用上の特徴から分析を抽象化できる。その結果、例えばニューヨーク証券取引所（New York Stock Exchange）の取引量や売買価格差がリスク–リターン関係に影響を与えないと仮定する。具体的には、取引コスト、税金、空売りに関するコストや制約が存在せず、無リスク利子率での借入・貸付が可能であるとする。多くの大規模機関にとって取引コストは無視できるほど小さく、多くの機関は税金を支払わない。仮にこれらが存在してもCAPMの一般的な結論には大きな影響はないが、空売りのコストや制約は資産価格に上方バイアスをもたらし得る。
  3. 投資家は同一の単一保有期間を前提として計画する。CAPMは単一期間モデルであり、すべての投資判断はその1期間に基づいて行われる。この仮定は便宜的なものであり、多期間モデルが複雑であるためである。しかし単一期間モデルでは学習が起こらず、誤った意思決定が持続する可能性がある。また、多期間の視点では単一期間では非最適に見える意思決定が必要となる場合もある。それでも、この仮定はCAPMの多期間への適用を大きく制限するものではない。
  4. 投資家は同質的な期待（homogeneous expectations）を持つ。この仮定は、すべての投資家が同じ確率分布および将来キャッシュフローの入力を用いて証券を分析することを意味する。また合理的であるため、同じ評価に到達する。その結果、同一の最適リスクポートフォリオ（市場ポートフォリオ）が形成される。この仮定は、期待の差異が最適ポートフォリオに大きな違いをもたらさない限り緩和可能である。
  5. すべての投資は無限に分割可能である。この仮定は、投資家が任意の量だけ資産に投資できることを意味する。これにより、離散的なジャンプ関数ではなく連続関数を用いた分析が可能になる。この仮定は便宜的なものであり、モデルの結論にほとんど影響を与えない。
  6. 投資家はプライステイカーである。CAPMは多数の投資家が存在し、いずれの投資家も価格に影響を与えるほど大きくないと仮定する。そのため、投資家の取引は価格に影響しないとされる。この仮定は一般に妥当であり、小型株では影響がある場合もあるが、CAPMの主要な結果には影響しない。

これらの仮定の主な目的は、合理的に平均–分散効率的（mean–variance-efficient）なポートフォリオを選択する限界投資家（marginal investor）を想定することである。市場の非効率性は運用面および情報面の両方で排除される。これらの仮定の一部は非現実的に見えるかもしれないが、多くの場合、それらを緩和してもモデルの結果にはわずかな影響しか与えない。さらに、CAPMはその限界や弱点にもかかわらず、比較や初期的なリターン推定の基準として有用である。

証券市場線（The Security Market Line）

本節ではCAPMを証券の価格付けに適用する。証券市場線（SML）は、x軸にベータ（システマティックリスク）、y軸に期待リターンを取ったCAPMのグラフ表現である。資本市場線（CML）と同様に、SMLはy軸上の無リスク利子率で始まり、その傾きは市場リスクプレミアム（R_m − R_f）である。CMLは効率的フロンティア上のポートフォリオにのみ適用されるのに対し、SMLは効率的かどうかに関係なくすべての証券に適用される。効率的ポートフォリオでは分散可能リスクが存在しないため、総リスクとシステマティックリスクは等しい。

図表7はCAPMおよび証券市場線のグラフである。前述のとおり、市場のベータは1であり、期待リターンはR_mである。この線を用いることで、任意の資産の期待リターンを計算できる。次の例でベータとリターンの計算を示す。

  Example 7Security Market Line and Expected Return

1.

無リスク利子率が3%、市場ポートフォリオの期待リターンが13%、その標準偏差が23%とする。インド企業Bajaj Autoは標準偏差50%であるが、市場とは無相関である。Bajaj Autoのベータと期待リターンを計算せよ。

解答を確認（Check Answer）

Solution

β_iの式を用いてβ_iを計算し、その後リターンを求める。

βi=ρi,mσiσm=0.0×0.500.23=0
E(Ri) = Rf + βi[E(Rm) – Rf] = 0.03 + 0 × (0.13 – 0.03) = 0.03 = 3.0%

市場ポートフォリオとの相関がゼロであるため、Bajaj Autoのベータはゼロである。ベータがゼロであるため、その期待リターンは無リスク利子率である3%となる。

2.

無リスク利子率が3%、市場ポートフォリオの期待リターンが13%、その標準偏差が23%とする。ドイツ企業Mueller Metalsは標準偏差50%、市場との相関が0.65である。Mueller Metalsのベータと期待リターンを計算せよ。

解答を確認（Check Answer）

Solution

β_iの式を用いてβ_iを計算し、その後リターンを求める。

βi=ρi,mσiσm=0.65×0.500.23=1.41
E(Ri) = Rf + βi[E(Rm) – Rf] = 0.03 + 1.41× (0.13 – 0.03) = 0.171 = 17.1%

市場との高い相関により、Mueller Metalsのベータは1.41、期待リターンは17.1%となる。Mueller Metalsは市場よりも大きなシステマティックリスクを持つため、その期待リターンは市場の期待リターンを上回る。

ポートフォリオのベータ（Portfolio Beta）

前述のとおり、証券市場線はすべての証券に適用される。それでは、ポートフォリオのような複数の証券の組み合わせにはどう適用されるのか。証券1と証券2の2つを考え、証券1のウェイトをw_i、残りを証券2に配分するとする。2つの証券のリターンおよびポートフォリオのリターンは次のように表される：

E(R1)=Rf+β1[E(Rm)−Rf]E(R2)=Rf+β2[E(Rm)−Rf]E(Rp)=w1E(R1)+w2E(R2)=w1Rf+w1β1[E(Rm)−Rf]+w2Rf+w2β2[E(Rm)−Rf]=Rf+(w1β1+w2β2)[E(Rm)−Rf]

最後の式はポートフォリオの期待リターンを表している。この式から、ポートフォリオのベータは w₁β₁ + w₂β₂ であると結論付けられる。一般に、ポートフォリオのベータは構成証券のベータの加重平均であり、次のように表される：

βp=∑i=1nwiβi; ∑i=1nwi=1

CAPMに基づくポートフォリオのリターンは次の通りである：

E(Rp) = Rf + βp[E(Rm) – Rf]

この式は、ポートフォリオの期待リターンと、そのシステマティックリスク（β_pで測定される）との間に線形関係が存在することを示している。

  Example 8Portfolio Beta and Return
1.

あなたは資金の20%を無リスク資産、30%を市場ポートフォリオ、50%をベータが2.0の米国株RedHatに投資しているとする。無リスク利子率が4%、市場リターンが16%であるとき、ポートフォリオのベータと期待リターンはいくらか。

解答を確認（Check Answer）

Solution

無リスク資産のベータは0、市場のベータは1、RedHatのベータは2.0である。ポートフォリオのベータは次の通り：

βp = w1β1 + w2β2 + w3β3 = (0.20 × 0.0) + (0.30 × 1.0) + (0.50 × 2.0) = 1.30
E(Rp) = Rf + βp[E(Rm) – Rf] = 0.04 + 1.30 × (0.16 – 0.04) = 0.196 = 19.6%

ポートフォリオのベータは1.30であり、期待リターンは19.6%である。

別解（Alternate Method）

ポートフォリオのリターンを計算する別の方法として、各証券のリターンを個別に計算し、その後ポートフォリオのリターン（すなわち各証券リターンの加重平均）を用いて全体のリターンを求める方法がある。

Return of the risk-free asset = 4 percent; return of the market = 16 percent
RedHat’s return based on its beta = 0.04 + 2.0 × (0.16 – 0.04) = 0.28
Portfolio return = (0.20 × 0.04) + (0.30 × 0.16) + (0.50 × 0.28) = 0.196 = 19.6%

当然ながら、この方法でもポートフォリオのリターンは19.6%となる。
