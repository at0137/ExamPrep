Real World Applications: Bloomberg's Contract Table Menu

モジュール6：先物契約（futures contracts）の価格決定（pricing）と評価（valuation） 実務への応用（Real World Applications）：Bloomberg Terminalのコントラクトテーブルメニュー（Contract Table Menu）

Bloomberg L.P.は、金融プロフェッショナルが業務全体で利用する主要な市場データプラットフォームの一つである。この任意セクションでは、CFAプログラム（CFA Program）のカリキュラムをBloomberg上でどのように実務に応用するかを探る。

{CTM} コントラクトテーブルメニュー（Contract Table Menu）

BloombergのCTM機能（CTM function）では、先物契約（futures contracts）の完全なカテゴリ一覧を閲覧し、それぞれをクリックすることでテナー（tenor）ごとのコモディティカーブ価格（commodity curve pricing）を確認することができる。

 

「CTM <GO> は、先物契約（futures contracts）を検索できる機能であり、カテゴリ（category）、取引所（exchange）、または地域（region）に基づいて関心のあるコモディティ（commodities）を容易に特定することができる。」―Bloomberg

 

この機能を確認するには、Bloombergのコマンドライン（command line）にCTMと入力し、Enterまたは<GO>を押す。Cornなどの番号付きメニュー項目の一つを選択すると、対応するティッカー（tickers）の一覧が表示される。次に、その中のいずれかのティッカーをクリックする。ポップアップメニューからCCRVを選択すると、先物価格カーブ（futures pricing curve）を確認できる。
