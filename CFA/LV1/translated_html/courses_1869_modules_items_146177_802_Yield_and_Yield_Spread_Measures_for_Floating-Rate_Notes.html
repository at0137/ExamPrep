8.02 | Yield and Yield Spread Measures for Floating-Rate Notes

モジュール8：変動利付証券（floating-rate instruments）の利回りおよび利回りスプレッド指標 8.02｜変動利付債（floating-rate notes）の利回りおよび利回りスプレッド指標

Learning Outcome

The candidate should be able to:

  • 変動利付証券の利回りスプレッド指標（yield spread measures）を計算し解釈する

変動利付証券の利回りおよび利回りスプレッド指標（Yield and Yield Spread Measures for Floating-Rate Instruments）

変動利付証券（floating-rate instruments）には、変動利付債（floating-rate notes, FRNs または floaters）や多くのローンが含まれ、固定利付債とは異なる。固定されたクーポン支払いではなく、参照金利（reference interest rate）の現在水準に応じて各期間ごとに変動する。これは、借り手の基準金利（base rate）を市場環境に応じて自動的に調整し、投資家または貸し手に対して市場金利の変動時における価格リスク（price risk）を低減することを目的としている。原則として、フローター（floater）は金利が大きく変動する期間においても価格が安定する。なぜなら、キャッシュフローが金利変動に応じて調整されるためである。一方、従来の固定利付証券では、将来キャッシュフローが一定であるため、金利の変動に応じて価格が変動する。

Exhibit 1は、前のレッスンで紹介されたドイツの新興技術企業であるAntelas AGが発行した4年物FRNのタームシート（term sheet）を示している。

Exhibit 1 Antelas AG 変動利付債発行（Floating-Rate Note Issue）

Antelas AG 4年物変動利付債（the “Notes”）目論見書要約（Prospectus Summary）
発行体（Issuer）: Antelas AG
決済日（Settlement Date）: [T＋5営業日（Business Days）]
満期日（Maturity Date）: [決済日から4年後（Four Years from Settlement Date）]
元本額（Principal Amount）: 2億5,000万ユーロ（EUR 250 million）
利息（Interest）: 利息支払（Interest Payment）: 年率でMRR＋250bps（MRR plus 250 bps p.a.）。MRRは四半期ごとにリセットされ、利息は四半期ごとに支払われる。決済日から3か月後に開始し、満期日に最終支払いが行われる。
優先順位（Seniority）: 本債券（Notes）は担保付（secured）かつ非劣後（unsubordinated）のAntelas AGの債務であり、他のすべての担保付かつ非劣後の債務と同順位（pari passu）である。
営業日（Business Days）: フランクフルト（Frankfurt）

利息が「MRR＋250bps」と記載されている点に注意せよ。FRNやローンにおける市場参照レート（MRR）は通常、短期のマネーマーケット金利である。通常、参照レートは期間の初めに決定され、利息支払いは期間の終わりに行われる。この支払い構造は「後払い（in arrears）」と呼ばれる。フローターの未収利息（accrued interest）を計算する際に最も一般的に用いられる日数計算はactual/360およびactual/365である。AntelasのFRNでは、MRRは四半期ごとにリセットされ、利息も四半期ごとに支払われる。

ほとんどのFRNおよびローンでは、市場参照レートに対して一定のスプレッドが加算または減算される。AntelasのFRNでは、MRRに対するスプレッドは250bpsであり、これをクォーテッド・マージン（quoted margin）と呼ぶ。クォーテッド・マージンの役割は、発行体の信用リスクと参照レートに内在する信用リスクとの差を投資家に補償することである。信用リスクが非常に低い企業は、負のクォーテッド・マージンを得られる場合もある。

必要マージン（required margin）は、FRNがリセット日に額面で価格付けされるような参照レートに対するスプレッドであり、市場によって決定される。例えば、フローターが額面で発行され、MRR＋0.50%を支払う場合、クォーテッド・マージンは50bpsである。発行体の信用リスクに変化がなければ、必要マージンも50bpsのままである。各四半期のリセット日において、フローターは額面で価格付けされる。クーポン支払日間では、MRRが低下すればフラット価格（flat price）はプレミアムに、MRRが上昇すればディスカウントになる。しかし、必要マージンがクォーテッド・マージンと同じであれば、次のリセット日が近づくにつれて価格は「パーに収束（pulled to par）」する。リセット日には、MRRの変化が次期間の利息支払いに反映される。

必要マージンの変化は通常、発行体の信用リスクの変化によって生じる。流動性や税務上の扱いの変化も影響する場合がある。これらは固定利付債の利回りスプレッドに影響を与える要因と同じである。例えば、発行体の格下げによりリセット日に必要マージンが75bpsに上昇したとする。クォーテッド・マージンが50bpsであれば、発行体は投資家に対して「不足（deficient）」な利息を支払うことになる。このFRNは額面を下回るディスカウントで価格付けされる。ディスカウントの大きさは、将来の不足キャッシュフローの現在価値であり、これは残存期間にわたる各期間25bpsの年金（annuity）に相当する。これは必要マージンとクォーテッド・マージンの差である。逆に、必要マージンが40bpsに低下した場合（信用リスクの低下など）、FRNはプレミアムで価格付けされる。このプレミアムは、各期間10bpsの「超過（excess）」利息支払いの現在価値である。

固定利付債と変動利付債は、信用リスクの変化に関しては本質的に同じである。固定利付債では、プレミアムまたはディスカウントは固定クーポン率と必要な満期利回りとの差から生じる。一方、変動利付債では、プレミアムまたはディスカウントは固定されたクォーテッド・マージンと必要マージンとの差から生じる。ただし、ベンチマーク金利の変化に対しては両者は大きく異なる。

変動利付債の評価には価格付けモデルが必要である。固定利付債の価格が市場割引率rおよび各期間のクーポンPMTによって決定されることを思い出そう。

PV=PMT(1+r)1+PMT(1+r)2+...+PMT+FV(1+r)N.

FRNでは、PMTはMRRとクォーテッド・マージンの関数であり、rはMRRとディスカウント・マージンの関数である。この直感を用いて、簡略化されたFRN価格モデルである式1を導出することができる。

PV=(MRR+QM)×FVm(1+MRR+DMm)1+(MRR+QM)×FVm(1+MRR+DMm)2+...+(MRR+QM)×FVm+FV(1+MRR+DMm)N, 1

where

  • PV = 現在価値（present value）、すなわちFRNの価格
  • MRR = 市場参照レート（market reference rate）、年率で表示（一般にIndexとも呼ばれる）
  • QM = クォーテッド・マージン（quoted margin）、年率で表示
  • FV = 満期時に支払われる将来価値（future value）、すなわち額面（par value）
  • m = FRNの周期性（periodicity）、すなわち年間の支払回数
  • DM = ディスカウント・マージン（discount margin）＝必要マージン（required margin）、年率で表示
  • N = 満期までの等間隔の期間数

式1では、MRR、QM、DMが年率であるため、年間の期間数mで割る必要があることに注意せよ。

このモデルは複数の理由で簡略化されている。第一に、PVはリセット日での値であり、満期までのN期間が等間隔であるため、未収利息はなくフラット価格がフル価格となる。第二に、周期性mが整数となるように30/360の日数計算を仮定しているが、実務では多くのフローターがactual/360を用いる。第三に、最も重要な点として、すべてのキャッシュフローに同一のMRRを用いている。より複雑なFRN価格モデルでは、分子には将来予想レート、分母にはスポットレートを用いる。したがって、ディスカウント・マージンの計算は、この簡略化モデルの仮定に依存する。

2年物で半年払いのFRNを評価すると仮定し、MRR＋0.50%を支払うとする。MRRは1.25%、投資家が要求する利回りスプレッドは40bpsとする。このとき式1の入力は以下のとおりである。

  • MRR = 0.0125
  • QM = 0.0050
  • FV = 100、
  • m = 2、
  • DM = 0.0040、そして
  • N = 4。

式1を用いてN = 4とすると、このFRNは額面100あたり100.196で価格付けされる。このフローターは、クォーテッド・マージンがディスカウント・マージンより大きいため、額面を上回るプレミアムで取引される。

PV=(MRR+QM)×FVm(1+MRR+DMm)1+(MRR+QM)×FVm(1+MRR+DMm)2+...+(MRR+QM)×FVm+FV(1+MRR+DMm)N.
PV=(0.0125+0.0050)×1002(1+0.0125+0.00402)1+(0.0125+0.0050)×1002(1+0.0125+0.00402)2+(0.0125+0.0050)×1002(1+0.0125+0.00402)3+(0.0125+0.0050)×1002+100(1+0.0125+0.00402)4
PV = 100.196

FRNの市場価格が分かっている場合、式1を用いてディスカウント・マージンを推定することもできる。例えば、2年物のFRNが半年ごとにMRR＋0.75%を支払うとする。現在、MRRは1.10%であり、このフローターの価格は額面100あたり95.50で、信用リスクの悪化によりディスカウントで取引されている。このとき、式1の入力は以下のとおりである。

  • PV = 95.50、
  • MRR = 0.0110、
  • QM = 0.0075、
  • FV = 100、
  • m = 2、そして
  • N = 4。

Using the Solver add-in for Microsoft Excel, we find DM = 3.12%, or 312 bps, by setting the price (sum of present values of cash flows) equal to 95.50 as the objective and DM as the change variable. Please see the candidate learning ecosystem online for a spreadsheet demonstrating the calculation in the downloadable Excel workbook Download downloadable Excel workbook.

このFRNが額面で発行されたとき、投資家が要求したスプレッドはMRRに対して75bpsであった。現在、信用格付けの引き下げ後、投資家は推定で312bpsのディスカウント・マージンを要求している。クォーテッド・マージンは75bpsのままであるため、各期間あたり237bps（312−75）の「不足（deficient）」が生じており、このフローターはディスカウントで取引される。ディスカウント・マージンは、簡略化されたFRN価格モデルに基づく推定値である。

式1のFRN価格モデルは、銀行などによる変動金利ローン（adjustable-rate loans）や、MRRに基づく他の固定利付証券にも同様に適用される。銀行の資金調達の多くは短期預金やその他の変動金利資金から構成されているため、貸借対照表（balance sheet）上で資産と負債のマッチングを維持するために、固定金利ローンではなく変動金利ローンを企業や個人に提供することを好む。

例1：変動利付債のディスカウント・マージンの計算（Calculating the Discount Margin for a Floating-Rate Note）

Antelas AGの4年物変動利付債において、MRRが3か月ベースであり、クォーテッド・マージンが250bpsであるとする。このフローターは額面100あたり97で取引されている。3か月のMRRが–0.55%で一定であると仮定し、30/360の日数計算および等間隔の期間を前提として、このフローターのディスカウント・マージンを計算せよ。

解答（Solution）:

We can use the Solver add-in in Microsoft Excel to quickly solve for DM = 3.29%, or 329 bps, in the same manner as in the prior example. Please see the candidate learning ecosystem online for a spreadsheet demonstrating the calculation in the downloadable Excel workbook Download downloadable Excel workbook.

または、式1の分子であるPMTをまず求めたうえで、Microsoft ExcelやGoogle Sheets、または金融計算機のRATE関数を用いることもできる。

PMT=(MRR+QM)×FVm.
PMT=(−0.0055+0.0250)×1004.
PMT=0.4875.

次に、PV = –97、FV = 100、PMT = 0.4875、N = 16を用いて、期間当たりの割引率0.686122%を求める。これを用いてディスカウント・マージン（DM）を算出する。

r=MRR+DMm.
0.00686122=−0.0055+DM4.
DM=0.0329=3.29%.

クォーテッド・マージンは参照レートに対して250bpsである。簡略化されたFRN価格モデルを用いると、ディスカウント・マージン（FRNが額面で価格付けされるために投資家が要求するスプレッド）は329bpsとなる。

3.

必要マージン（required margin）とフローター（floater）のリセット日における価格の関係を特定せよ。

1. 必要マージン ＞ クォーテッド・マージン（quoted margin） A. フローターはパー（par）で価格付けされる
2. 必要マージン ＝ クォーテッド・マージン B. フローターはディスカウント（discount）で価格付けされる
3. 必要マージン ＜ クォーテッド・マージン C. フローターはプレミアム（premium）で価格付けされる

解答を確認（Check Answer）

解答（Solution）

1. 必要マージン ＞ クォーテッド・マージン B. フローターはディスカウントで価格付けされる
2. 必要マージン ＝ クォーテッド・マージン A. フローターはパーで価格付けされる
3. 必要マージン ＜ クォーテッド・マージン C. フローターはプレミアムで価格付けされる

必要マージン ＞ クォーテッド・マージンの場合、Bが正しい。必要マージンの変化は通常、発行体の信用リスクの変化によるものである。必要マージンが上昇し、リセット日においてクォーテッド・マージンを上回る場合、フローターは「不足（deficient）」な利息を支払うことになる。このため、フローターは額面を下回るディスカウントで価格付けされる。

必要マージン ＝ クォーテッド・マージンの場合、Aが正しい。必要マージンがクォーテッド・マージンと同じである場合、次のリセット日が近づくにつれてフラット価格（flat price）は「パーに収束（pulled to par）」する。

必要マージン ＜ クォーテッド・マージンの場合、Cが正しい。必要マージンが低下してクォーテッド・マージンを下回る場合、フローターはプレミアムで価格付けされる。
