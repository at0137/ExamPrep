8.08 | Factors Affecting Option Value

モジュール8：オプションの価格付けと評価（Pricing and Valuation of Options）8.08｜オプション価値に影響を与える要因（Factors Affecting Option Value）

 Learning Outcome

The candidate should be able to:

  • オプションの価値を決定する要因を特定し、それぞれがオプション価値にどのように影響するかを説明せよ。

ここでは、オプション価値に影響を与える要因を特定し説明する。これらの要因のいくつかはフォワード契約（forward commitments）とオプションの両方に共通し、またいくつかは条件付請求権（contingent claims）特有のものである。オプション価値を決定する要因には、原資産の価値、行使価格、満期までの時間、無リスク金利、原資産価格のボラティリティ、ならびに原資産保有に伴う収益またはコストが含まれる。

原資産の価値（Value of the Underlying）

原資産の価値の変化は、オプション契約における取引の権利に対して、フォワード契約における取引義務と同じ方向の影響を与える。例えば、原資産のスポット価格が上昇すると、コールオプションおよびロング・フォワードポジションの価値は上昇する。一方、スポット価格が下落すると、プットオプションおよびショート・フォワードポジションの価値は上昇する（図表6参照）。

フォワード契約と条件付請求権の重要な違いは、原資産価値の変化に対するデリバティブ価格の変化の大きさである。フォワード契約では、デリバティブの価値は原資産価格に対して線形（1対1）で変化するのに対し、オプションでは非線形であり、将来における行使の可能性に依存する。オプションがイン・ザ・マネーであるなど、行使される可能性が高いほど、オプション価値は原資産価格に対してより敏感に反応する。この可能性は、原資産価値と行使価格との関係に依存し、これはオプション特有の特徴である。

行使価格（Exercise Price）

行使価格は、満期時にオプション買い手が取引を行うかどうかを決定する閾値である。原資産を購入する権利を持つコールオプションでは、行使価格は満期時の行使価値の下限となるため、行使価格が低いほどオプション価値は高くなる。コールオプションは満期時に Max（0, Sₜ − X）で決済されるため、Xが低いほど行使される可能性と行使時の価値の双方が増加する。

一方、プットオプションの行使価格は満期時の行使価値の上限となる。したがって、より高い行使価格で原資産を売却できるプットオプションは、その価値が高くなる。図表7は、コールおよびプットにおける行使価格の変化の影響を示している。

満期までの時間（Time to Expiration）

以前のレッスンで示したように、時間価値（time value）は、オプション価格と行使価値の差であり、原資産価格の有利な変化によって行使の収益性が高まる可能性を表す。ロング・フォワードポジションでは、長い期間において価格が上昇または下落する可能性は対称的であるが、オプションの非対称なペイオフ構造により、買い手は行使されないケースを無視できる。

コールオプションでは、満期までの期間が長いほど常に価値は上昇する。原資産の価格上昇余地は実質的に無制限であり、期間が長いほどその可能性が増加する一方、下方リスクはプレミアムに限定される。

原資産を売却する権利を持つプットオプションも通常は時間の経過によって価値が増加する。満期までの期間が長いほど、原資産価格が行使価格を下回る可能性が高まるためである。ただし、プットの買い手は行使によって（X − Sₜ）を受け取るため、満期までの期間が長いほどその現在価値は低下する。このため、特にディープ・イン・ザ・マネーのプットや高い無リスク金利の環境では、満期までの期間が長いほどプットの価値が低下する場合もある。

無リスク金利（Risk-Free Interest Rate）

以前のレッスンで、無リスク金利は資産保有の機会費用であり、デリバティブの無裁定評価にも影響することを示した。例えば、オプションの行使価値はスポット価格と行使価格の現在価値との差として表される。

Call Option Exercise Value: Max (0, (St − PV(X)))
Put Option Exercise Value: Max (0, (PV(X) − St ))

無リスク金利が上昇すると、行使価格の現在価値は低下する（オプションがイン・ザ・マネーである場合）。したがって、無リスク金利の上昇はコールオプションの行使価値を増加させ、プットオプションの行使価値を減少させる。なお、無リスク金利は時間価値には直接影響しない。

原資産のボラティリティ（Volatility of the Underlying）

ボラティリティは、原資産の将来価格変動の分散の期待値を表す指標である。ボラティリティが高いほど、正の行使価値が大きくなる可能性が高まり、一方でオプションが無価値で満期を迎える場合の下方リスクには影響しない（図表8参照）。この効果はコールおよびプットの双方に共通である。

例えば、ボラティリティが上昇すると、原資産価格の取り得る範囲が広がり、オプションの時間価値およびイン・ザ・マネーとなる可能性が高まる。逆にボラティリティが低下すると、コールおよびプットの時間価値は減少する。

原資産保有に伴う収益またはコスト（Income or Cost Related to Owning Underlying Asset）

以前のレッスンで、原資産保有に伴う便益やコストがスポット価格とフォワード価格の関係に影響することを示した。例えば、配当や利便益（convenience yield）などの収益は原資産保有者には帰属するが、デリバティブ保有者には帰属しない。この効果はフォワード契約およびオプションの両方に当てはまる。収益や便益はコールの価値を低下させ、プットの価値を上昇させる。一方、保管費用や保険料などのキャリーコスト（carry costs）は逆の効果を持ち、コールの価値を上昇させ、プットの価値を低下させる。図表9には、各要因の変化がオプション価値に与える影響（正または負）がまとめられている。

  Question Set

Factors That Affect Option Value
1.

次の文を完成させる正しい語句を選べ：原資産を購入する権利を持つコールオプションにおいて、行使価格は満期時の行使価値の________となり、行使価格が低いほどオプション価値は________なる。

解答を確認（Check Answer）

Solution

原資産を購入する権利を持つコールオプションにおいて、行使価格は満期時の行使価値の下限（lower bound）となり、行使価格が低いほどオプション価値は高くなる。

2.

次の各要因の変化（他の要因は一定とする）と、それに対応するオプション価値の変化を対応付けよ：

1. 行使価格（X）の低下 A. コールオプションとプットオプションの両方の価値を上昇させる
2. 原資産価格（Sₜ）の低下 B. コールオプションの価値を上昇させる
3. 原資産価格のボラティリティの上昇（a rise in the volatility of the underlying price） C. プットオプションの価値を増加させる（Increases the value of a put option）

解答を確認（Check Answer）

Solution
  1. Bが正しい。行使価格（X）が低下すると、コールオプションの価値は増加する。満期時の原資産価格 Sₜ が一定であれば、コールオプションの決済価値 Max（0, Sₜ − X）は X が低いほど大きくなるためである。
  2. Cが正しい。原資産価格（Sₜ）が低下すると、プットオプションの価値は増加する。プットオプションは行使価格（X）で原資産を売却する権利であるため、決済価値 Max（0, X − Sₜ）は Sₜ が低下するほど増加する。
  3. Aが正しい。原資産価格のボラティリティが上昇すると、コールオプションとプットオプションの双方の価値は増加する。原資産価格の変動幅が大きくなることで、コールまたはプットの正の行使価値が大きくなる可能性が高まり、一方でオプションが行使されずに満期を迎える場合の下方リスクには影響しないためである。
3.

次の文を完成させる正しい語句を選べ：フォワード契約（forward commitments）と条件付請求権（contingent claims）の重要な違いは、原資産価値の変化に対するデリバティブ価格変化の________である。フォワード契約ではデリバティブ価値は原資産価格の________関数であるのに対し、オプションでは________関係である。

解答を確認（Check Answer）

Solution

フォワード契約と条件付請求権の重要な違いは、原資産価値の変化に対するデリバティブ価格変化の大きさ（magnitude）である。フォワード契約ではデリバティブ価値は原資産価格の線形（linear）関数であるのに対し、オプションでは非線形（non-linear）関係である。

4.

無リスク金利の低下がプットおよびコールオプションの価値に与える影響を説明し、その理由を述べよ。

解答を確認（Check Answer）

Solution

無リスク金利の低下は、オプションの行使価格の現在価値（present value, PV(X)）を増加させる。イン・ザ・マネーのコールオプションでは、（Sₜ − PV(X)）が小さくなるため行使価値は低下する。一方、イン・ザ・マネーのプットオプションでは、（PV(X) − Sₜ）が大きくなるため行使価値は上昇する。なお、オプションの時間価値は無リスク金利の影響を直接は受けない。
