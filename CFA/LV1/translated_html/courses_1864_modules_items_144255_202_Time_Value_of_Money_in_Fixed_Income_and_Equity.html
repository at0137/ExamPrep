2.02 | Time Value of Money in Fixed Income and Equity

モジュール2：金融における貨幣の時間価値（Time Value of Money in Finance） 2.02｜債券および株式における貨幣の時間価値

 Learning Outcome

The candidate should be able to:

  • 将来の予想キャッシュフローに基づき、債券（fixed-income）および株式（equity）商品の現在価値（present value, PV）を計算し解釈する

金融商品に関連するキャッシュフローのタイミングはその価値に影響を与え、キャッシュの流入は早く受け取るほど高く評価される。貨幣の時間価値（time value of money）は、現在受け取るキャッシュフローと将来受け取るキャッシュフローとのトレードオフを表し、異なる時点で受け取るキャッシュフローの現在価値を比較することを可能にする。この差は、前の学習モジュールで示した適切な割引率rに基づいており、これは商品の種類や予想キャッシュフローのタイミングおよびリスクに応じて異なる。

一般に、現在価値（present value, PV）と将来価値（future value, FV）の関係は、rを各期間の割引率、tを複利期間数とすると、次のように表される：

FVt = PV(1 + r)t .1

複利期間数tが非常に大きい、すなわちt → ∞の場合、初期キャッシュフローは次のように連続複利で計算される：

FVt = PVer t .2

逆に、現在価値は将来価値の形でも表現でき、その場合、式1を次のように変形する：

FVt=PV1+rtPV=FVt11+rtPV=FVt1+r−t.3

式3の連続時間における等価表現は次の通りである：

PVt = FVe−r t.4

債券（Fixed-Income Instruments）と貨幣の時間価値

債券（fixed-income instruments）は、債券やローンのような負債性商品であり、発行体が将来の返済を約束する代わりに投資家から資金を借り入れる契約を表す。債券の割引率は利子率であり、債券やローンの収益率は通常、満期利回り（yield-to-maturity, YTM）と呼ばれる。

債券に関連するキャッシュフローは、通常、次の3つの一般的なパターンのいずれかに従う：

  • ディスカウント型（Discount）：投資家は債券またはローンに対して初期価格（PV）を支払い、満期時に単一の元本キャッシュフロー（FV）を受け取る。（FV − PV）の差は、期間中に得られる利息を表す。
  • 定期利息型（Periodic Interest）：投資家は初期価格（PV）を支払い、あらかじめ定められた間隔で利息キャッシュフロー（PMT）を受け取り、満期時に最終利息と元本（FV）を受け取る。
  • 定額支払い型（Level Payments）：投資家は初期価格（PV）を支払い、満期までの各期間において一定額のキャッシュフロー（A）を受け取る。これらは利息と元本返済の両方を含む。

ディスカウント型のキャッシュフローパターンは図表1に示されている：

3つのバーからなる図。最初のバーは「PV」とラベル付けされ、投資家が発行時に割引価格で債券を購入することを示す。2番目のバーは「5年後のFV」と示され、満期時に元本が支払われることを表す。3番目のバーは2色に分かれており、赤が「FV」、黄色が「PV」を示す。「FV − PVが期間中の利息を表す」という説明が付されている。

各期間の市場割引率rのもとで、時点tに元本（FV）が支払われるディスカウント債の現在価値（PV）は次のように計算される：

PV(Discount Bond) = FVt / (1 + r)t .5

投資家の唯一のリターン源は、支払価格（PV）と満期時に受け取る元本（FV）の差である。この種の債券は、中間の利息キャッシュフロー（クーポン）が存在しないため、ゼロクーポン債（zero-coupon bond）と呼ばれる。

  Example 1Discount Bonds under Positive and Negative Interest Rates

多くの政府は満期時に元本を支払う固定クーポン債を発行するが、米国、英国、インドなどの多くの政府発行体では、これらの債券から分離（ストリップ）された利息または元本のキャッシュフローがディスカウント債として売買されている。例えば、満期利回り（YTM）が6.70%の時に発行されたインド政府債において、20年後に支払われる単一の元本キャッシュフローを考える。この簡略化された例では年1回複利とし、式5のtはキャッシュフローまでの年数を表す。

1.

このディスカウント債について、元本100インドルピー（INR100）あたり投資家はどの程度の価格を支払うべきか。

解答を確認

Solution

INR27.33

式5を用いて、r = 6.70%、t = 20、FV = INR100としてPVを求める：

PV = INR100 / (1 + 0.067)20 = INR27.33.

また、Microsoft ExcelまたはGoogle SheetsのPV関数を使用することもできる：

= PV (rate, nper, pmt, FV, type),

ここで：

  • rate = 各期間の市場割引率、
  • nper = 期間数、
  • pmt = 各期のクーポン支払い（ディスカウント債では0）、
  • FV = 将来価値または額面価値、
  • type = 各期末に支払われる場合は0（本例）、各期首の場合は1。

キャッシュアウトフロー（支払価格）であるため、ExcelのPVの結果は負の符号となる：

PV = (27.33) = PV (0.067,20,0,100,0).

元本（FV）はINR100で一定であるが、価格（PV）は時間の経過および金利の変化に応じて変動する。

2.

金利が変化しないと仮定した場合、3年後の債券価格（PV）はいくらか。

解答を確認

Solution

INR33.21

式5においてt = 17を代入してPVを求める：

PV = INR100 / (1 + 0.067)17 = INR33.21.

一定のrのもとでのINR5.88の価格上昇は、3年間で得られる利息を意味する。金利が正である場合、PVは時間の経過とともに一般に上昇（またはアクリート）し、tがゼロに近づくにつれてFVに収束する。

3.

価格は金利の変化によっても変動する。t = 0で購入した後、債券価格が直ちに元本INR100あたりINR22.68224に下落したとする。このディスカウント債のインプライド金利はいくらか。

解答を確認

Solution

7.70%

ここでは式5を用いてrを次のように求める：

INR22.68224 = INR100 / (1 + r)20.

式5を変形すると次のようになる：

r = 7.70 percent = (100/22.68224)1/20 − 1.

あるいは、Microsoft ExcelまたはGoogle SheetsのRATE関数を使用することもできる：

= RATE (nper, pmt, PV, FV, type, guess)

上記と同じ引数を用い、推定値（guess）は任意の引数であり、0から1の間の値を取り、デフォルトは0.1である：

7.70 percent = RATE (20,0,-22.68224,100,0,0.1).

これは、rが100ベーシスポイント（1.00%）上昇すると、価格がINR4.65下落することを示しており、価格と満期利回り（yield-to-maturity, YTM）の逆の関係を強調している。

4.

次に、負の金利で発行された債券を考える。2016年7月、ドイツはユーロ圏で初めて、負の利回りで10年物国債を発行した。発行時のドイツ国債の年率YTMが−0.05%であった場合、元本EUR100に対する現在価値（PV）を求めよ。

解答を確認

Solution

EUR100.50

式1を用いて、r = −0.05%、t = 10、FV = EUR100としてPVを求める：

PV = EUR100.50 = EUR100 / (1 − 0.0005)10,

or

PV = (100.50) = PV (−0.0005,10,0,100,0).

発行時、この債券はプレミアム（premium）で価格付けされており、投資家は満期時に受け取る元本よりもEUR0.50高い価格を支払っている。

5.

6年後、ドイツのインフレ率が数十年ぶりの高水準に達し、投資家が要求する名目リターンが上昇した結果、このドイツ国債（設問4）が元本EUR100あたりEUR95.72で取引されているとする。この債券のYTMはいくらか。

解答を確認

Solution

1.10%

式5を用いて、PV = 95.72、FV = 100、t = 4としてrを求める：

EUR95.72 = EUR100 / (1 + r)^4；r = 1.10%

or

1.10%

= RATE (4,0,-95.72,100,0,0.1)

.

インド共和国のディスカウント債の場合、金利の上昇は債券価格を低下させる。一方、ディスカウント債が時間とともに価格が上昇（アクリート）するのに対し、プレミアム債は満期時のEUR100に向かって時間とともに価格が低下（アモータイズ）する。

クーポン債（Coupon Instrument）

元本が満期時に返済される定期的な利息キャッシュフローのパターンは図表2に示されている。この場合、すべての定期キャッシュフローは同一であり、半年ごとに発生する。

 Exhibit 2Coupon Bond Cash Flows

債券投資のタイムラインを示す図。最初のバー「PV」は投資家が発行時に現在価値で債券を購入することを示す。その後の小さなバー「PMT」は0.5から6.0までの値を持つ定期的なクーポン支払いを示す。最後のバーは満期時に投資家が最終クーポンと元本を受け取ることを示し、「PMT + FV」と表記されている。

クーポン債の価格付けは、ディスカウント債の単一キャッシュフローの計算を拡張し、市場割引率が与えられたときの債券価格（PV）を求める一般式となる：

PV(Coupon Bond) = PMT1 / (1 + r)1 + PMT2 / (1 + r)2 + … + (PMTN + FVN) / (1 + r)N.6

Solution

= 21.02+21.022+21.023+21.024+21.025+21.026+1021.027.

これは先に紹介したMicrosoft ExcelまたはGoogle SheetsのPV関数（PV(0.02, 7, 2, 100, 0)）でも求めることができる。発行価格は元本と等しい（PV = FV）。この関係は、固定クーポンが割引率と等しい場合、クーポン支払日において成立する。

2.

次に、ちょうど2年後にユーロ圏のインフレ率の急上昇により、このギリシャ国債の価格が元本EUR100あたりEUR93.091まで下落したとする。この新たな市場環境において、投資家が期待するインプライドYTMはいくらか。

この場合、PV = 93.091として式6を用いてrを求める：

PV = 93.091 = 2/(1+r) + 2/(1+r)2 + 2/(1+r)3 + 2/(1+r)4 + 102/(1+r)5

ここではMicrosoft ExcelまたはGoogle SheetsのRATE関数（RATE(5, 2, 93.091, 100, 0, 0.1)）を用いて、r = 3.532%を求めることができる。固定クーポン債の投資家は、より高いYTMが要求される場合、キャピタルロスを被る。

式6においてすべてのキャッシュフローを割り引くために用いられる金利rは債券のYTMであり、通常は年率で表示される。しかし、多くの公的・民間発行体の債券は半年ごとに利息を支払う。例3では、先の簡略例でキャッシュフローが分離（ストリップ）されたインド共和国の債券を再び取り上げる。

例1でディスカウント債として分離された20年物インド共和国政府債を考える。この債券は年率6.70%のクーポンとYTMで発行され、クーポンは半年ごとに支払われる。

例2で示したように、クーポン率（coupon rate）がYTMと等しいため、この債券の発行時の現在価値（PV）はINR100になると期待される。

永久債（perpetual bond）は満期が定められていないクーポン債の一種であり、比較的まれである。多くの永久債は企業が株式に類似した資金調達を行うために発行し、しばしば償還条項（redemption features）を含む。式6においてN → ∞とすると、早期償還のない永久的な定期キャッシュフロー（perpetuity）の現在価値は、r > 0のもとで次のように簡略化できる：

KRW97.03 = KRW0.825 / r ;

r = 各期0.85%、年率では3.40%（= 0.85% × 4）となる。

株式（Equity Instruments）と貨幣の時間価値

優先株（preferred stock）や普通株（common stock）などの株式投資は企業の所有持分を表し、投資家に配当（dividends）という形で裁量的なキャッシュフローを受け取る権利を与える。債券（fixed-income instruments）とは異なり、株式には満期がなく、企業が売却・再編・清算されない限り無期限に存続すると仮定される。企業の株式を評価する一つの方法は、期待収益率（expected rate of return, r）を用いて将来の予想キャッシュフローを割り引くことである。これらのキャッシュフローには、定期的な配当と投資期間終了時に受け取る予想株価が含まれる。

配当キャッシュフローに基づく株式評価における一般的な仮定は、通常、次の3つのアプローチのいずれかに従う：

  • 配当一定（Constant Dividends）：投資家は株式に初期価格（PV）を支払い、一定の配当（D）を定期的に受け取る。
  • 配当成長率一定（Constant Dividend Growth Rate）：投資家は株式に初期価格（PV）を支払い、1期間後に配当（D_{t+1}）を受け取り、その後配当は一定成長率gで成長すると仮定される。
  • 配当成長率変化（Changing Dividend Growth Rate）：投資家は株式に初期価格（PV）を支払い、1期間後に配当（D_{t+1}）を受け取る。配当は、企業が初期の高成長期から成熟期の低成長へ移行するにつれて、時間とともに変化する成長率で増加すると予想される。

永続的に一定配当を支払う株式の最も単純なケースは図表4に示されている。

GBP10

PV = GBP10.00 = GBP1.50/0.15.

あるいは、一般的な株式評価手法として、配当が永続的に一定成長率gで成長すると仮定する方法もある（図表5）。

 Exhibit 5Equity Cash Flows with Constant Dividend Growth

配当が各期間gの割合で成長し、各期末に支払われる場合、次期配当（時点t+1）は次のように表される：

Dt+1 = Dt (1 + g),11

または一般にi期間後では：

Dt+ i = Dt (1 + g)i .12

配当キャッシュフローが無期限にgで成長し続ける場合、式10は次のように書き換えられる：

PVt= ∑i=1∞Dt1+gi1+ri ,13

これは次のように簡略化される：

PVt= Dt1+gr−g=Dt+1r−g ,14

ただし、r − g > 0である。

一定成長の代替として、成長率が変化する仮定がある。これは、初期に急速なキャッシュフローの成長を経験し、その後成熟に伴い成長が鈍化する企業の評価に一般的である。最も単純な形は、図表6に示される2段階モデル（two-stage model）である。

 Exhibit 6Equity Cash Flows with Two-Stage Dividend Growth

図表6の例では、最初の3年間は高い短期成長率g_s、その後は低い長期成長率g_l（g_s > g_l）で無期限に成長する企業を示している。初期成長期間をn期間に一般化すると、式14は次のように修正される：

PVt= ∑i=1nDt(1+gs)i(1+r)i+ ∑j=n+1∞Dt+n(1+gl)j−n(1+r)j.15

式15の第2項はn期間後からの一定成長を含んでおり、幾何級数の簡略化を用いて表すことができる：

PVt= ∑i=1nDt1+gsi1+ri+ E(St+n)1+rn ,16

where the stock value of the stock in n periods ( E(St+n ) is referred to as the terminal value) and is equal to the following:

E(St+n)= Dt+n+1r− gl.17
