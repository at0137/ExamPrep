12.01 | Introduction

モジュール12：利回りベースの債券コンベクシティ（Yield-Based Bond Convexity）とポートフォリオ特性 12.01 | 導入（Introduction）

デュレーション（duration）は債券価格の利回り変化に対する感応度を線形近似したものであるが、実際の債券価格と最終利回り（yield-to-maturity）との関係は曲線（コンベクス、convex）である。この非線形関係を考慮するために、修正デュレーション（modified duration）のみに基づく価格変化推定を改善する補完的なリスク指標としてコンベクシティ（convexity）を導入する。コンベクシティ調整は、利回り変化が大きい場合や満期が長い債券において、より重要となる。また、本章では債券ポートフォリオのデュレーションおよびコンベクシティの推定方法と、その前提に基づく制約についても説明する。

Most of the examples and exhibits used throughout the reading can be downloaded as a Microsoft Excel workbook Download downloaded as a Microsoft Excel workbook. Each worksheet in the workbook is labeled with the corresponding example or exhibit number in the text.

学習モジュール概要（Learning Module Overview）

  • コンベクシティ（convexity）は、オプションなし固定利付債（option-free fixed-rate bond）における利回り変化の価格への二次（非線形）効果を測定する補完的なリスク指標である。コンベクシティ調整は、修正デュレーションが与える線形的な価格推定に加算される。
  • オプションなし固定利付債において、コンベクシティ（convexity）は常に正である。このため、利回り低下時の価格上昇はデュレーションのみで示されるよりも大きく、利回り上昇時の価格下落はデュレーションのみで示されるよりも小さくなる。したがって、コンベクシティは投資家にとって有用である。
  • コンベクシティ（convexity）はデュレーションと同様の関係を債券特性と持つ。すなわち、固定利付債は満期までの期間（time-to-maturity）が長いほど、クーポンレートが低いほど、そして最終利回り（yield-to-maturity）が低いほど、コンベクシティは大きくなる。
  • マネー・コンベクシティ（money convexity）は、債券の年率コンベクシティとフルプライス（full price）との積であるため、ポジションに対して通貨単位または額面比率でコンベクシティを表す。
  • ポートフォリオのデュレーションおよびコンベクシティは、(1) 集約されたキャッシュフローの受取までの時間の加重平均として、または (2) ポートフォリオを構成する個々の債券のデュレーションおよびコンベクシティの加重平均として、計算することができる。
  • 第1の方法は理論的には正しいが、実務上は使用が困難である。第2の方法はポートフォリオ・マネージャーによって一般的に用いられるが、イールドカーブの平行シフト（parallel shifts）を暗黙に仮定しており、これは実際には稀である。

6.

Using weighted-average duration and convexity measures to estimate bond portfolio interest rate risk:


Solution

C is correct. Using the weighted-average duration and convexity measures to estimate bond portfolio interest rate risk is easy in practice, but it is not the theoretically correct approach (using the weighted average of time to receipt of the aggregate cash flows is theoretically correct). It does, however, implicitly assume a parallel shift in the yield curve.
