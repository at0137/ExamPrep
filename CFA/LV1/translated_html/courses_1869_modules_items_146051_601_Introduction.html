6.01 | Introduction

モジュール6：債券評価（Fixed-Income Bond Valuation）：価格と利回り（Prices and Yields）6.01｜イントロダクション（Introduction）

本章では、割引キャッシュフロー分析（discounted cash flow analysis）を用いて債券価格を算出し、使用する割引率やクーポン利率、残存期間（time-to-maturity）といった債券の特性が価格にどのように影響するかを示す。債券価格と将来キャッシュフローからは内部収益率（internal rate of return）を算出でき、これは満期利回り（yield-to-maturity, YTM）と呼ばれ、一定の仮定の下で固定利付投資家にとって有用なリターン指標となる。本モジュールで適用する仮定の一つは、すべての利息および元本のキャッシュフローが約定通りに支払われるというものである。債券価格とその特性の関係を検討し、異なる特性が価格に与える影響を示し、クーポン支払日およびその間の価格算定を説明する。最後に、価格や満期利回りが不明な場合に、類似債券を用いて推定するマトリックス・プライシング（matrix pricing）を紹介する。

Most of the examples and exhibits used throughout the reading can be downloaded as a Microsoft Excel workbook Download downloaded as a Microsoft Excel workbook. Each worksheet in the workbook is labeled with the corresponding example or exhibit number in the text.

学習モジュール概要（Learning Module Overview）

  • 債券価格は割引キャッシュフロー分析の応用である。債券価格は、そのキャッシュフローや将来キャッシュフローの割引に用いられる金利など、債券の特性の関数である。
  • 債券価格と額面、またはクーポン利率と割引率を比較することで、債券がディスカウント（discount）、パー（par）、またはプレミアム（premium）で取引されているかを判定できる。
  • 債券の市場価格が既知であれば、キャッシュフローの内部収益率を算出でき、これを満期利回り（yield-to-maturity, YTM）という。YTMは将来キャッシュフローの現在価値を債券価格に等しくする単一の利率である。
  • 債券投資家の収益率は、(1)満期まで保有し、(2)発行体が予定通りに利息および元本を全額支払い、(3)すべてのクーポンを同一のYTMで再投資する場合に、YTMと一致する。
  • クーポン支払日と支払日の間で債券が価格付けまたは取引される場合、前回のクーポン支払日以降に発生した経過利息（accrued interest）を売り手に補償するために追加する必要がある。これは買い手が次回のクーポンを全額受け取るためである。
  • 任意の日における経過利息を計算するには、クーポン額に対して、クーポン期間に経過した日数の割合（経過日数／クーポン期間の日数）を乗じる。日数計算には複数の慣行があり、代表的なものにactual/actualおよび30/360がある。
  • 債券価格はYTMの変化と逆方向に変動する。債券の特性は、YTMの変化に対する価格感応度を決定する。
  • 固定クーポン債では、クーポン利率が低いほど、YTMの一定の変化に対する価格変動率は大きくなる。一般に、残存期間が長いほど、YTMの一定の変化に対する価格変動率は大きくなる。
  • 上場株式とは異なり、多くの債券は取引が活発ではなく（thinly traded）、価格発見（price discovery）が困難になる。
  • マトリックス・プライシング（matrix pricing）は、新発債や流動性の低い債券の価格を、同一または類似の特性を持つ証券の利回りを用いて推定する手法である。マトリックス・プライシングは債券の価格提示に広く用いられている。

1.

2035年1月1日に満期を迎える企業債は、年率3.25%の半年ごとのクーポンを支払い、額面は100である。市場割引率は4.0%である。取引受渡日が2030年1月1日の場合、30/360日数計算を仮定したときの債券価格（額面比％）として最も近いものはどれか。


解答

Aが正しい。債券価格は、クーポンおよび元本支払いを市場割引率で割り引いた現在価値の合計である。

PV = PMT 1/(1 + r)1 + PMT 2/(1 + r)2 + … + (PMTN + FVN )/(1 + r) N ,

where

  • PMT = 1.625
  • r = 0.02
  • FV = 100
  • N = 10
PV=1.625(1+0.02)1+1.625(1+0.02)2+1.625(1+0.02)3+1.625(1+0.02)4+1.625(1+0.02)5+
1.625(1+0.02)6+1.625(1+0.02)7+1.625(1+0.02)8+1.625(1+0.02)9+101.625(1+0.02)10
PV=96.632.

注：計算では数値は丸めていないが、表示のために小数第3位まで示している。解答は小数第3位まで丸める。

この計算はMicrosoft ExcelやGoogle SheetsのPV関数またはPRICE関数でも行うことができる：

  • =-PV(rate, nper, pmt, [FV], type)
  • =-PV(0.02,10,1.625,100,0)
  • = 96.632

  • =PRICE(DATE(2030,1,1),DATE(2035,1,1),0.0325,0.04,100,2)
  • = 96.632

3.

2030年1月1日に発行された3年物のソブリン債は、毎年1.5%の半年ごとのクーポンを支払い、各年の6月30日と12月31日に支払われる。額面は100であり、市場割引率は2.0%である。取引受渡日が2031年8月29日の場合、actual/actual日数計算を仮定したときのフラット価格（flat price）（額面比％）として最も近いものはどれか。


解答

Aが正しい。フラット価格はフル価格（full price）から経過利息（accrued interest）を差し引いたものである。フル価格は受渡日時点での現在価値であり、クーポン支払日の間であるため部分期間を含む。

PVFull=PMT1+r1−t/T+PMT1+r2−t/T+⋯+PMT+FV1+rN−t/T.

Here,

  • PMT = 0.75
  • r = 0.01
  • t = 60（2031年6月30日から2031年8月29日までの日数）
  • T = 184（2031年6月30日から次回クーポン支払日の2031年12月31日までの日数）
  • FV = 100
  • N = 3
PVFull=0.75(1+0.01)1−60/184+0.75(1+0.01)2−60/184100.75(1+0.01)3−60/184
PVFull=99.587.

経過利息は、次回クーポン支払いのうち売り手に帰属する比例配分分である。

AI=tT×PMT,
AI=60184×0.75
AI=0.245.

フラット価格は 99.587 − 0.245 = 99.343。

現在価値の計算はスプレッドシートでも容易に行える。また、Microsoft ExcelやGoogle SheetsのPRICE関数を用いてフラット価格（フル価格ではない）を直接求めることもできる。

  • =PRICE(受渡日（settlement）, 満期日（maturity）, クーポン利率（rate）, 利回り（yield）, 償還額（redemption）, 支払頻度（frequency）, [日数計算基準（basis）])
  • =PRICE(DATE(2031,8,29),DATE(2032,12,31),0.015,0.02,100,2)
  • = 99.343
