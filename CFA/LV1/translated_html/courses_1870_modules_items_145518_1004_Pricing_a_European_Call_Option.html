  Example 1Hightest Capital

Hightest Capital は、配当を支払わないある株式（non-dividend-paying stock）が現在$50で取引されていると考えており、行使価格$55の1年満期ヨーロピアン・コール・オプション（European call option）の売却を検討している。以下の問いに答えよ：

  1. 株価が今後1年間で20%上昇または下落すると予想される場合、Hightest は売却するコール・オプションに対していくらの価格を受け取るべきか。無リスク利子率は5%とする。
  2. 株価が今後1年間で40%上昇または下落すると予想される場合、コール・オプション価格はどのように変化するか。
  3. Hightest が将来の株価に対してより楽観的な見通し（すなわち、オプションがイン・ザ・マネー（in-the-money）で満期を迎える確率が高いと見積もる）を持っていた場合、期待されるコール・オプション価格はどのように変化するか。
  4. 株価が20%変動すると予想される場合、および40%変動すると予想される場合における、行使価格$55の1年満期プット・オプション（put option）の価格はいくらか。

解答1：

原資産株式の初期価格を S_0 = $50、コール・オプションの行使価格を X = $55 とする。株価が20%上昇または下落する場合：

S1u=RuS0=1.2×$50=$60.
S1d=RdS0=0.8×$50=$40.

価格は$60に上昇するか、$40に下落する。満期におけるコール・オプションのペイオフは

c1u=Max0, S1u−X=Max0, $60−$55=$5.
c1d=Max0, S1d−X=Max0, $40−$55=$0.

原資産株価が20%上昇した場合、コール・オプション価値は$5となり、20%下落した場合は0となる。オプションのヘッジ比率（hedge ratio）は：

h*=c1u−c1dS1u−S1d=$5.00−$0$60.00−$40.00=$5.00$20.00=0.25.

無リスク・ポートフォリオを構築するために、本日コール・オプションを売却し、原資産を0.25単位購入する。満期時のヘッジ済みポートフォリオ価値は：

V1=V1u=h*S1u−c1u=0.25×$60.00−$5.00=$10.00.
V1=V1d=h*S1d−c1d=0.25×$40.00−$0.00=$10.00.

確定値

V 0 = V 1(1 + r)−1 = $10.00 × 1.05−1 = $9.52.

コール・オプションの価値は：

c 0 = h*S 0 − V 0 = 0.25 × $50.00 − $9.52 = $12.50 − $9.52 = $2.98.

コール・オプションの無裁定価格（no-arbitrage price）は$2.98である。そうでなければ、投資家はオプションとその原資産を用いて、無リスク利子率を上回るリターンを持つ合成無リスク資産（synthetic risk-free asset）を構築できてしまう。

この例ではデリバティブを売却し原資産を購入しているため、ヘッジ比率（この場合0.25）は正であることに注意する。負のヘッジ比率は、デリバティブと原資産の両方を購入または売却することでヘッジを構築することを意味する。また、ヘッジ比率は通常、原資産単位に対する割合で表されるが、オプション契約は通常より大きな単位で取引されるため、ヘッジとして整数単位の原資産を売買することが可能である。ポートフォリオにおいては、オプションと原資産の比率（この場合4:1）を維持することが重要である。

解答2：

株価が40%変動する場合、満期におけるコール・オプションのペイオフは：

c1u=Max0, S1u−X=Max0, $70−$55=$15.
c1d=Max0, S1d−X=Max0, $30−$55=0.

オプションのヘッジ比率は：

h*=c1u−c1dS1u−S1d=$15−$0$70−$30=$15$40=0.375.

満期時における完全にヘッジされたポートフォリオ価値は：

V1=V1u=h*S1u−c1u=0.375×$70−$15=$11.25.
V1=V1d=h*S1d−c1d=0.375×$30−$0=$11.25.

時点

V 0 = V 1(1 + r)−1 = $11.25 × 1.05−1 = $10.71.

したがって、時点

c 0 = h*S 0 − V 0 = 0.375 × $50 − $10.71 = $18.75 − $10.71 = $8.04.

期待される価格変動が大きいほどボラティリティ（volatility）は高い。将来の価格変動範囲が広がるほど、コール・オプションの価値は増加する。

解答4：

プット・コール・パリティ（put–call parity）

p 0 = c 0 − S 0 + X(1 + r)− T = $2.98 − $50 + $55(1 + 0.05)−1 = $5.36.

原資産株価が40%変動しコール・オプション価格が$8.04の場合、プット・オプション価格は：

p 0 = c 0 − S 0 + X(1 + r)− T = $8.04 − $50 + $55(1 + 0.05)−1 = $10.42.

 

  Question Set

オプションの二項評価（Binomial Valuation of Options）

1.

空欄を埋めて正しい答えを選べ：オプションのような条件付請求権（contingent claim）を価格付けするには、契約のペイオフが__________な性質を持つため、原資産の__________のモデルが必要である。

解答を確認（Check Answer）

解答

オプションのような条件付請求権（contingent claim）を価格付けするには、契約のペイオフが非対称的（asymmetric）であるため、原資産の将来の価格の振る舞い（future price behavior）のモデルが必要となる。

2.

条件付請求権（contingent claim）の価格付けとフォワード契約（forward commitment）の価格付けの主な違いを説明せよ。

解答を確認（Check Answer）

解答

フォワード契約（forward commitment）のペイオフは対称的（symmetric）（すなわち満期において取引する義務がある）であるため、原資産の将来の価格の振る舞いをモデル化することなく完全に複製（replicate）できる。一方、条件付請求権（contingent claim）のペイオフは非対称的（asymmetric）（すなわち満期において取引する権利はあるが義務はない）であるため、将来の価格の振る舞いをモデル化する必要がある。

5.

配当を支払わない株式（non-dividend-paying stock）に対するプット・オプション（put option）は、行使価格

  1. 投資家がプット・オプションとその原資産株式を用いて、どのように完全にヘッジされたポートフォリオ（perfectly hedged portfolio）を構築できるか説明せよ。
  2. 株価が上昇した場合と下落した場合におけるヘッジ・ポートフォリオの価値はいくらか（無リスク利子率は4%とする）。
  3. プット・オプションの無裁定価格（no-arbitrage price）はいくらか。

解答を確認（Check Answer）

解答

a. 原資産株式の初期価格を

S1u=RuS0=1.1×£20.00=£22.00.

株価が10%下落した場合：

S1d=RdS0=0.9×£20.00=£18.00.

満期におけるプット・オプションのペイオフは

p1u=Max0, X−S1u=Max0, £21.00−£22.00=£0.
p1d=Max0, X−S1d=Max0, £21.00−£18.00=£3.00.

原資産株価が10%下落した場合、プット・オプションの価値は£3.00となり、10%上昇した場合は無価値となる。オプションのヘッジ比率（hedge ratio）は：

h*=p1u−p1dS1u−S1d=£0−£3£22−£18= −£3£4=−0.75.

したがって、無リスク・ポートフォリオを構築するために、プット・オプションを購入し、原資産を0.75単位購入する。

b. 満期時における完全にヘッジされたポートフォリオの価値は：

V1=V1u=h*S1u+p1u=0.75×£22+£0=£16.50.
V1=V1d=h*S1d+p1d=0.75×£18+£3.00=£16.50.

確定値

V 0 = V 1(1 + r)−1 = £16.50 × 1.04−0.5 = £16.18.

c. プット・オプションの価格は：

p 0 = V 0 − h*S 0− = £16.18 − 0.75 × £20 = £16.18 − £15 = £1.18.

プット・オプションの無裁定価格は£1.18であるべきである。

6.

空欄を埋めて正しい答えを選べ：________________が成立する場合、すべての（実在または合成の）無リスク資産の収益率は__________に等しくなるべきである。

解答を確認（Check Answer）

解答

一物一価の法則（law of one price）が成立する場合、すべての（実在または合成の）無リスク資産の収益率は無リスク利子率（risk-free rate）に等しくなるべきである。

デリバティブ・ポジションに伴うリスクを相殺するために必要な原資産投資ポジションの割合。
