Real World Applications: Bloomberg's Financial Analysis - Dupont Analysis

モジュール11：財務分析手法（Financial Analysis Techniques）実務応用（Real World Applications）：Bloombergの財務分析—デュポン分析（Dupont Analysis）

Bloombergは、金融プロフェッショナルが業務全体を通じて世界的に利用している主要なマーケットデータプラットフォームの一つである。本オプションセクションでは、CFAプログラムのカリキュラムがBloomberg上でどのように実務で応用されるかを探る。

{FA ROE} 財務分析：デュポン分析（FINANCIAL ANALYSIS: DUPONT ANALYSIS）

BloombergのFA ROE機能は、任意の上場企業について自己資本利益率（ROE）の完全なデュポン分解（DuPont disaggregation）を表示し、その構成要素の変化の影響を確認することを可能にする。

 

「FA <GO>は、企業または株式指数に関する包括的な財務情報（過去のファンダメンタルデータおよび将来予測を含む）を提供し、潜在的な投資、提携、または買収対象の価値を分析できるようにする。Ratiosタブは、企業の指標を同一基準で比較するのに役立つ。DuPont Analysisサブタブ、すなわちFA ROE <GO>は、自己資本利益率の要因分解を表示することを可能にする。」—Bloomberg

 

この機能を利用するには、BloombergのコマンドラインにFA ROEと入力し、Enterまたは<GO>を押す。次に、左上のアンバー色のフィールドに銘柄（例：Sociedad Quimica y Minera de Chile SA（SQM/B CI Equity））を入力し、Enterまたは<GO>を押す。
