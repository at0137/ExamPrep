2.07 | Forward Commitments vs. Contingent Claims

モジュール2：フォワード・コミットメントおよびコンティンジェント・クレームの特徴と商品 2.07 | フォワード型契約とコンティンジェント・クレームの比較（Forward Commitments vs. Contingent Claims）

Learning Outcome

The candidate should be able to:

  • フォワード型契約（forward commitments）とコンティンジェント・クレーム（contingent claims）を比較せよ。

確定契約（firm commitment）では、両当事者がデリバティブ契約に基づいて履行する義務を負うのに対し、オプションの買い手は、満期時における原資産価格と行使価格との関係に応じて契約を履行するかどうかを選択できる。市場参加者は、これら異なるデリバティブ商品を用いて類似のエクスポージャーを構築することが多い。例えば、ロング・フォワードとロング・コールオプションはいずれも原資産価格の上昇から利益を得る。図表12は、行使価格 X がフォワード価格 F₀(T) に等しい場合のこれら2つの契約のペイオフと利益を比較している。

図表12 ロング・フォワードとロング・コールオプションのペイオフ構造

Expand / Collapse Extended Description

横軸にスポット価格、縦軸にペイオフまたは利益をとる。右上がりの直線（フォワードの利益）は第3象限から第1象限へ伸び、X = F₀(T) で横軸と交差する。破線の「オプションのペイオフ」は原点から始まり、フォワードの利益線と交差し、その後重なる。別の「オプションの利益」線は第3象限の点から始まり、フォワードの利益線と交差し、X = F₀(T) を通過してその後はフォワードの利益線と平行に上昇する。

先に示したように、ロング・フォワードとコールオプションのペイオフはいずれも S_T の上昇に伴って増加する。フォワードの場合、この線形関係は [S_T − F₀(T)] に等しく、契約開始時にキャッシュフローがないためペイオフと利益は一致する。行使価格が F₀(T) のコールオプション買い手の場合、式2は次のように変形される。

Π = max[0, ST – F0(T)] – c0.

フォワードのペイオフ／利益 [S_T − F₀(T)] をコールオプションの利益 Π と等置すると、両者の相対的な利益関係は以下のようになる。

  • S_T − F₀(T) > −c₀ の場合、フォワードの利益はオプションの利益を上回る。
  • S_T − F₀(T) = −c₀ の場合、フォワードの利益はコールオプションの利益と等しい。
  • S_T − F₀(T) < −c₀ の場合、オプションの利益はフォワードの利益を上回る。

図表12におけるフォワードとコールオプションの利益図の比較は、ロング・コールオプションが、プレミアムを支払う代わりに下方リスクを限定した原資産のロングポジションに類似していることを示している。

原資産価格の上昇から利益を得る別のコンティンジェント・クレームとして、プットオプションの売り（ショート・プット）がある。ロング・コールおよびロング・フォワードのペイオフは原資産価格が行使価格を上回ると上昇するが、プット売り手の利益は受け取ったプレミアムに限定される。図表13は、行使価格 X が F₀(T) に等しい場合のショート・プットとロング・フォワードのペイオフおよび利益を比較している。

図表13 ロング・フォワードとショート・プットオプションのペイオフ構造

Expand / Collapse Extended Description

横軸にスポット価格、縦軸にペイオフまたは利益をとる。右上がりの直線（フォワードの利益）は第3象限から第1象限へ伸び、X = F₀(T) で横軸と交差する。破線の「オプションのペイオフ」は横軸の少し上から始まり、横軸と平行になる。「オプションの利益」はフォワードの利益線と交差し、横軸付近の点を境に横軸と平行になる。

ロング・フォワードのペイオフ／利益 [S_T − F₀(T)] を、式6を修正した形と比較できる。

Π = –max[0, F0(T) – ST ] + p0.

フォワードの利益 [S_T − F₀(T)] をプットオプションの利益 Π と等置すると、両者の相対的な利益関係は以下のようになる。

  • S_T − F₀(T) > p₀ の場合、フォワードの利益はオプションの利益を上回る。
  • S_T − F₀(T) = p₀ の場合、フォワードの利益はオプションの利益と等しい。
  • S_T − F₀(T) < p₀ の場合、オプションの利益はフォワードの利益を上回る。

図表13におけるフォワードとショート・プットの利益図の比較は、ショート・プットが、プレミアムを受け取る代わりに価格上昇による利益を放棄した原資産のロングポジションに類似していることを示している。ロング・コール、ショート・プット、およびロング・フォワードの関係に見られる対称性は、後の章でさらに詳しく検討される。

問題セット（Question Set）

確定契約とコンティンジェント・クレーム（Firm Commitments and Contingent Claims）

1.

空欄に入る正しい語を選べ：原資産価格の下落から利益を得るのは、_____ フォワードポジション、_____ コールオプションポジション、および _____ プットオプションポジションである。

解答を確認（Check Answer）

解答（Solution）

ショート・フォワードポジション、ショート・コールオプションポジション、およびロング・プットオプションポジションは、原資産価格の下落から利益を得る。

2.

行使価格 X が F₀(T) に等しいと仮定した場合、以下の満期時の利益プロファイルに対応するデリバティブポジションを識別せよ。

A. ショート・フォワードポジション 1.
B. ロング・プットポジション 2.
C. ショート・コールポジション 3.

解答を確認（Check Answer）

解答（Solution）

1. 正解はBである。

2. 正解はAである。

3. 正解はCである。

3.

行使価格 X がフォワード価格 F₀(T) に等しい場合において、ショート・フォワードとロング・プットが同じ利益となる点を説明せよ。

解答を確認（Check Answer）

解答（Solution）

ショート・フォワードとロング・プットのポジションは、F₀(T) − S_T = −p₀ のときに同じ利益となる。ショート・フォワードは原資産価格の下落から利益を得て、そのペイオフ／利益は [F₀(T) − S_T] である。一方、ロング・プットの利益は、式4より Π = max[0, F₀(T) − S_T] − p₀ であり、ここで行使価格はフォワード価格に等しく X = F₀(T) である。
