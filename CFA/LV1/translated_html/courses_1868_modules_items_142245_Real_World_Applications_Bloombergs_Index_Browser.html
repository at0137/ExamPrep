Real World Applications: Bloomberg's Index Browser

モジュール2：証券市場指数（Security Market Indexes）実務応用（Real World Applications）：ブルームバーグのインデックス・ブラウザ（Bloomberg's Index Browser）

ブルームバーグ（Bloomberg）は、金融プロフェッショナルが業務全体を通じて世界的に利用している主要な市場データプラットフォームの一つである。本オプションセクションでは、CFAプログラム（CFA Program）のカリキュラムの実務応用をブルームバーグ上で探る。

{IN} ブルームバーグ・インデックス・ブラウザ（BLOOMBERG INDEX BROWSER）

ブルームバーグのIN機能（IN function）を使用すると、資産クラス（asset class）別に最も広く利用されている指数を閲覧することができる。

 

「IN <GO> は、ブルームバーグのグローバルなマルチ資産クラス（multi-asset class）の指数ファミリー（family of indices）（ブルームバーグ債券指数（Bloomberg Fixed Income indices）を含む）の最新のパフォーマンスデータおよび履歴を表示できる機能であり、リスク分析（risk analysis）に基づいて情報に基づく投資判断を行うために、期間収益（periodic returns）や統計データ（statistical data）を分析することができる。」—ブルームバーグ

 

この機能を利用するには、ブルームバーグのコマンドラインに「IN」と入力し、Enterまたは<GO>を押す。タブ（tabs）を使用して分析したいブルームバーグ指数の資産クラスを選択する。サブタブ（subtabs）では、各資産クラスが関連する指数ファミリーごとに分類されている。
