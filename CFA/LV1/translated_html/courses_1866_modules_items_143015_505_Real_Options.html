5.05 | Real Options

モジュール5：資本投資（Capital Investments）と資本配分（Capital Allocation） 5.05｜リアルオプション（Real Options）

 Learning Outcome

The candidate should be able to:

  • 資本投資に関連するリアルオプション（real options）の種類を説明する

これまで説明してきた資本配分プロセスは、企業がプロジェクト開始時にすべての資本投資意思決定を行い、その後は単一の方針を維持することを前提としている。しかし実務では、企業はリアルオプション（real options）と呼ばれる選択肢を持ち、それが資本投資の価値を変化させる可能性がある。すなわち、資本投資の意思決定は一度きりではなく、現在行うものと将来に延期できるものの連続である。金融オプション（financial options）と同様に、リアルオプションは企業に将来の行動に関する権利を与えるが、義務は伴わない。企業は価値を高める場合にのみリアルオプションを行使（exercise）すべきである。

金融オプションの価値が基礎資産（underlying asset）の将来価値に依存するのと同様に、リアルオプションも将来の事象に依存する。リアルオプションは企業に柔軟性を提供し、以下の形で資本投資のNPVを大きく向上させる可能性がある。

  • タイミングオプション（Timing option）：企業は今すぐ投資する代わりに投資を延期することができる。この場合、短期的なリターンを放棄する代わりに、より良い情報を得てNPVを改善することを期待する。また、投資は段階的に行うことができ、現在の投資が将来の投資機会を生み出すこともある。
  • 規模オプション（Sizing option）：放棄オプション（abandonment option）は、投資実行後に財務成果が不十分であれば投資を中止することを可能にする。将来、投資を継続するキャッシュフローの現在価値よりも放棄によるキャッシュフローが大きい場合、企業は放棄オプションを行使すべきである。一方、将来の業績が良好な場合に追加投資が可能であれば、それは成長オプション（growth option）または拡張オプション（expansion option）と呼ばれる。
  • 柔軟性オプション（Flexibility option）：投資完了後も、放棄や拡張以外の運営上の柔軟性が価値を生む場合がある。例えば需要が供給能力を上回る場合、企業は価格設定オプション（price-setting option）を行使して価格を引き上げることができる。また、需要が予測と異なる場合、生産量を調整する柔軟性もある（例：残業や追加シフト）。
  • 基礎オプション（Fundamental option）：投資価値が企業外の要因（例：商品価格）に依存する場合である。例えば油田や精製設備の価値は原油価格に依存し、金鉱山は金価格に依存する。価格が低ければ投資は行われず、高ければ実行される。多くの研究開発（R&D）プロジェクトもこれに類似する。

企業はこれらの特性を持つ投資を評価するために、以下の方法を用いる：

  1. オプションを考慮しない投資分析（Investment analysis without considering options）：リアルオプションを考慮せずにNPVが正であり、かつオプションが価値を高める可能性がある場合、そのNPVは最低リターンを示すため投資すべきである。

  2. リアルオプションを含めたNPVの計算（Calculation of NPV with real options）：通常のNPVに加え、リアルオプションの価値を加え、そのコストを差し引いて評価する（式4）。

    Project NPV = NPV (without options) – Option cost + Option value.4
  3. 意思決定木（decision trees）およびオプション価格モデル（option pricing models）：将来の複数の意思決定や結果を伴う投資の評価に用いられ、確率とタイミングを考慮してNPVを計算する。

リアルオプションの有無にかかわらず、投資のリターンを評価するには将来事象の確率を仮定する必要がある。以下の例では意思決定木を用いて評価する。

2.

ScolarCorpは、新製品に対する関心を測定するための市場調査キャンペーン（market research campaign）を計画している。新製品への関心が高ければ、同社は製品製造を可能にするため既存の生産プラントをアップグレードする必要がある。一方、関心が低ければ、ScolarCorpはプラントを9,000万ドルで売却する予定である。本プロジェクトの必要収益率（required rate of return）は7.5%である。ScolarCorpのアナリストは、各代替利用に関連するキャッシュフローのタイミングと確率を、以下の意思決定ツリー（decision tree）にまとめた。

Expand / Collapse Extended Description

図は3つの時間期間（t = 1, 2, 3）にわたり、一連の意思決定を示している。t = 0では、初期の選択は市場調査であり、そのコストは30ドルである。t = 1では、市場調査の結果に基づき、プラントのアップグレードまたは売却のいずれかを選択することができ、それぞれに異なるコストと便益があり、調査結果の影響を受ける。プラントをアップグレードした場合、新製品を投入するか、またはプラントを売却することができる。

意思決定ツリーに基づき、以下の要因と資本プロジェクトの価値（オプション価値）への影響を対応させよ：

  • 市場調査コストの増加
  • 市場調査コストの減少
  • プラント売却価格の上昇
  • プラント売却価格の低下
  • 必要収益率の上昇

すべての代替案とその確率を考慮したプロジェクトの正味現在価値（NPV）の計算は、以下のように行うことができる：

NPV=CF0+CF1(1+r)1+CF1(1+r)2.
NPV=−30+−55×0.7+90×0.3(1+0.075)1+200×0.7×0.5−80×0.7×0.4+100×0.7×0.1(1+0.075)2.
NPV = $6.549.

市場調査コスト増加の影響

市場調査コストが増加した場合、例えば10増加して（–40となると）、プロジェクト全体のNPVは同額だけ減少する。初期支出はt = 0で発生するため、割引する必要はない。

プラント売却価格上昇の影響

プラントの売却価格が（1年目または2年目のいずれにおいても）上昇した場合、対応するキャッシュフローは確率調整およびt = 0への割引が必要であるため、プロジェクト全体のNPVはより小さい金額だけ増加する。

必要収益率の上昇の影響

必要収益率の上昇は、将来キャッシュフローの割引に用いられるため、プロジェクト全体のNPVを減少させる。

3.

XYZ社は、海外の配送センターの拡張を検討している。現地政府は、再生可能エネルギーに関連する投資に対して補助金や税制優遇を導入する新たな環境規制を検討している。この新法が承認されれば、XYZ社はより低コストで配送センターをアップグレードできる可能性がある。企業が新規制の施行を待つことを選択した場合、180万ドルのプロジェクト関連コストを負担する必要がある。XYZ社の財務チームは、待つ場合のプロジェクトのNPVは970万ドルとなり、現在の価値890万ドルと比較して高くなると見積もっている。オプション価値を計算せよ。

解答を確認

解答

オプション価値を求めるには、オプションありの場合となしの場合のプロジェクト価値を比較し、さらにオプションコストを考慮する必要がある。

Project NPV (with option) = $9.7 million.
Project NPV (without option) = $8.9 million.
Option cost = $1.8 million.
Project NPV (with option) = Project NPV (without option) – Option cost + Option value.
$9.7 million = $8.9 million – $1.8 million + Option value.
Option value = $2.6 million.

将来の予測シナリオからのキャッシュフローを変更するような資本投資に関して、経営者が意思決定を行う権利（ただし義務ではない）。

オプション保有者による原資産（underlying）の取引を行う意思決定。

財務的な成果が期待外れであった場合に、将来のある時点で投資を終了するオプション。

財務的な成果が良好であった場合に、将来のある時点で追加投資を行うオプション。拡張オプション（expansion option）とも呼ばれる。

需要または供給が予測と異なる場合に価格を調整するオプション。

需要が予測と異なる場合に生産量を変更するオプション。
