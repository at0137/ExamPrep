4.03 | Probability Trees and Conditional Expectations

モジュール4：確率ツリーと条件付き期待値（Probability Trees and Conditional Expectations）4.03 | 確率ツリーと条件付き期待値（Probability Trees and Conditional Expectations）

 Learning Outcome

The candidate should be able to:

  • 投資問題を確率ツリーとして定式化し、投資応用における条件付き期待値（conditional expectations）の利用を説明する

投資においては、予測を行う際に利用可能なあらゆる関連情報を活用する。期待や予測を精緻化する際には、通常、新しい情報や出来事に基づいて更新する。このような場合、条件付き期待値（conditional expected values）を用いる。事象またはシナリオSが与えられたときの確率変数Xの期待値は E(X | S) と表される。確率変数Xが n 個の異なる結果 X₁, X₂, …, Xₙ（これらは互いに排反かつ網羅的な事象の集合を形成する）を取りうるとする。S を条件とした X の期待値は、第1の結果 X₁ に対してその条件付き確率 P(X₁ | S) を掛けたもの、第2の結果 X₂ に対してその条件付き確率 P(X₂ | S) を掛けたもの、というように加え合わせて次のように表される。

E(X | S) = P(X1 | S)X1 + P(X2 | S)X2 + … + P(Xn | S)Xn.4

この式は後ほど説明する。

条件付き確率を用いて無条件確率を表す全確率の法則（total probability rule）と同様に、条件付き期待値を用いて（無条件の）期待値を表す原理がある。この原理は期待値に関する全確率の法則（total probability rule for expected value）である。

期待値に関する全確率の法則（Total Probability Rule for Expected Value）

その式は次のとおりである。

E(X) = E(X | S)P(S) + E(X | SC)P(SC),5

ここで Sᶜ は「Sの補集合（complement of S）」であり、事象またはシナリオ「S」が発生しないことを意味する。

E(X) = E(X | S1)P(S1) + E(X | S2)P(S2) + … + E(X | Sn)P(Sn),6

ここで S₁, S₂, …, Sₙ は互いに排反かつ網羅的なシナリオまたは事象である。

一般形（式6）は、Xの期待値が、シナリオ1が与えられた場合のXの期待値 E(X | S₁) にシナリオ1の確率 P(S₁) を掛けたもの、シナリオ2が与えられた場合のXの期待値 E(X | S₂) にシナリオ2の確率 P(S₂) を掛けたもの、というように加え合わせたものであることを示している。

この原理を用いるためには、確率変数の結果を理解するのに有用な、互いに排反かつ網羅的なシナリオを設定する。このアプローチは、例1におけるBankCorpのEPSの確率分布の構築において用いられており、以下で説明する。

  Example 2BankCorp’s Earnings per Share, Part 2

BankCorpの収益は金利に敏感であり、金利低下環境から恩恵を受けるとする。今期において、BankCorpが金利低下環境で事業を行う確率は0.60、金利が安定した環境で事業を行う確率は0.40とする（なお、金利上昇環境の可能性は無視できるほど小さいと評価する）。金利低下環境が発生した場合、EPSがUSD2.60となる確率は0.25、EPSがUSD2.45となる確率は0.75と見積もられる。ここで、金利低下環境の確率0.60に、金利低下環境下でEPSがUSD2.60となる確率0.25を掛けると0.15となり、これは図表1の表に示されたUSD2.60の（無条件）確率と一致する。同様に、0.60×0.75 = 0.45 は図表1におけるUSD2.45の確率と一致する。これらの確率は整合的である。図表2の確率ツリー図は残りの分析を示している。

 Exhibit 2BankCorp’s Forecasted EPS

金利低下環境は、USD2.60およびUSD2.45の結果に分岐するツリーのノードへと導く。式6を用いて、金利低下環境が与えられた場合の期待EPSは次のように求められる。

E(EPS | declining interest rate environment) = 0.25(USD2.60) + 0.75(USD2.45)
= USD2.4875

金利が安定している場合、

E(EPS | stable interest rate environment) = 0.60(USD2.20) + 0.40(USD2.00)
= USD2.12

例えば金利が安定しているという新たな情報が得られると、EPSの当初の期待値USD2.34をUSD2.12へと下方修正する。ここで、期待値に関する全確率の法則を用いると、

EEPS=EEPS | declining interest rate environmentP(declining interest rate environment) +EEPS | stable interest rate environmentP(stable interest rate environment)

よって、E(EPS) = USD2.4875(0.60) + USD2.12(0.40) = USD2.3405、すなわち約USD2.34となる。

この値は、例1において確率分布から直接計算されたEPSの期待値と一致する。確率が整合的でなければならないのと同様に、無条件および条件付きの期待値も整合的でなければならない。そうでなければ、我々の投資行動は他の投資家に利益機会を与える可能性がある。

まとめると、まず関心のある事象の結果に影響を与える要因またはシナリオを設定した。次にこれらのシナリオに確率を割り当て、それぞれのシナリオに条件づけられた期待値を形成した。その後、現在時点での期待値を逆算した。本問では、EPSが関心対象の事象であり、金利環境がEPSに影響を与える要因であった。

また、各シナリオのもとでのEPSの分散も計算できる。

σ2(EPS|declining interest rate environment)
= P(USD2.60|declining interest rate environment)
× [USD2.60 − E(EPS|declining interest rate environment)]2
+ P(USD2.45|declining interest rate environment)
× [USD2.45 − E(EPS|declining interest rate environment)]2
= 0.25(USD2.60 − USD2.4875)2 + 0.75(USD2.45 − USD2.4875)2 = 0.004219

同様に、σ²(EPS | 安定した金利環境) は次のように求められる。

= 0.60(USD2.20 − USD2.12)2 + 0.40(USD2.00 − USD2.12)2 = 0.0096

これらは条件付き分散（conditional variances）であり、金利低下環境が与えられた場合のEPSの分散、および金利安定環境が与えられた場合のEPSの分散である。無条件分散と条件付き分散の関係はやや高度なトピックである。重要な点は、(1) 分散は期待値と同様に無条件概念に対応する条件付き概念を持つこと、(2) 特定のシナリオのもとでのリスク評価に条件付き分散を利用できることである。

  Example 3BankCorp’s Earnings per Share, Part 3

BankCorpの例を続けると、今度は同社のコスト構造に注目する。BankCorpの営業費用について調査している単純な線形回帰モデル（simple linear regression model）は次のとおりである。

Y^=a+bX,

where Y^ is a forecast of operating costs in millions of US dollars and X is the number of branch offices; and Y^ represents the expected value of Y given X, or E(Y | X). You interpret the intercept a as fixed costs and b as variable costs. You estimate the equation as follows:

Y^=12.5+0.65X.

BankCorpは現在66の支店を有しており、この式により営業費用は 12.5 + 0.65(66) = USD55.4百万と推定される。成長に関しては2つのシナリオがあり、図表3のツリー図に示されている。

 Exhibit 3BankCorp’s Forecasted Operating Costs
1.

Compute the forecasted operating costs given the different levels of operating costs, using Y^=12.5+0.65X . State the probability of each level of the number of branch offices. These are the answers to the questions in the terminal boxes of the tree diagram.

解答を確認（Check Answer）

解答（Solution）

Using Y^ = 12.5 + 0.65X, from top to bottom, we have

営業費用（Operating Costs） 確率（Probability）
Y^ = 12.5 + 0.65(125) = USD93.75 million 0.80(0.50) = 0.40
Y^ = 12.5 + 0.65(100) = USD77.50 million 0.80(0.50) = 0.40
Y^ = 12.5 + 0.65(80) = USD64.50 million 0.20(0.85) = 0.17
Y^ = 12.5 + 0.65(70) = USD58.00 million 0.20(0.15) = 0.03
合計 = 1.00
2.

高成長シナリオにおける営業費用の期待値を計算せよ。また、低成長シナリオにおける営業費用の期待値も計算せよ。

解答を確認（Check Answer）

解答（Solution）

E(operating costs|high growth) = 0.50(USD93.75) + 0.50(USD77.50)

3.

ツリーの初期ボックスの問いを参照せよ：BankCorpの期待営業費用はいくらか。

解答を確認（Check Answer）

解答（Solution）

米ドル金額は百万単位である。

E(operating costs) = E(operating costs|high growth)P(high growth)
+ E(operatingcosts|low growth)P(low growth)
= 85.625(0.80) + 63.525(0.20) = 81.205

BankCorpの期待営業費用はUSD81.205百万である。

本節では、EPSのような確率変数を独立した量として扱ってきた。EPSの期待値や分散といった記述量が、他の確率変数の関数となり得ることについては検討していない。ポートフォリオのリターンは、ポートフォリオ内の個々の証券のランダムなリターンという他の確率変数の関数であることが明らかな確率変数の一例である。ポートフォリオの期待リターンおよびリターンの分散を分析するには、これらの量が個々の証券リターンの特性の関数であることを理解する必要がある。ポートフォリオのリターン分散を見ると、個々の証券リターンがどのように共に動くか、すなわち共分散（covary）が重要であることが分かる。ポートフォリオの期待リターン、リターン分散、そして重要な共分散（covariance）および相関（correlation）については、別の学習モジュールで扱う。

  Question Set

1.

デフォルトした債券の元本回収の見通しが、2つの経済シナリオのいずれが実現するかに依存するとする。シナリオ1の確率は0.75であり、元本USD1あたりUSD0.90を回収する確率が0.45、またはUSD0.80を回収する確率が0.55である。シナリオ2の確率は0.25であり、元本USD1あたりUSD0.50を回収する確率が0.85、またはUSD0.40を回収する確率が0.15である。

シナリオ1およびシナリオ2のデータを用いて、次を計算せよ。

  1. シナリオ1が与えられた場合の期待回収額を求めよ。
  2. シナリオ2が与えられた場合の期待回収額を求めよ。
  3. 期待回収額を求めよ。
  4. 確率ツリー図（probability tree diagram）で情報を図示せよ。
  5. 4つの回収額USD0.90、USD0.80、USD0.50、USD0.40それぞれの確率を求めよ。

解答を確認（Check Answer）

解答（Solution）

  1. E(recovery | Scenario 1) = 0.45(USD0.90) + 0.55(USD0.80) = USD0.845
  2. E(recovery | Scenario 2) = 0.85(USD0.50) + 0.15(USD0.40) = USD0.485
  3. E(recovery) = 0.75(USD0.845) + 0.25(USD0.485) = USD0.755
  4. 詳細説明の展開／折りたたみ（Expand / Collapse Extended Description） 確率0.75で最初の分岐は2つのノードに至る。ノード1は確率0.45であり、USD0.90の回収を示し、その確率は0.3375である。ノード2は確率0.55であり、USD0.80の回収を示し、その確率は0.4125である。2番目の分岐は2つのノードに至る。ノード1は確率0.85であり、USD0.50の回収を示し、その確率は0.2125である。ノード2は確率0.15であり、USD0.40の回収を示し、その確率は0.0375である。
  5. シナリオ1に関連する結果：シナリオ1のもとで、元本USD1あたりUSD0.90を回収する確率が0.45であり、シナリオ1自体の確率が0.75であるため、USD0.90を回収する確率は0.45(0.75) = 0.3375である。同様に、USD0.80を回収する確率は0.55(0.75) = 0.4125である。シナリオ2に関連する結果：シナリオ2のもとで、元本USD1あたりUSD0.50を回収する確率が0.85であり、シナリオ2自体の確率が0.25であるため、USD0.50を回収する確率は0.85(0.25) = 0.2125である。同様に、USD0.40を回収する確率は0.15(0.25) = 0.0375である。

ある事象が発生したという条件のもとで、別の事象の期待値。
