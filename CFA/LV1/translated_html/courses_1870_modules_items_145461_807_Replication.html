8.07 | Replication

モジュール8：オプションの価格付けと評価（Pricing and Valuation of Options）8.07｜レプリケーション（Replication）

 Learning Outcome

The candidate should be able to:

  • フォワード契約（forward commitments）および条件付請求権（contingent claims）の価格付けにおける裁定取引（arbitrage）とレプリケーション（replication）の概念の違いを対比せよ。

これまでのレッスンでは、原資産のロングまたはショートポジションと資金の借入または貸付を組み合わせることで、フォワード契約のキャッシュフローをレプリケート（replicate）する方法を示した。市場参加者がこのようなレプリケーション戦略を利用できることにより、一物一価の法則（law of one price）が成立し、無リスクの裁定利益機会は存在しなくなる。

これまでに示したように、コールオプション（call option）は、原資産価格の上昇に伴って価値が増加する点でロング・フォワードポジションに類似しているが、行使によって利益が生じる場合にのみ決済される点で異なる。この正の結果における見かけ上の対称性は、図表4（Exhibit 4）に示されており、ここでは行使価格（X）がフォワード価格（F₀(T)）と等しい。

 Exhibit 4Call Option versus Long Forward Position

上段のグラフでは、オプションのペイオフ線は原点から始まり、x軸に重なり、フォワードの利益線と交差して第1象限へ上昇する。オプションの利益線は第3象限の原点から始まり、X = F₀(T) に達した後、第1象限へ上昇する。下段のグラフでは、時間に対するペイオフが示されている。波状の曲線は原点（0,0）から始まり、時点Tの縦線上の点Xで終わる。Sₜ > X の場合はこの曲線の上側に位置し、X > Sₜ の場合は下側に位置する。

契約開始時（t = 0）にコールオプションをレプリケートするためには、無リスク金利（risk-free rate, r）で借入を行い、その資金で原資産を価格S₀で購入する必要がある。満期時（t = T）には、フォワード契約とは異なり、以下の2つの結果があり得る：

  • 行使（Sₜ > X）：原資産をSₜで売却し、その収益でXを返済する。
  • 不行使（Sₜ < X）：決済は行われない。

行使が確実であれば、フォワード契約と同様に、契約開始時に X(1 + r)^(−T) を借り入れる。しかし実際には不確実であるため、満期時における行使の可能性（オプションのマネーネス（moneyness）と密接に関連）に基づいて、その一部を借り入れる。オプションの非線形なペイオフ構造により、このレプリケーション取引は行使確率の変化に応じて調整が必要となるが、フォワード契約のレプリケーション取引は一定である。オプションのレプリケーションについては、後のレッスンでさらに詳しく扱う。

次に、プットオプションのレプリケーションを考える。これまでに、行使価格がフォワード価格（X = F₀(T)）と等しいショート・プットポジションは、満期時に原資産価格がフォワード価格を下回る場合（Sₜ < X = F₀(T)）にロング・フォワードポジションと同様の結果をもたらすことを示した。売却されたプットは原資産価格が下落すると価値が増加するが、行使によって利益が生じる場合にのみ決済される。この関係は図表5（Exhibit 5）に示されている。

 Exhibit 5Short Put Option versus Long Forward Position

Expand / Collapse Extended Description

上段のグラフでは、ペイオフ線は第3象限から始まりフォワードの利益線に重なりながら上昇し、x軸に重なる。利益線は第3象限から第1象限へと上昇し、オプションのペイオフがx軸と交差する点まで進んだ後、x軸と平行になる。下段のグラフでは時間に対するペイオフが示されている。波状の曲線は原点（0,0）から始まり、時点Tの縦線上の点X = F₀(T)で終わる。Sₜ > X の場合は曲線の上側に位置し、X > Sₜ の場合は下側に位置する。

したがって、ショート・プットオプションがレプリケーションのために原資産のロングポジションを必要とするのと同様に、契約開始時（t = 0）にロング・プットポジションをレプリケートするには、原資産を価格S₀で空売りし、その収益を無リスク金利rで運用する必要がある。満期時（t = T）には、フォワード契約とは異なり、以下の2つの結果があり得る：

  • 行使（Sₜ < X）：無リスク運用の資金を用いて、原資産をSₜで購入する。
  • 不行使（Sₜ > X）：決済は行われない。

行使が確実であれば、フォワード契約と同様に契約開始時に X(1 + r)^(−T) を貸し付けることになる。しかし、コールオプションと同様に、非対称なペイオフ構造により、行使確率に応じて時間とともに調整が必要となる。

次のレッスンでは、満期前のオプション行使の可能性を左右する要因について検討する。

  Question Set

裁定取引とレプリケーション（Arbitrage and Replication）

1.

次の文を完成させる正しい語句を選べ：コール価格の下限は、原資産価格からその________価格の現在価値を________したもの、またはゼロのいずれか大きい方である。

解答を確認（Check Answer）

解答（Solution）

コール価格の下限は、原資産価格から行使価格（exercise price）の現在価値を差し引いたもの、またはゼロのいずれか大きい方である。

2.

追加キャッシュフローのない株式を原資産とする6か月満期のヨーロピアン・コールオプションについて、行使価格（X）は50ポンド、初期の原資産価格（S₀）は49.75ポンド、無リスク金利（risk-free rate）は2%である。3か月後において Sₜ = 65ポンドである場合、コール価格の下限を求めよ。

解答を確認（Check Answer）

解答（Solution）

コールオプションの買い手は満期時に Sₜ > X の場合にのみ行使するため、3か月後の時点における下限は Sₜ − PV(X) またはゼロのいずれか大きい方である：

ct,Lower bound=Max0, St−X1+r−T−t

Sₜ = 65ポンド、X = 50ポンド、r = 2%、（T − t）= 0.25 を用いて、コールオプションの下限は次のように求められる：

ct,Lower bound=Max0, GBP65−GBP501.02−0.25
= GBP 15.25
3.

次のレプリケーション戦略に関する記述を、それに対応するデリバティブ商品と対応付けよ：

1. 時点 t = 0 において、無リスク金利で借入を行い、原資産を S₀ で購入する。 A. コールオプションとプットオプションの両方のレプリケーション戦略
2. 行使確率の変化に応じて時間とともに調整が必要である。 B. コールオプションのレプリケーション戦略
3. 時点 t = T において、貸付の返済を受け取り、原資産を Sₜ で購入する。 C. 行使された場合のプットオプションのレプリケーション戦略

解答を確認（Check Answer）

解答（Solution）

  1. Bが正しい。時点 t = 0 において、コールオプションのレプリケーション戦略では、無リスク金利で借入を行い、原資産を購入する。
  2. Aが正しい。コールオプションとプットオプションはいずれも非線形のペイオフ構造を持つため、行使確率の変化に応じてレプリケーション戦略は時間とともに調整が必要となる。
  3. Cが正しい。プットオプションのレプリケーション戦略（行使される場合）では、時点 t = T において貸付の返済を受け取り、その資金で原資産を Sₜ で購入する。
4.

条件付請求権（contingent claim）とフォワード契約（forward commitment）のレプリケーションの変化に関する主な違いを説明せよ。

解答を確認（Check Answer）

解答（Solution）

オプションは非対称なペイオフ構造を持ち、オプションの買い手は行使価値が正の場合にのみ行使する。そのため、オプションの行使確率は時間とともに変化し、オプション契約をレプリケートする取引は時間に応じて調整する必要がある。一方、フォワード契約は対称的なペイオフ構造を持ち、将来の特定時点で確実に決済されるため、そのレプリケーション取引は契約期間中に調整を必要としない。
