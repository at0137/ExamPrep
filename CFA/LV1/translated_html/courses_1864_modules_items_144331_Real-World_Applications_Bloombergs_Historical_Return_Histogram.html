Real-World Applications: Bloomberg's Historical Return Histogram

モジュール3：資産リターンの統計的尺度（Statistical Measures of Asset Returns） 実務応用（Real-World Applications）：Bloombergのヒストリカル・リターン・ヒストグラム（Bloomberg's Historical Return Histogram）

Bloombergは、金融プロフェッショナルが業務全体で使用する主要なマーケットデータプラットフォームの一つである。このオプションセクションでは、CFA Programカリキュラムの実務への応用をBloomberg上で探る。

{HRH} ヒストリカル・リターン・ヒストグラム（HISTORICAL RETURN HISTOGRAM）

BloombergのHRH機能は、任意のインデックスの価格リターン（price return）およびトータルリターン（total return）を可視化することを可能にする。選択期間における平均リターンは、画面左上の白い数値で表示される。

「HRH <GO>は、証券のヒストリカル・リターンの分布をヒストグラムで可視化することを可能にし、リターンが理論上の正規分布に従っているかどうかを確認できる。リターンの分布は、価格モデルの仮定の妥当性、すなわちモデルが現実にどの程度近いかを判断するのに役立つ。分析したい日付範囲や、リターン計算に使用する期間をカスタマイズできる。分布ヒストグラムに加えて、時系列チャートとしてデータを表示することもでき、分析期間中のデータの変化を検出することが可能である。」-Bloomberg

この機能を試すには、コマンドラインにHRHと入力し、Enterまたは<GO>を押す。画面左上のアンバー色のフィールドにインデックスを入力する；ここではS&P 500 Indexに対してSPXを使用している。左上の日付範囲を、例えば01/01/1960からのような長期系列に変更し、その後、画面左上付近の単一文字のアンバー色ドロップダウンを「D」から年次（Yearly）を表す「Y」に変更する。「Type」の銅色ドロップダウンを「Price Return」と「Total Return」の間で切り替え、Enterまたは<GO>を押すと、ヒストグラムがそれに応じて変化する。
