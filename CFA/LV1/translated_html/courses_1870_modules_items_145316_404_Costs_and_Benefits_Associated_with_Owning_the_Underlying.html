4.04 | Costs and Benefits Associated with Owning the Underlying

モジュール4：デリバティブ価格付けにおける裁定（arbitrage）、レプリケーション（replication）、およびキャリーコスト（cost of carry） 4.04｜原資産保有に伴うコストと便益（Costs and Benefits Associated with Owning the Underlying）

 Learning Outcome

The candidate should be able to:

  • 原資産のスポット価格と将来予想価格の違い、および原資産保有に伴うキャリーコストの違いを説明せよ

前のレッスンでは、現物取引とフォワード契約の関係を説明するためにレプリケーションが用いられた。追加的なキャッシュフローを持たない資産のスポット価格とフォワード契約との関係は、無リスク金利rによって結び付けられることが示された。本レッスンでは、特定期間における原資産保有に関連するコストと便益の純額としてのキャリーコストについて説明する。

前レッスンのフォワード契約の例では、原資産にコストや便益が存在しない場合、スポット価格とフォワード価格の間に以下の関係が成立することが示された。

F0(T) = S0(1 + r)T. 3

この関係は、連続複利のもとで式4として表される。

F0(T) = S0erT. 4

無リスク金利rは、投資家が資産を保有することによる機会費用（キャリーコスト）を表す。これは、投資家が資産購入のために借入を行うか否かに関わらず存在し、以下で扱うすべての資産クラスに適用される。

式3および式4は、追加的なキャッシュフローを持たない原資産という特別なケースを表している。しかし、多くの資産には保有に伴う追加的なコストや便益があり、スポット価格とデリバティブ価格の間で裁定機会が生じないように、これらはフォワード価格に反映されなければならない。図表7は、コストおよび便益（通常は配当または利息収入）がスポット価格、フォワード価格、および無リスク金利の関係に与える影響を示している。

 Exhibit 7Spot Prices, Forward Commitment Prices, and the Risk-Free Rate with Underlying Asset Costs and Benefits

Expand / Collapse Extended Description

原点から右上に伸びる直線があり、時間Tの位置でF₀(T)に達する。直線下の領域はrと表示される。F₀(T)より上ではコストが便益を上回り、F₀(T)より下では便益がコストを上回る。

原資産保有者が機会費用に加えてコストを負担する場合、その追加コストに対する補償としてフォワード価格F₀⁺(T)は高くなる。一方で、原資産保有者が収益やその他の便益を得る場合、これらはフォワード価格をF₀⁻(T)へと低下させる。

保有による便益または収入（I）やコスト（C）が、時点t = 0における現在価値（PV₀()）として既知の金額で表される場合、離散複利のもとでのスポット価格とフォワード価格の関係は次のように示される。

F0(T) = [S0 – PV0(I) + PV0(C)](1 + r)T. 5

他の場合では、追加的なコストや便益は契約期間にわたる収益率として表される。収入（i）およびコスト（c）が収益率として表される場合、連続複利のもとでのスポット価格とフォワード価格の関係は次の通りである。

F0(T) = S0e(r + c–i)T. 6

現在価値で表される場合でも収益率で表される場合でも、フォワード価格は原資産保有に関連するすべてのコストと便益の純効果を反映しなければならず、これには以下が含まれる。

  • 機会費用（無リスク金利r）：無リスク金利が正であれば、他の条件が一定である場合、フォワード価格はスポット価格より高くなる。また、無リスク金利が高いほど、その差は大きくなる。この機会費用はすべての資産に適用される。
  • その他の保有コスト（C, c）：物理的コモディティなどの原資産の保有者は、保管、輸送、保険、または劣化コストなどを負担する必要がある。将来引渡しの契約を行う保有者はこれらのコストの補償を期待するため、他の条件が一定であればフォワード価格はスポット価格より高くなる。
  • 保有による便益（I, i）：一方で、原資産の保有者は、その資産のデリバティブを保有する場合と比較して、キャッシュフローやその他の便益を享受する場合がある。将来引渡しのデリバティブ契約を結ぶ相手方はこれらの便益を享受できないため、その分フォワード価格は低下する。株式配当や債券クーポンはその例である。

図表8は、コストと便益が存在する場合のフォワード価格とスポット価格の関係を示している。例えば、機会費用およびその他のコストが便益を上回る場合、フォワード価格はスポット価格を上回る。

 Exhibit 8Forward vs. Spot Price

コストと便益の比較（Cost vs. Benefit） フォワード価格とスポット価格の関係（Forward vs. Spot Price）
機会費用およびその他のコスト ＞ 便益 F₀(T) ＞ S₀

各原資産の種類ごとにコストと便益は異なり、時間や市場によっても変化する。例えば、個別株式の保有者は定期的な配当という便益を受け取る場合がある（例4）が、そうでない場合もある。株価指数の場合、便益は通常収益率として表される（例5）。

  Example 4Hightest Equity Forward with Dividend

Hightest Capitalが、6か月後に合意価格でユニリーバ（Unilever, UL）の株式1,000株を金融仲介業者に引き渡すフォワード契約を締結すると仮定する。ULのスポット価格S₀は50ユーロ、配当はない（I = 0）、無リスク金利rは5%とする。このとき、式3を用いて6か月後のフォワード価格F₀(T)を求めることができる。

F0(T) = S0(1 + r)T.
EUR51.23 = EUR50(1.05)0.5.

次に、ユニリーバが四半期ごとに0.30ユーロの配当を支払うと仮定する。この配当は3か月後と時点Tの双方で支払われ、その他の条件は同じとする。式5を用い、PV₀(C) = 0として6か月後のF₀(T)を求める。

F0(T) = [S0 – PV0(I)](1 + r)T.

まず、1株当たり配当の現在価値PV₀(I)を次のように求める。

PV0(I) = EUR0.30(1.05)–0.25 + EUR0.30(1.05)–0.5.
EUR0.5892 = 0.2964 + 0.2928.
Substitute PV0(I) = EUR0.5892 into Equation 5 to solve for F0(T):
F0(T) = (EUR50 – EUR0.5892)(1.05)0.5
= EUR50.6310.

したがって、HightestのUL株1,000株に対するフォワード契約価格は50,631.00ユーロ（= 1,000 × 50.6310ユーロ）となる。無リスク金利での借入による機会費用は6か月で1株当たり1.23ユーロであり、ユニリーバが配当を支払わない場合にはF₀(T) − S₀に等しい。四半期配当0.30ユーロは、フォワード価格とスポット価格の差を約0.63ユーロに縮小させる。

 

  Example 5Stock Index Futures with Dividend Yield

Viswan Family Office（VFO）が、インド国立証券取引所で取引されるNIFTY 50株価指数を対象とする3か月フォワード契約を締結したいとする。NIFTY 50のスポット価格は15,200インドルピー、配当利回りは2.2%、無リスク金利は4%である。式6（c = 0）を用いてフォワード価格を求める。

F0(T) = S0e(r + c –i)T
= 15,200e(0.04 – 0.022)0.25
= INR15,268.55.

外国為替は、スポット価格とフォワード価格の無裁定条件を導く際にいくつかの調整が必要である。まず、「価格」という概念を区別することが重要である。株式、債券、コモディティでは、価格は1単位あたりの通貨額を指す。一方、為替レートはスポットレートS₀,f/dとして表され、分子に価格通貨（foreign currency, f）、分母に基軸通貨（domestic currency, d）を置き、基軸通貨1単位あたりの価格通貨の単位数を示す。例えば、USD/EURのスポットレートが1.20の場合、米ドルが価格通貨（f）、ユーロが基軸通貨（d）であり、USD1.20 = EUR1を意味する。

外国為替フォワード契約（FX forward contract）は、将来のある時点において、一方の通貨を売却し他方の通貨を購入する契約であり、契約時にフォワードレートF₀,f/dが決定される。ロングのFXフォワードポジションは、基軸通貨を購入し、価格通貨を売却することを意味する。例えば、USD/EURのロングフォワードは米ドルを売却しユーロを購入することである。

前のレプリケーションの例では、デリバティブのキャッシュフローは原資産のロングまたはショートポジションと現金の借入または貸付の組み合わせで再現された。ここでは、外国通貨と自国通貨それぞれに機会費用が存在し、それぞれ外国無リスク金利r_fと自国無リスク金利r_dで表される。

  Example 6AUD/USD Foreign Exchange Forward Replication

現在のスポットレートS₀,f/dに対して、一方の通貨のリターンを他方の通貨建てで再現するためには、「将来価格が既知の資産はその現在価値で取引されるべきである」という裁定概念を用いて、フォワードレートF₀,f/d(T)を求めることができる。

現在のAUD/USDスポットレートが1.3335であるとする。豪ドルが価格通貨（foreign currency）、米ドルが基軸通貨（domestic currency）であり、AUD1.3335 = USD1を意味する。6か月の豪ドル無リスク金利は0.05%、米ドル無リスク金利は0.20%とする。

  • 時点t = 0：0.20%の米ドル無リスク金利で6か月間1,000米ドルを借り入れる。AUD/USDスポットレート（S₀,f/d = 1.3335）で1,333.50豪ドルを購入する。受け取った1,333.50豪ドルを0.05%の豪ドル無リスク金利で6か月間貸し付ける。
  • 6か月後の時点 t = T において：オーストラリアドル貸付の元利合計1,333.83（= 1,333.50 e^(0.0005 × 0.5)）を受け取る。オーストラリアドルの受取額を S_T, f/d で米ドルに交換して米ドル借入を返済する。利息込みの米ドル借入1,001（= 1,000 e^(0.002 × 0.5)）を返済する。

時点Tにおけるステップ5の交換が、オーストラリアドル貸付の受取額（ステップ4）が米ドル借入（ステップ6）とちょうど相殺されるようなスポット価格 S_T, f/d で行われる場合、無リスク裁定機会は存在しない。S_T, f/d を求めるには、ステップ4のオーストラリアドル額をステップ6の米ドル額で割る。

ST,f/d = 1.3325 (= AUD1,333.83/USD1,001).

以下の図は、時点 t = 0 および時点 T におけるキャッシュフローを要約している：

Expand / Collapse Extended Description

t = 0 においてUSD1,000を売却する。これは、r_USD = r_d = 0.20%でUSD1を借り入れることに対応する。t = T において、USD1,000 e^r = USD1,001（= 1,000 e^(0.002 × 0.5)）を返済する。t = 0 においてAUD1,333.50を購入する。これは、r_AUD = r_f = 0.05%でAUD1,333.50を貸し付けることに対応する。t = T において、AUD1,333.50 e^r = AUD1,333.83（= 1,333.50 e^(0.00005 × 0.5)）を受け取る。これより1.3325と求められる。

FXスポット価格とフォワード価格の間で無裁定条件が成立するためには、時点TにUSD1を購入するために必要なオーストラリアドル額（F₀, f/d (T)）のスポット価格（S₀, f/d）が、将来価格の割引現在価値に等しくなければならない。ここで用いられる割引率は、以下の修正版の式4に示されるように、外国と国内のリスクフリーレートの差に関係する。

F0,f/d(T) = S0,f/de(rf–rd)T. 7

例6における F₀, AUD/USD (T) を以下のように求める：

F0,f/d(T) = 1.3325 (= 1.3335e(–0.0015 × 0.5)).

式7より、スポットとフォワードのFX価格関係を決定するのは金利の絶対水準ではなく、リスクフリー金利差（r_f – r_d）であることが分かる。例えば例6では、オーストラリアドルのリスクフリーレートは米ドルのそれより0.15%低い（r_f – r_d < 0）。より高い米ドル金利で借り入れ、より低いオーストラリアドル金利で貸し付けると、将来USD1を購入するのに必要なオーストラリアドルが少なくなるような無裁定フォワード価格が成立する。このため、オーストラリアドルは米ドルに対してフォワード市場でプレミアムで取引されていると言われる。図表9はフォワードとスポットのFX価格の関係を要約している。

次に、前のレッスンで扱ったMontau AGの例において、このFXスポットとフォワードの関係を検討する。

  Example 7Montau AG’s FX Forward Rate

以前のレッスンでは、ドイツの資本財メーカーであるMontau AGが紹介された。Montauは、韓国のメーカーであるJeon, Inc.と、75日後に6億5,000万ウォン（KRW650 million）でレーザー切断機を納入する商業契約を締結する。Montauは、国内で発生するユーロ建てコストと、機械の納入およびスポットFX市場で韓国ウォンをユーロに交換することで実現するユーロ収益との間にタイミングのミスマッチに直面する。このKRW/EUR為替レートのミスマッチを解消するため、MontauはKRW/EURのFXフォワード契約のロングポジションを取る。すなわち、Montauは75日後に固定価格F₀(T)で韓国ウォンを売却しユーロを購入することに合意する。

このMontau AGの例では、本日のスポット為替レートと国内（r_d）および外国（r_f）のリスクフリーレートを用いて、KRW/EURのフォワードレートを求める。式7に示されるように、スポットとフォワードのFXレートの差はリスクフリーレートの差に依存する。KRW/EURの場合、韓国ウォンは価格通貨（price currency）または外国通貨であり、したがってr_fは韓国ウォンの金利である。ユーロはベース通貨（base currency）または国内通貨である。したがって、式7は以下のように書き換えられる。

F0,KRW/EUR(T) = S0,KRW/EURe(rKRW–rEUR)T.

スポットのKRW/EURレート（S₀）を1,300（すなわちKRW1,300 = EUR1）、韓国ウォンのリスクフリーレートを0.75%、ユーロのリスクフリーレートを−0.25%（r_d）と仮定する。無裁定と整合的な75日後のKRW/EURフォワードレートを計算せよ。

F0,KRW/EUR(T) = 1,300 × e(0.0075 + 0.0025) × (75/365).

KRW/EURフォワードレートを解くと、F₀(T) = 1,302.67が得られる。韓国ウォンとユーロの金利差が1%であることにより、フォワード価格がスポット価格を上回る（F₀, f/d(T) > S₀, f/d）ことに注目せよ。すなわち、6か月後には1ユーロを購入するためにより多くの韓国ウォンが必要となり、韓国ウォンはユーロに対してフォワードディスカウント（forward discount）で取引されていると言われる。

フォワードレートとスポットレートの間の無裁定条件を示すために、MontauがKRW650百万を1,300 KRW/EURのスポットレートでユーロに交換し、EUR500,000（= KRW650,000,000 / 1,300）を受け取り、これを連続複利（continuously compounded）r_f = −0.25%で75日間運用すると仮定する：

EUR499,743.22 = EUR500,000 × e[–0.0025 × (75/365)].

Jeon Inc.がKRW650百万を韓国ウォンのリスクフリーレート（r_d = 0.75%）で75日間運用し、以下を受け取ると仮定する。

KRW651,002,484.59 = KRW650,000,000 × e[0.0075 × (75/365)].

したがって、裁定機会のないフォワード契約価格は、Montauが75日後にKRW651,002,484.59をEUR499,743.22に交換できるものでなければならない。

F0(T) = 651,002,484.59/499,743.22 = 1,302.67.

これは、2つの価格（スポットおよびフォワード為替レート）が、2つの異なる通貨におけるリスクフリー金利差と整合的であることを確認している。

証券や電子的に保管される現金とは対照的に、コモディティ（commodities）は通常、保管、保険、輸送、および（ソフトコモディティの場合）腐敗といった、これらの実物資産に関連する既知のコストを伴う。現物コモディティを保有することによる非金銭的な便益は、利便性利回り（convenience yield）として知られる。現物市場では、経済状況によって市場参加者が現物コモディティの保有を好む場合がある。単純な例として、原油在庫が非常に少ない場合、精製業者はスポットの原油価格を押し上げ、フォワード価格が保管コストや金利を十分に反映しないことがある。以下の例では、これらのキャリーコスト（carry costs）がコモディティのスポット価格とフォワード契約価格の関係に与える影響、および利便性利回りの可能性を示す。

  Example 8Procam’s Gold Forward Contract with Storage Costs

以前の例を思い出すと、Procamは無リスク利益を得るために、スポット市場で金を借り入れて購入し、同時に3か月のフォワード市場で金を売却した。リスクフリーレートが2%であると仮定すると、保管コストがゼロのもとで、与えられたフォワード価格USD1,792.13に対して、以前の裁定機会を排除するためには、スポット金価格（S₀）はUSD1,783.28（= USD1,792.13(1.02)^(-0.25)）まで上昇する必要があることを示した。

スポット価格がUSD1,783.28であるとき、契約終了時に支払われる1オンスあたりUSD2の保管および保険コストを導入した場合、無裁定条件を満たすためにフォワード金価格はどのように変化するか。

既知の保管コストを伴うコモディティのフォワード契約価格は、式5を用いて求めることができる（ここで PV₀(I) = 0）：

F0(T) = [S0 + PV0(C)](1 + r)T.

まず、1オンスあたりの保管コストの現在価値 PV₀(C) を以下のように求める：

PV0(C) = USD2(1.02)–0.25
= USD1.99.

PV₀(C) = USD1.99 を式5に代入して、F₀(T) を求める：

F0(T) = (USD1,783.28 + USD1.99)(1.02)0.25
= USD1,794.13.

保管および保険コストの追加により、スポット価格とフォワード価格の差が拡大することに注意せよ。最後に、無裁定価格を大きく下回るフォワード価格 F₀(T) は、利便性利回り（convenience yield）の存在を示唆する可能性があることに留意せよ。

  Question Set

キャリーコスト（Cost of Carry）

1.

原資産の便益が機会費用およびその他の保有コストを上回る場合の、スポット価格とフォワード価格の関係を説明せよ。

解答を確認（Check Answer）

解答（Solution）

原資産の保有者にとっての便益が、機会費用およびその他の保有コストを上回る場合、スポット価格はフォワード価格よりも高くなる。

2.

以下のスポット価格とフォワード価格の関係に対応する例をそれぞれ特定せよ：

1. F₀(T) > S₀ A. クーポンがリスクフリーレートを上回る、パーで価格付けされた固定利付債（fixed-coupon bond priced at par）
2. F₀(T) < S₀ B. 外国のリスクフリーレートが国内のリスクフリーレートより高い外国為替フォワード（foreign currency forward）
3. F₀(T) と S₀ の関係を判断するための情報が不十分 C. 利便性利回り（convenience yield）および保管・保険コストを伴うコモディティ（commodity）

解答を確認（Check Answer）

解答（Solution）

1. Bが正しい。外国のリスクフリーレートが国内のリスクフリーレートより高い場合、FXフォワードレートはスポットレートより大きくなる。

2. Aが正しい。パーで価格付けされた固定利付債は、リスクフリーレートという機会費用を上回る収益をもたらすため、F₀(T) < S₀ となる。

3. Cが正しい。この例では、現物コモディティを保有することによる便益（利便性利回り）とコスト（リスクフリーレート、保管費、保険費）を完全に評価するための情報が不足している。

3.

空欄を埋めよ：正のリスクフリーレートは、他の条件が一定であれば、フォワード価格を原資産のスポット価格よりも_______させる。また、リスクフリーレートが高いほど、両者の差は_______なる。

解答を確認（Check Answer）

解答（Solution）

正のリスクフリーレートは、他の条件が一定であれば、フォワード価格を原資産のスポット価格よりも高くする。また、リスクフリーレートが高いほど、両者の差は大きくなる。

4.

あるアナリストは、現在のスポットMXN/USD為替レートが19.50、メキシコペソの6か月リスクフリーレートが4%、米ドルの6か月リスクフリーレートが0.25%であることを観察した。MXN/USDのスポットと6か月フォワードレートの関係を説明し、その理由を述べよ。

解答を確認（Check Answer）

解答（Solution）

MXN/USDのFXスポットとフォワード価格の関係は、メキシコペソと米ドルのリスクフリー金利差に依存する（r_f – r_d > 0、なぜなら3.75% = 4.0% − 0.25%）。メキシコペソの金利が米ドルより3.75%高いため、高いメキシコペソ金利で借り入れ、低い米ドル金利で貸し付けると、将来USD1を購入するのにより多くのメキシコペソが必要となるような無裁定フォワード価格が成立する。このため、メキシコペソは米ドルに対してフォワード市場でディスカウント（discount）で取引される（F₀, MXN/USD (T) > S₀, MXN/USD）。

これは、MXN/USDの場合について式7を用いて以下のように示すことができる：

F0,MXN/USD(T) = S0,MXN/USDe(rMXN–rUSD)T.

現在の S₀, f/d = 19.50、r_f = 4%、r_d = 0.25%、T = 0.5 に基づき、無裁定フォワード価格 F₀, f/d は次のように求められる：

F0,MXN/USD(T) = 19.50e[(0.04 – 0.0025) × 0.5]
= 19.8691.
5.

石油の保管および保険に関連する新たでより厳しい環境規制により、これらのサービスのコストが急激に上昇すると仮定する。これらのコスト増加が、石油のスポット価格とフォワード契約価格の関係に与える影響を説明せよ。

F0(T) = [S0 – PV0(I) + PV0(C)] (1 + r)T. 8
