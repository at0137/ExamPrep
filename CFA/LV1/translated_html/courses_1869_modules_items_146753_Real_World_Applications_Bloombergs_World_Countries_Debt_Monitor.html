Real World Applications: Bloomberg's World Countries Debt Monitor

モジュール15：政府発行体向け信用分析 実務での応用：Bloomberg(ブルームバーグ) の World Countries Debt Monitor(世界各国債務モニター)

Bloomberg(ブルームバーグ) は、金融専門家がワークフロー全体で世界的に使用する主要な市場データプラットフォームのひとつです。この任意のセクションでは、CFAプログラムカリキュラムの実務応用をBloomberg(ブルームバーグ) 上で探ります。

{WCDM} WORLD COUNTRIES DEBT MONITOR(世界各国債務モニター)

Bloomberg(ブルームバーグ) の WCDM 関数は、主権発行体の信用力を評価することを可能にします。

 

「WCDM は包括的な市場監視ツールであり、国際債務市場における透明性を提供するため、トレンドや異常値を容易に特定し、それらが投資およびリスク戦略にどのように影響するかを評価できます。WCDM は各国のマクロ状況および財務状況を表示し、未払い債務の額、信用保護購入コスト、信用格付け、為替レート、金利、最新の国内総生産(GDP)データなどの情報を含みます。」 – Bloomberg(ブルームバーグ)

 

この機能を試すには、Bloomberg(ブルームバーグ) のコマンドラインに WCDM と入力して Enter または <GO> を押します。画面右上の GDP(国内総生産) のアンバードロップダウンフィールドをクリックし、「Debt as % of GDP(債務/GDP比)」を選択します。表の最右列に世界の各経済の債務/GDP比が表示されます。また、掲載されている政府の5年間の CDS(クレジット・デフォルト・スワップ) スプレッドや信用格付けも確認できます。
