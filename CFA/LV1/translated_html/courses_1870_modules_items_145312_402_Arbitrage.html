4.02 | Arbitrage

モジュール4：デリバティブ価格付けにおける裁定（arbitrage）、レプリケーション（replication）、およびキャリーコスト（cost of carry） 4.02｜裁定（Arbitrage）

 Learning Outcome

The candidate should be able to:

  • デリバティブの価格付けにおいて、裁定およびレプリケーションの概念がどのように用いられるかを説明せよ

市場効率性に関する以前のレッスンでは、取引コストが存在しない場合、市場価格は無リスク利益、すなわち裁定の可能性を許さないべきであることが示された。最も単純な形では、「一物一価の法則（law of one price）」が成立しない場合、すなわち同一の資産が同時に異なる場所で異なる価格で取引されている場合に裁定機会が生じる。

原資産価格に関連する将来キャッシュフローから価値が導かれるデリバティブ契約の場合、裁定機会は、①同一の将来キャッシュフローを持つ2つの資産が異なる価格で取引されている場合、または②将来価格が既知の資産が、その将来価格を適切な割引率で現在価値に割り引いた価格で取引されていない場合に生じる。

同一の将来キャッシュフローを持つ資産が異なる価格で取引される最初のケースは、図表1で示されている。

Different Prices

t = 0 において、S₀^A = EUR99 < S₀^B = EUR99.15。資産AをS₀^Aで購入し、資産BをS₀^Bで売却する。CF₀ = (S₀^B − S₀^A) = (EUR99.15 − EUR99) = +EUR0.15 > 0。t = T において、S_T^A = S_T^B = EUR100。資産AをS_T^Aで売却し、資産BをS_T^Bで購入する。CF_T = (S_T^A − S_T^B) = (EUR100 − EUR100) = 0。

例えば、これら2つの資産が同一の特徴と同一の発行体を持つゼロクーポン債（zero-coupon bonds）であると仮定する。両債券は同じ将来日に満期を迎え、額面の支払いを行い、現在から満期までのデフォルトリスクも同一である。

  1. 債券Aの価格は時点t = 0でEUR99である（S₀^A = EUR99）。
  2. 債券Bの価格は時点t = 0でEUR99.15である（S₀^B = EUR99.15）。
  3. 両債券の将来価格はEUR100である（S_T^A = S_T^B = EUR100）。

この状況は投資家にとって裁定機会を意味する。

  • 時点t = 0において、投資家は次のことができる：債券Bを空売りしてEUR99.15を受け取り、債券AをEUR99で購入する；t = 0においてEUR0.15の純キャッシュインフローを得る。
  • 時点t = T（両債券の満期時）において、投資家は債券AからEUR100を受け取り、それを用いてEUR100で債券Bを購入し、空売りポジションを解消する。時点Tにおける相殺されたキャッシュフローにより、投資家は時点0における債券AとBの価格差EUR0.15の無リスク利益を得る。

この価格の不一致に気付いた他の投資家も、t = 0において債券Bを売却して価格を押し下げ、債券Aを購入して価格を押し上げることで無リスク利益を得ようとするため、価格は収束する。両債券の価格が等しくなる（S₀^A = S₀^B）と、裁定機会は消滅する。

デリバティブに関連する第2の裁定機会は、将来価格が既知の資産が、その将来価格の現在価値（present value, PV）で取引されていない場合に生じる。以前の時間価値（time value of money）のレッスンでは、現在価値と将来価値の計算における離散複利（discrete compounding）と連続複利（continuous compounding）の違いが示された。単一キャッシュフローの将来価値は、離散的な期間数に基づいて、式1の一般式に従う。

FVN = PV(1 + r)N ,1

ここで、rは各期間の利率、Nは複利計算期間の数である。連続複利は、期間の長さがゼロに近づき、年間の期間数が無限大に近づく場合であり、自然対数（natural logarithm）を用いて式2のように計算される。

FVT = PVerT . 2

デリバティブはどちらの方法でも価格付け可能であるが、本レッスンおよび後続のレッスンでは、個別の原資産については離散複利を用いる。一方で、株式指数、債券指数、コモディティ指数、クレジット指数などのポートフォリオ型原資産や、2通貨の金利が関与する外国為替の場合には、連続複利がより適切である。資産保有に伴う追加的なコストや便益を無視すると、適切な割引率rは無リスク金利であり、これは例1で示される。

  Example 1Spot vs. Discounted Known Future Price of Gold

以前のレッスンでは、プロカム・インベストメンツ（Procam Investments）が3か月後に1オンスあたり1,792.13米ドルで金100オンスを購入する契約を締結した。本例では、スポット価格と割り引かれた将来価格の不一致に基づき、プロカムは逆の取引を行う。現在の金スポット価格（S₀）は1オンスあたり1,770米ドルであり、年率無リスク金利（r）は2%とする。本例では、プロカムは無リスク金利で借入可能であり、金の保管コストはゼロと仮定する。この条件のもとで、プロカムがどのように無リスク利益を得るかを示す。

  • 時点t = 0：
  • プロカムは、2.0%の金利で3か月間177,000米ドルを借り入れ、現在のスポット価格で金100オンスを購入する。
  • プロカムは、3か月後に1オンスあたり1,792.13米ドルで金100オンスを売却するフォワード契約を締結する。

Expand / Collapse Extended Description

t = 0 において、S₀ < S_T (1 + r)^{-T}。S₀を金利rで借り入れて原資産を購入し、将来価格S_Tで売却する契約を結ぶ。具体的には、2%で1,770米ドルを借り入れて金を購入し、1オンスあたり1,792.13米ドルで売却する契約を結ぶ。t = T において、S₀(1 + r)^T < S_T。1オンスあたり1,792.13米ドルで売却し、借入元利S₀(1 + r)^T = 1,778.78米ドルを返済する。1オンスあたり13.35米ドルの無リスク利益が得られる。CF_T = (1,792.13 − 1,778.78) > 0。

  • 時点t = T（3か月後）：
  • プロカムはフォワード契約に基づき金100オンスを引き渡し、179,213米ドル（= 100 × 1,792.13米ドル）を受け取る。

  • プロカムは元本および利息を含めて借入金を返済する。

    USD177,878.44 = USD177,000(1.02)0.25.
  • 時点Tにおけるプロカムの無リスク利益は、フォワード売却収入と借入元利の差であり、1,334.56米ドル = 179,213米ドル − 177,878.44米ドルである。

例1では、金のスポット価格S₀は、3か月後の既知の将来価格の現在価値を下回っている（S₀ < S_T(1 + r)^{-T}、すなわち1,770米ドル < 1,783.28米ドル）。プロカムは、現物市場で借入して金を購入し、同時にフォワード市場で合意価格で金を売却することにより、1オンスあたり13.35米ドルの無リスク利益を得る。他の投資家がこの機会を認識して取引を行うにつれて、スポット価格は上昇（フォワード価格は下落）し、スポット価格が既知の将来価格の割引現在価値（S₀ = S_T(1 + r)^{-T}）と等しくなるまで、この裁定機会は消滅すると考えられる。

追加的なキャッシュフローを持たない原資産に対してデリバティブを価格付けする際に用いられる2つの重要な裁定概念は、以下の通りまとめられる。

  • 同一の資産、または同一のキャッシュフローを持つ資産は、同時点で同一の価格で取引されなければならない（S₀^A = S₀^B）。
  • 将来価格が既知の資産は、その将来価格を無リスク金利で割り引いた現在価値と等しいスポット価格を持たなければならない（S₀ = S_T(1 + r)^{-T}）。

これらの裁定条件は、図表2に示されているように、スポット価格、フォワード契約価格、および無リスク金利の関係を規定する。

 Exhibit 2Spot Prices, Forward Commitment Prices, and the Risk-Free Rate

図表2は、追加的な収益やコストがない資産について、離散複利のもとでr > 0の場合を示している。一定の期間Tにおいて、無リスク金利rが高いほど、フォワード価格はスポット価格に対して相対的に高くなる。また、一定の無リスク金利rのもとでは、Tが長くなるほどフォワード価格はスポット価格に対して相対的に上昇する。なお、多くの市場参加者にとって関連する無リスク金利は、以前のレッスンで導入されたレポレート（repo rate）であり、これは高流動性証券を担保とした借入金利である。

以前のレッスンで、フォワード契約は対称的なペイオフ構造を持つことを確認した。すなわち、時点Tにおいて、取引はフォワード価格F₀(T)と原資産価格S_Tの差、すなわち売り手の観点ではF₀(T) − S_Tに基づいて決済される（図表3参照）。

 Exhibit 3Forward Commitment Seller Payoff Profile

フォワードの売り手は、原資産を市場価格S_Tより低い事前合意価格F₀(T)で引き渡すことができる場合に利益を得る。例1では、プロカムはS_Tの現在価値よりも低いスポット価格で原資産を購入するために借入を行い、無リスク利益を確定させている。ここまではフォワード価格F₀(T)を所与として扱ってきたが、以下では、裁定条件を用いてフォワード契約の完全に相殺されるポジションをレプリケーション（replication）することにより、スポット価格とフォワード価格の関係をどのように導出できるかを示す。
