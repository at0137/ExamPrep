3.05 | Correlation between Two Variables

モジュール3：資産リターンの統計的測定 3.05｜2変数間の相関（Correlation between Two Variables）

 Learning Outcome

The candidate should be able to:

  • 投資問題に対処するために、2つの変数間の相関を解釈する

散布図（Scatter Plot）

散布図（scatter plot）は、2つの変数間の潜在的な関係を表示し理解するための有用なツールである。あるアナリストが、特定の5年間における情報技術（information technology, IT）セクターと公益事業（utilities）セクターの市場指数に対する相対的パフォーマンスに関心を持っているとする。このアナリストは、調査期間中の各月について、各セクターおよび市場指数のリターンを取得している。図表24はITセクター指数とS&P500指数（S&P 500）のリターンの散布図を示し、図表25は公益事業セクター指数とS&P500のリターンの散布図を示している。

データ点が密集している（疎である）ほど、2つの変数間の関係は強い（弱い）可能性を示す。

 Exhibit 24Scatter Plot of Information Technology Sector Index Return versus S&P 500 Index Return

 Exhibit 25Scatter Plot of Utilities Sector Index Return versus S&P 500 Index Return

散布図は単純な構造であるにもかかわらず、有益な情報を提供する。まず、2つの変数間に関連性があるかどうかを確認することが重要である。散布図のパターンは、関係がない、線形関係がある、または非線形関係があることを示す場合がある。さらに、データ点が線の周りにどれだけ密集しているかによって、関係の強さを判断できる。

図表24を見ると、データ点が正の傾きを持つ直線上に密集しているため、ITセクターのリターンはS&P500指数のリターンと強い正の関係を持つことが分かる。一方、図表25では、公益事業セクターとS&P500指数の関係は異なり、データ点に明確なパターンが見られず、これらの変数間に明確な関係がないことを示している。

次に、各軸の端に位置するデータ点（最大値や最小値）を観察することで、データの範囲を把握できる。さらに、変数間に関係がある場合、散布図を用いることで極端な値（外れ値（outliers））を特定できる。例えば、図表24では矢印で示された外れ値が容易に識別できる。このような極端値を特定し適切に処理することは、金融モデリングにおいて重要である。

散布図は、2変数間のパターンの発見、データ範囲の把握、外れ値の検出において強力なツールである。しかし、予測モデル構築のために多数の変数から特徴量選択を行う場合など、複数の変数間のペアごとの関係を調べる必要がある場合もある。

サンプル分散と標準偏差について理解したところで、散布図で視覚的に検討してきた2つの確率変数間の相関の概念を、より形式的に考察する。

共分散と相関（Covariance and Correlation）

相関（correlation）は、2つの確率変数間の線形関係の強さを測る指標である。ただし、2変数がどのように共に変動するかを考える第一歩は、共分散（covariance）を求めることである。

サンプル共分散の定義（Definition of Sample Covariance）：サンプル共分散（s_XY）は、サンプル内の2変数がどのように共に変動するかを測る指標である：

sXY=∑i=1nXi−X¯Yi−Y¯n−1.10

式10は、サンプル共分散が2つの確率変数（X_iおよびY_i）の観測値のサンプル平均からの偏差の積の平均であることを示している。これらがリターンであれば、単位はリターンの二乗となる。また、分母にn−1を用いることで、サンプル共分散が母共分散の不偏推定量となる。

簡単に言えば、共分散は2つの確率変数の同時変動の度合いを測る指標である。2変数が同じ方向に変動する場合（例えば、Yが平均を上回るときにXも平均を上回り、Yが平均を下回るときにXも平均を下回る場合）、共分散は正となる。逆方向に変動する場合は、共分散は負となる。

共分散の大きさだけでは解釈が難しい。これは単位が二乗であり、変数の大きさに依存するためである。そこで、共分散を標準化した指標である相関係数（correlation coefficient）を用いる。

サンプル相関係数の定義（Definition of Sample Correlation Coefficient）：サンプル相関係数（r_XY）は、2変数がどのように共に変動するかを標準化して表す指標であり、サンプル共分散を各変数の標準偏差の積で割ったものである：

rXY=sXYsXsY.11

相関係数は、2つの確率変数間の線形関係の強さを表す。

相関の性質（Properties of Correlation）

相関係数の性質について詳しく述べる：

  1. 相関は−1から+1の範囲をとる：

    −1 ≤ rXY ≤ +1.
  2. 相関が0（uncorrelated）の場合、2変数間に線形関係が存在しないことを示す。
  3. +1に近い正の相関は強い正の線形関係を示す。相関が1の場合は完全な線形関係を意味する。
  4. −1に近い負の相関は強い負（逆）の線形関係を示す。相関が−1の場合は完全な逆線形関係を意味する。

相関を視覚的に理解するために、再び散布図を用いる。相関係数は単一の数値で関係を表すのに対し、散布図は関係をグラフとして表す。そのため、散布図は相関係数の解釈に有用である。

図表26は散布図の例を示している。パネルAは相関+1の散布図であり、すべての点が正の傾きを持つ直線上に位置している。Xが1単位増加するとYは2単位増加する。すべての点が直線上にあるため、Xの水準に関わらずこの関係は成立する。傾きが異なっていても、すべての点が直線上にあれば相関は+1である。パネルBは相関−1の場合であり、点は負の傾きを持つ直線上にある。この場合、Xが1単位増加するとYは2単位減少する。

相関分析の限界（Limitations of Correlation Analysis）

図表26は相関が線形関係を測ることを示しているが、必ずしも信頼できるとは限らない。強い非線形関係があっても相関は低い場合がある。

外れ値が存在する場合も相関は不安定になる。外れ値はサンプルの極端な観測値であり、相関に大きな影響を与える。これらを除外すべきかどうかは、ノイズか有用な情報かを判断する必要がある。除外する場合は、トリミングやウィンザー化（winsorizing）を用いる。

重要なのは、相関は因果関係を意味しないことである。高い相関があっても、一方が他方の原因とは限らない。

また、散布図などの視覚化においても、因果関係を無意識に推測しないよう注意が必要である。

疑似相関（spurious correlation）とは以下を指す：

  • 特定のデータセットにおける偶然の関係による相関；
  • 2つの変数それぞれが第三の変数と混合されることによって生じる相関；
  • 2つの変数間に直接的な関係があるためではなく、第三の変数との関係から生じる相関。

偶然の関係の例として、2000年から2018年における米国のビール、ワイン、蒸留酒の月次小売売上と大気中の二酸化炭素濃度を考える。この2つの間の相関は0.824であり、正の関係が存在することを示している。しかし、大気中の二酸化炭素濃度がこれらの酒類の小売売上と関連していると考える理由はない。

第2のタイプの疑似相関（spurious correlation）の例として、相関のない2つの変数でも第三の変数で割ることで相関が生じる場合がある。例えば、企業の配当と総資産のクロスセクションデータを考える。これらの間の相関は低いかもしれないが、それぞれを時価総額（market capitalization）で割ると相関が高まる可能性がある。

第3のタイプの疑似相関の例として、身長と語彙力の間に正の相関がある場合があるが、実際の関係は年齢と身長、年齢と語彙力の関係である。

投資専門家は、高い相関に基づいて投資戦略を構築する際には注意が必要である。疑似相関は、一見すると有益に見えるが、実際には機能しない投資戦略を示唆する可能性がある。

さらに、相関はデータの全体像を示すものではない。例7で扱うアンスコムの四重奏（Anscombe’s Quartet）を考えると、平均、標準偏差、相関が同じであっても、異なるグラフが得られる。

  Example 7Anscombe’s Quartet

英国の統計学者フランシス・アンスコム（Francis Anscombe）は、要約統計量（平均、標準偏差、相関）だけではデータを十分に記述できないことを示すデータセットを作成した。彼は4つのデータセット（I、II、III、IV）を作成し、それぞれ2つの変数XとYを持ち、以下の特徴を持つ：

  • 各データセットのXは同じ平均と標準偏差（それぞれ9.00と3.32）を持つ。
  • 各データセットのYは同じ平均と標準偏差（それぞれ7.50と2.03）を持つ。
  • 各データセットのXとYは同じ相関（0.82）を持つ。
