2.07 | Calculation and Interpretation of Beta

モジュール2：ポートフォリオのリスクとリターン：パートII 2.07 | ベータ（beta）の計算と解釈

Learning Outcome

The candidate should be able to:

  • ベータを計算し、解釈せよ

まず、先に導入したシングルインデックスモデルを実現リターンを用いて再度表す：

Ri = (1 – βi)Rf + βi × Rm + ei

システマティックリスクは資産と市場との相関に依存するため、R_iとR_mの共分散からシステマティックリスクの尺度を導くことができる。ここで、R_iは上式で定義される。無リスク金利は定数であるため、R_iの第1項は消えることに注意する。

Cov(Ri,Rm) = Cov(βi×Rm+ei,Rm)=βiCov(Rm,Rm) + Cov(ei,Rm)=βiσm2+0

第1項はベータとR_mの分散の積である。誤差項は市場と無相関であるため、第2項は消える。したがって、式は以下のようにベータの形に書き換えられる：

βi=Cov(Ri,Rm)σm2=ρi,mσiσmσm2=ρi,mσiσm

上式はベータβ_iの表現を示しており、これは先に示したシングルインデックスモデルのファクターローディング（factor loading）に類似している。例えば、ある資産と市場の相関が0.70であり、それぞれのリターンの標準偏差が0.25および0.15である場合、その資産のベータは(0.70)(0.25)/0.15 = 1.17となる。もし資産と市場の共分散および市場分散がそれぞれ0.026250および0.02250で与えられている場合、計算は0.026250/0.02250 = 1.17となる。市場モデルにおけるベータは、資産iと市場との相関を考慮した調整を含んでいる。なぜなら、市場モデルはすべての資産を対象とするのに対し、CMLは完全に分散されたポートフォリオにのみ適用されるためである。

上式が示すように、ベータは資産のリターンが市場全体のリターンにどれだけ敏感であるかを示す尺度であり、資産iのリターンと市場リターンの共分散を市場リターンの分散で割ったものとして計算される。この表現は、資産と市場の相関と標準偏差の比（すなわち資産の標準偏差を市場の標準偏差で割ったもの）との積と等しい。前述のとおり、ベータは資産のシステマティックリスク、すなわち分散によって除去できないリスクの部分を捉える。ベータの計算に必要な分散および相関は通常、過去のリターンに基づいている。

正のベータは資産のリターンが市場全体の動きに従うことを示し、負のベータは資産のリターンが市場と逆方向に動く傾向を示す。言い換えれば、正のベータは資産のリターンが市場と同方向に動くことを意味し、負のベータは逆方向に動くことを意味する。無リスク資産のベータは、他の資産との共分散がゼロであるためゼロである。すなわち、ベータがゼロであるとは、その資産のリターンが市場の動きと無相関であることを示す。市場のベータは、分子のσ_iにσ_mを代入することで求められる。また、任意の資産はそれ自身と完全相関（1）であるため、市場のベータは1となる：

βi=ρi,mσiσm=ρm,mσmσm=1

市場のベータが1であるため、定義上、市場に含まれる株式の平均ベータは1となる。相関の観点では、特に先進国市場では、ほとんどの株式は市場と高い相関を持ち、0.70を超えることが多い。S&P500、Dow Jones 30、NASDAQ 100などの米国の広範な市場インデックスでは、相関は0.90を超えることもある。異なるセクター間の相関も高く、企業が同じ経済的・市場的変化に類似した反応を示すことを示している。その結果、実務的には、常に負のベータを持つ資産を見つけることはまれである。これは市場の広範な影響がすべての資産に及ぶためである。

ベータの推定（Estimation of Beta）

別のより実務的な方法として、上記で説明した市場モデルを用いてベータを直接推定する方法がある。市場モデル R_i = α_i + β_i R_m + e_i は回帰分析（regression analysis）を用いて推定される。回帰分析とは、ある変数（従属変数）と1つ以上の他の変数（独立変数）との関係を評価する統計的手法である。過去の証券リターン（R_i）と市場リターン（R_m）が、α_iおよびβ_iの推定に用いられる入力となる。

回帰分析は、資産リターンと市場リターンのすべての組み合わせ（R_i, R_m）をプロットし、それらの点からの二乗誤差の合計が最小となるような直線を引くことに類似している。図表6は市場モデルと推定されたパラメータを示している。切片α_i（定数項とも呼ばれる）と傾きβ_iがあれば、証券特性線（security characteristic line）を定義し、ベータを推定することができる。

図表6 証券リターンと市場リターンのプロットを用いたベータ推定（Beta Estimation Using a Plot of Security and Market Returns）

このグラフは、証券と市場の長期リターンを表す多数の散布点に対する最小二乗回帰直線である。縦軸は証券のリターン、横軸は対応する市場リターンである。この回帰直線は証券特性線と呼ばれ、その傾きは証券のベータの推定値であり、株式リターンが市場リターンに対して長期的にどれだけ敏感であるかを表す。直線の切片は証券のアルファ（alpha）と呼ばれる。

ベータの推定値は将来のリスク水準を予測する上で重要であるが、その正確性については多くの懸念がある。一般に、短期間（例：12か月）の推定は、資産の現在のシステマティックリスクにより近いベータを表す。一方で、短期間のベータ推定は、その期間中の特殊要因の影響を受ける可能性があるため、3〜5年の期間で測定されたベータよりも精度が低い。長期間の推定はより正確であるが、資産に大きな変化があった場合には将来の期待を適切に反映しない可能性がある。したがって、計算または回帰分析によって得られたベータ推定値は、現在または将来のシステマティックリスクを必ずしも正確に表すとは限らないことを認識する必要がある。

ベータと期待リターン（Beta and Expected Return）

シングルインデックスモデル（capital asset pricing model, CAPM）については次節で詳しく説明するが、本節では資産のベータに基づいてリターンを推定するためにCAPMを用いる。CAPMは通常、無リスク金利を右辺に置いて次のように表される：

E(Ri) = Rf + βi[E(Rm) – Rf]

このモデルは、証券の期待リターンの主な決定要因がベータ、すなわちその証券が市場とどの程度相関しているかであることを示している。資産のベータが高いほど、その期待リターンも高くなる。ベータが1より大きい資産は市場リターンより高い期待リターンを持ち、ベータが1未満の資産は市場リターンより低い期待リターンを持つ。

特定の場合には、無リスク利子率よりも低いリターンが要求される資産も存在する。例えば、資産のベータが負である場合、要求リターンは無リスク利子率より低くなる。このような資産は市場と組み合わせることでポートフォリオ全体のリスクを低減するため、非常に価値がある。保険（insurance）はその一例である。保険は、被保険者の資産が災害などによって減少した場合に正のリターンをもたらす。一方、そのような損失が発生しない場合や資産が増加している場合には、保険料の支払いが必要となる。したがって、保険は負のベータと負の期待リターンを持つが、全体のリスクを低減する役割を果たす。

例6 期待リターンの計算（Calculation of Expected Return）

石炭生産企業であるAlpha Natural Resources（ANR）は、中国の非上場の大規模石炭企業を買収した。このクロスボーダー買収の結果、ANRのリターンの標準偏差は50%から30%に低下し、市場との相関は0.95から0.75に低下したとする。市場の標準偏差とリターンはそれぞれ25%および10%で変わらず、無リスク利子率は3%とする。

1.

買収前のANR株のベータと期待リターンを計算せよ。

解答を確認（Check Answer）

Solution

β_iの式を用いてβ_iを計算し、その後リターンを計算する。

βi=ρi,mσiσm=0.95×0.500.25=1.90
E(Ri) = Rf + βi[E(Rm) – Rf] = 0.03 + 1.90 × (0.10 – 0.03) = 0.163 = 16.3%
2.

買収後の期待リターンを計算せよ。

解答を確認（Check Answer）

Solution

買収後の相関とリスクを用いて、同様の手順に従う。

βi=ρi,mσiσm=0.75×0.300.25=0.90
E(Ri) = Rf + βi[E(Rm) – Rf] = 0.03 + 0.90 × (0.10 – 0.03) = 0.093 = 9.3%

市場リスクプレミアムは7%（10% − 3%）である。ベータが変化すると、証券の期待リターンの変化は市場リスクプレミアムにベータの変化を掛けたものとなる。このシナリオでは、ANRのベータは1.0低下したため、新しい期待リターンは7パーセントポイント低下する。

Miles氏（Mr. Miles）は、iPodやiPhoneの強い需要を観察し、Apple株に投資したいと考えている。しかしながら、彼は投資からどの程度のリターンを期待すべきか分からない。無リスク利子率3%、市場リターン10%、Appleのベータ1.5が与えられている。

3.

Appleの期待リターンを計算せよ。

解答を確認（Check Answer）

Solution
E(Ri) = Rf + βi[E(Rm) – Rf] = 0.03 + 1.5 × (0.10 – 0.03) = 0.135 = 13.5%
4.

同じ情報を見たアナリストは、Appleの過去のパフォーマンスは将来を代表していないと判断した。Appleの時価総額の増加を踏まえると、Appleは以前より市場に近い動きをするようになっており、そのベータは1.1に近いと考えた。この場合、アナリストが考えるApple株の期待リターンはいくらか。

解答を確認（Check Answer）

Solution
E(Ri) = Rf + βi[E(Rm) – Rf] = 0.03 + 1.1 × (0.10 – 0.03) = 0.107 = 10.7%

この例は、過去リターンの推定と将来への予測との間の関連性の弱さを示している。投資家は、将来リターンの推定に過去リターンを用いることの限界を認識すべきである。

投資またはポートフォリオが市場全体の動きに対してどれだけ敏感であるかを示す尺度。

任意の資産（またはポートフォリオ）の期待リターンを、市場ポートフォリオに対するベータの線形関数として表す方程式。
