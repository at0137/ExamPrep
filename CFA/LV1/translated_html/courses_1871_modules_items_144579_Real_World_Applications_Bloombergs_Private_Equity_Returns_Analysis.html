Real World Applications: Bloomberg's Private Equity Returns Analysis

モジュール2：オルタナティブ投資のパフォーマンスとリターン 実務応用：Bloombergのプライベートエクイティ・リターン分析（Bloomberg's Private Equity Returns Analysis）

Bloombergは、金融の専門家が業務全体を通じて世界的に利用している主要な市場データプラットフォームの一つである。このオプションセクションでは、CFAプログラムのカリキュラムをBloomberg上でどのように実務に適用するかを探る。

{PEBM} プライベートエクイティ・リターン分析（PRIVATE EQUITY RETURNS ANALYSIS）

BloombergのPEBM機能は、世界中のファンドについて、戦略、ヴィンテージ（vintage）、地域ごとに、さまざまなリターン指標（IRR、MOIC、DPI、PIC、RVPI）を閲覧することを可能にする。

 

「PEBM <GO> は、純内部収益率（net IRR）や投下資本倍率（multiple of invested capital）などの個別ファンド特性に基づいてプライベートエクイティ・ファンドを比較できるため、同一のヴィンテージ年（vintage year）および／または戦略に属するファンドの文脈でパフォーマンスを分析することができる。PEBMはコントロールエリアと2つのセクションで構成されており、分析対象のファンドの選択と、四分位ランキングチャート（Quartile Ranking chart）で比較するファンドのリストを選択できる。チャート内のファンドブロックをクリックすると、画面下部にヴィンテージまたは戦略別にグループ化されたファンドの期間別パフォーマンス順位が表示される。」—Bloomberg

 

この機能を利用するには、BloombergのコマンドラインにPEBMと入力し、Enterまたは<GO>を押す。
