5.03 | Forecasting Correlation of Returns: Covariance Given a Joint Probability Function

モジュール5：ポートフォリオ数学（Portfolio Mathematics）5.03｜リターンの相関の予測：同時確率関数による共分散（Forecasting Correlation of Returns: Covariance Given a Joint Probability Function）

 Learning Outcome

The candidate should be able to:

  • リターンの同時確率関数（joint probability function）を用いて、ポートフォリオ・リターンの共分散（covariance）および相関（correlation）を計算し解釈する

リターンの共分散や相関はどのように推定するのか。一般的には、過去の共分散に基づいて予測を行うか、過去のリターンデータに基づく市場モデル回帰（market model regression）などの方法を用いる。また、確率変数の同時確率関数が推定できる場合には、それを用いて共分散を計算することも可能である。2つの確率変数XとYの同時確率関数 P(X, Y) は、XとYの値が同時に発生する確率を与える。例えば、P(X = 3, Y = 2) は、Xが3でありYが2である確率である。

BankCorp株のリターン（RA）とNewBank株のリターン（RB）の同時確率関数が、図表6に示されるような単純な構造を持つとする。

最初の2列は、それぞれBankCorpおよびNewBankのリターンが平均（期待値）からどれだけ乖離しているかを示している。次の列は偏差の積を示す。例えば、業界が良好な場合、(25 − 14)(20 − 15) = 11 × 5 = 55 となる。これに、業界が良好である確率0.20を掛けると 55 × 0.20 = 11 となる。平均および不調のケースでも同様の計算を行う。これらの確率加重された積を合計すると、Cov(RA, RB) = 16 が得られる。

確率変数 RA と RB の共分散を計算する式は次のとおりである。

CovRA,RB=∑i∑jPRA,i,RB,jRA,i−ERARB,j−ERB.8

この式は、すべての可能な偏差の積を、それぞれの同時確率で加重して合計することを意味する。

次に、2つの確率変数が独立である場合には、同時確率関数が単純化されることに注意する。

2つの確率変数XとYは、P(X, Y) = P(X)P(Y) が成り立つとき、かつそのときに限り独立である。

例えば独立であれば、P(3, 2) = P(3)P(2) となる。個々の確率を掛け合わせることで同時確率が得られる。独立性は無相関（uncorrelated）よりも強い性質である。なぜなら、相関は線形関係のみを扱うからである。次の関係は独立な確率変数について成立し、したがって無相関な変数についても成立する。2変数について、E(XY) = E(X)E(Y) + Cov(X, Y) であり、無相関の場合 Cov(X, Y) = 0 である。

無相関な確率変数の積の期待値は、それぞれの期待値の積に等しい。

E(XY) = E(X)E(Y) if X and Y are uncorrelated.

売上高（価格×数量）など、多くの金融変数は確率変数の積である。この規則が適用できる場合、確率変数の積の期待値の計算を簡略化できる。
