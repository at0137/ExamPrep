Real World Applications: Bloomberg's Security Description

モジュール1：固定利付証券の特徴（Fixed-Income Instrument Features） 実務応用（Real World Applications）：Bloombergの証券情報表示（Security Description）

Bloombergは、金融プロフェッショナルが業務の中で利用する主要な市場データプラットフォームの一つである。本オプションセクションでは、CFAプログラム（CFA Program）の内容がBloomberg上でどのように実務で活用されるかを見ていく。

{DES} 証券情報表示（SECURITY DESCRIPTION）

BloombergのDES機能は、米国債（US Treasury bond）の特徴を確認することを可能にする。

 

「DES <GO>は、株式に関する幅広く包括的な情報を一か所で提供する機能である。DESを使用すると、特定の株式、ワラント、投資信託（mutual fund）、株式オプション（equity option）、シンセティック・オプション（synthetic option）、または上場投資信託（exchange-traded fund, ETF）に関する統合された財務データおよび基礎情報にアクセスできる。」―Bloomberg

 

DES機能を試すために、まず調査する債券を特定する。BloombergのコマンドラインにSECFと入力し、Enterまたは<GO>を押す。左上のアンバー色のカテゴリードロップダウンメニューでFixed Incomeを選択する。表示された債券のいずれかをクリックし、ポップアップメニューから「16）DES」を選択すると、その債券の詳細情報が表示される。
