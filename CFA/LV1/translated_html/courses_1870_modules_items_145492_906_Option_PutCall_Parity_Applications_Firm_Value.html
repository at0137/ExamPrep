9.06 | Option Put–Call Parity Applications: Firm Value

モジュール9：プット・コール・パリティ（put–call parity）を用いたオプションの複製 9.06｜オプションにおけるプット・コール・パリティの応用：企業価値（Firm Value）

 Learning Outcome

The candidate should be able to:

  • ヨーロピアン・オプション（European options）におけるプット・コール・パリティ（put–call parity）を説明する
  • ヨーロピアン・オプション（European options）におけるプット・コール・フォワード・パリティ（put–call forward parity）を説明する

プット・コール・パリティ（put–call parity）によって得られる洞察は、オプション取引戦略をはるかに超えて、企業価値のモデリングへと拡張され、資本提供者、すなわち株主（equity holders）と債権者（debtholders）の利害や金融請求権を説明することができる。

時点t = 0において、市場価値V₀を持つ企業が、額面Dのゼロクーポン債（zero-coupon debt）という形で借入資本にアクセスできると仮定する。企業の資産価値V₀は、負債の現在価値PV(D)と株式価値E₀の合計に等しい：V₀ = E₀ + PV(D)。

負債が満期Tに達したとき、企業の負債と資産は、時点Tの企業価値（V_T）に応じて、株主と債権者の間で次の2つの結果のいずれかとして配分される。

  1. 支払能力あり（Solvency）（V_T > D）：時点Tにおいて企業価値V_Tが負債の額面Dを上回る場合、企業は支払能力を有し、株主と債権者の双方に資本を返済できる。債権者はDを受け取り、全額返済される。株主は残余を受け取る：E_T = V_T − D。
  2. 支払不能（Insolvency）（V_T < D）：時点Tにおいて企業価値V_Tが負債の額面Dを下回る場合、企業は支払不能である。この場合、株主は何も受け取らず、債権者は企業資産価値を上回る債権を有する。したがって、債権者は債務Dの返済としてV_Tを受け取る。債権者は資産に対して優先請求権（priority claim）を持ち、V_T < Dを受け取る。株主の残余はE_T = 0である。

前のプット・コール・パリティのレッスンで示した無リスク債券とは異なり、この企業の負債はリスクを伴う。なぜなら、債権者は支払能力がある場合（V_T > D）のみDを受け取るからである。したがって、債権者は支払不能リスクを引き受ける対価として、株主からプット・オプションのプレミアムに類似したプレミアムを要求する。

株主は無限の上昇余地（企業が支払能力を維持し負債を返済できる場合）と限定された下方リスク（企業が支払不能になる場合）を持つ。一方で、債権者の上昇余地は支払能力がある場合の債務返済に限定され、支払不能の場合には元本および利息がリスクにさらされる。

次に、満期時のそれぞれのペイオフ・プロファイルをより詳しく検討する。時点t = Tにおいて、

  • 株主のペイオフは max(0, V_T − D ) であり、
  • 債権者のペイオフは min( V_T , D ) である。

これらのペイオフをオプションの観点から考えると：

  • 株主は企業資産（V_T）に対するロングポジションを持ち、行使価格Dの企業価値に対するプット・オプション（max(0, D − V_T )）を購入していると解釈できる。
  • 債権者は無リスク債券Dのロングポジションを持ち、企業価値（V_T）に対する行使価格Dのプット・オプションを株主に売却していると解釈できる。

株主の「プット購入＋企業資産のロング」の組合せは、企業資産に対するコール・オプションに等価である点に注意せよ。債権者が保有するリスク付債務は、無リスク債務Dと株主に売却したプット・オプションの組合せである。

プット・コール・パリティ（put–call parity）の関係 S₀ + p₀ = c₀ + PV(X)（式1）に戻ると、原資産S₀の代わりに企業価値V₀を、無リスク債券Xの代わりに負債Dを代入し、V₀について解くと次のようになる：

V 0 + p 0 = c 0 + PV(D).
V 0 = c 0 + PV(D) – p 0. 6

式6は、時点t = 0における企業資産価値を株主および債権者の観点から表している。株主は企業価値に対するコール・オプション（c₀）に類似したペイオフを持つポジションを有する。一方、債権者はPV(D) − p₀、すなわち企業の無リスク債務と企業価値に対するプット・オプションの売りポジションを有する。プット・オプション（p₀）は企業債務の信用スプレッド（credit spread）として解釈でき、企業が支払不能リスクを引き受けてもらうために債権者に支払う無リスク金利を上回るプレミアムを表す。このプットの価値は、支払不能の可能性が高まるほど株主にとって増加する。債権者の観点では、売却したプットの価値が大きいほど、企業の債務に含まれる信用リスクは高い。

  Question Set

プット・コール・フォワード・パリティ（put–call forward parity）とオプションの応用（Option Applications）

1.

次の記述が正しいか誤りかを判定し、その理由を述べよ：プット・コール・パリティ（put–call parity）とプット・コール・フォワード・パリティ（put–call forward parity）を結び付ける重要な点は、現物原資産ポジションがフォワード購入と無リスク債券を用いた合成原資産ポジションに置き換えられることである。

解答を確認（Check Answer）

解答（Solution）

この記述は正しい。現物原資産ポジションを、フォワード購入と無リスク債券を用いた合成原資産ポジション（synthetic underlying position）に置き換えることにより、プット・コール・フォワード・パリティ（put–call forward parity）のもとで、コール・オプション、プット・オプション、およびフォワード価格の関係が結び付けられる。

2.

次のうち、プット・コール・フォワード・パリティ（put–call forward parity）のもとで、どのポジションがどのポートフォリオと同じ無裁定価値（no-arbitrage value）を持つかを特定せよ：

1. コール・オプションのロング（long call option）（c₀） A. フォワードの売り（forward sale）、無リスク債券のロング、コール・オプションのロング
2. 無リスク債券のロング（long risk-free bond）（X(1 + r)^(−T)） B. フォワードの買い（forward purchase）、プット・オプションのロング、無リスク債券のショート
3. プット・オプションのロング（long put option）（p₀） C. フォワードの買い（forward purchase）、コール・オプションのショート、プット・オプションのロング

解答を確認（Check Answer）

解答（Solution）

プット・コール・フォワード・パリティ（put–call forward parity）の関係は次のように表されることを思い出そう。

F 0(T)(1 + r)– T + p 0 = c 0 + X(1 + r)– T .

1. Bが正しい。コール・オプションのロングは、フォワードの買い、プット・オプションのロング、および無リスク債券のショートポジションと無裁定のもとで等価である。

2. Cが正しい。無リスク債券のロングは、フォワードの買い、コール・オプションのショート、およびプット・オプションのロングに等価である。

3. Aが正しい。プット・オプションのロングは、フォワードの売り、無リスク債券のロング、およびコール・オプションのロングに等価である。

3.

合成プロテクティブ・プット（synthetic protective put）の満期時のキャッシュフローを説明せよ。

解答を確認（Check Answer）

解答（Solution）

合成プロテクティブ・プット（synthetic protective put）は、合成原資産ポジション（フォワードの買いと、行使価格に等しい額面を持つ無リスク債券のロング）とプットの購入の組合せである。

オプション満期時（t = T）には、2つのケースがある：

プット行使（X > S_T）：プットが行使される場合、投資家は（X − S_T）を受け取る。無リスク資産はF₀(T)を返し、フォワードの買いは（S_T − F₀(T)）をもたらす。

合計リターンは（X − S_T） + [S_T − F₀(T)] + F₀(T) = X となる。

プット未行使（X ≤ S_T）：プットが行使されない場合、価値ゼロで満期を迎える。無リスク資産はF₀(T)を返し、フォワードの買いは（S_T − F₀(T)）をもたらす。

合計リターンは [S_T − F₀(T)] + F₀(T) = S_T となる。

4.

債権者（debtholder）のポジションが、企業価値に対するプット・オプションの売りに類似している理由を説明せよ。

解答を確認（Check Answer）

解答（Solution）

時点Tにおいて企業価値V_Tが負債の額面Dを下回る場合（V_T < D）、企業は支払不能であり、債権者は額面Dより少ない額しか受け取れない。言い換えると、債権者のペイオフは min(D, V_T) = D − max(0, D − V_T) と表される。これは、額面Dの債務から、行使価格Dの企業価値に対するプット・オプション（max(0, D − V_T)）を差し引いたものであり、企業価値に対するプットの売りポジションに相当する。

企業価値が資産を調達するための負債の額面を上回る状態を指す。

発行体の総負債がその資産の市場価値を上回る状況を指す。
