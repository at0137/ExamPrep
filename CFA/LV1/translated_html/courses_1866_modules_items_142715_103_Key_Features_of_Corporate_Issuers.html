1.03 | Key Features of Corporate Issuers

モジュール1：組織形態、企業発行体の特徴、および所有構造 1.03｜企業発行体の主な特徴

 Learning Outcome

The candidate should be able to:

  • 企業発行体の主な特徴を説明する

前回のレッスンでは、有限責任、所有者と経営者の分離、および外部資金調達へのアクセス向上といった、他の組織形態に対する企業形態のいくつかの利点を扱った。本レッスンでは、企業の特徴をより深く探る。金融市場で資本を調達する企業（corporate issuers）は、世界中の政府以上に多くの資本を投資家から調達しているため、金融アナリストにとって理解が不可欠である。

法的主体性

企業は、規制当局に定款（articles of incorporation）を提出することによって設立される、所有者とは別個かつ独立した法的主体である。企業は個人と同様に多くの権利と責任を共有し、類似の活動を行うことができる。例えば、契約を締結し、従業員を雇用し、訴訟を起こしまたは起こされ、資金を借りたり貸したりし、投資を行い、税金を支払うことができる。

大企業はしばしば複数の地理的地域で事業を展開しており、以下の各規制管轄に従う必要がある：

  • 企業が設立されている場所、
  • 事業が行われている場所、
  • 企業が資金調達を行う場所、

および次のような活動に関して：

  • 登録、
  • 財務および非財務の報告および開示、
  • 資本市場活動（発行、取引、投資）。

所有者と経営者の分離

ほとんどの企業の重要な特徴は、事業の所有者（株主）と、それを運営する者（取締役会および経営陣）との分離である。企業では、株主は日常業務から大きく切り離されている。その代わりに、株主は取締役会を選任し、取締役会が最高経営責任者（CEO）などの経営陣を任命する。これらの経営陣は、投資、資金調達、および業務運営に関する意思決定に責任を負う。取締役および経営者は、主として株主、そして間接的にはすべてのステークホルダーの最善の利益のために行動する責任を負う。所有と経営の分離により、企業は経営に関与する必要（または意思）のない、より広範な投資家から資金を調達できる。

取締役会や経営陣が株主の最善の利益のために行動しない場合、株主は株式に付随する議決権を行使することで変革を起こすことができる。例えば、取締役会の交代に投票することであるが、これには時間がかかる場合がある。エンゲージメントや議決権を通じて業務に影響を与えたり、経営陣を変更したりする戦略は、一部の投資家によって採用されており、後のレッスンで詳述される。

所有者／株主の責任

リスクはすべての株主に分散され、株主は有限責任を負う。すなわち、株主が被る最大の損失は投資額に限られ（株式価値はゼロまで下がり得るが、それ以上にはならない）、個別に明示的な保証をしていない限り、企業の負債について責任を負わない。

株主は、会社の定款に別段の定めがない限り、保有株式に比例してリスクとリターンを共有する。パートナーシップと異なり、所有単位はより小さい単位の株式に分割されているため、投資家は株式を通じて所有権をより容易に売買できる。例えば、発行済株式数が10億株を超える企業もあり、所有権は極めて細かい単位に分割されている。また、一部の企業は異なるリスク・リターン特性を持つ複数の種類またはクラスの株式を発行している（これについては後のレッスンで扱う）。図表7は、所有者と企業の関係を示している。

外部資金調達

所有と経営の分離により、株式を購入するだけで所有者になれるため、企業は他の事業形態よりも容易に外部資金にアクセスできる。他の形態より設立および運営コストは高いものの、必要資本が個人または少人数のパートナーで調達可能な範囲を超える場合、企業形態が一般に選好される。資金は個人または機関（ミューチュアルファンド（mutual funds）、年金基金（pension funds）、銀行、政府、非営利団体、他の企業など）によって提供される。

企業は2つの方法で資金調達を行う。すなわち、株式（equity）として株式の発行または利益の再投資による方法と、借入、すなわち負債（debt）としてローン、債券、リースによる方法である。株式投資家（株主）は、配当（dividends）と呼ばれる利益分配を受け取る権利を有する一方、負債は将来の特定日までに利息とともに返済されなければならない。株式または負債による資金調達は、発行体と投資家との間の私的契約に基づく場合もあれば、証券（security）という標準化された金融商品として公開市場で取引可能な形で行われる場合もある（次のレッスンで扱う）。

課税

企業の課税は国によって大きく異なるが、最終的には発行体の税務報告、納税、および税務上の地位を規定する税務当局および税法の対象となる。多くの国では、企業は利益に対して課税される。課税所得は通常、税法と会計基準が異なるため、財務諸表上の利益とは一致しない。

多くの国では、株主は受け取った配当（dividends）に対して追加で課税される。これは企業利益の二重課税と呼ばれる。一部の国では、企業がすでに配当原資となる利益に対して課税されている場合、株主は配当に対して個人課税を受けないか、企業が支払った税金の持分に応じた税額控除を受ける。他の国では、企業が全く課税されない場合や、一国内でも異なる税制が適用される場合がある。

  Example 3Double Taxation of Corporate Profits

1.

フランスの小売大手オーシャン・ホールディング（Auchan Holding）は、昨年8億3,800万ユーロの税引前利益を計上し、2億6,400万ユーロの法人税を支払った。フランスの投資家は、受け取った配当に対して30%の税金も支払う。もしオーシャンが税引後利益をすべて配当として投資家に分配し、すべての投資家がフランス在住である場合、オーシャンの税引前利益に対する総税率は何パーセントとなるか。

解答を確認

解答

税引前利益 €838
法人税（31.5%） €264
税引後利益 €838 − €264 = €574
分配配当（Distributed Dividend） €574
投資家の配当課税（30%）（Investor Dividend Tax） €574 × 0.30 = €172.2
総税率 (€264 + €172.2)/€838 = 52.1%

税引後利益5億7,400万ユーロが配当として投資家に支払われた場合、投資家は受け取った配当に対して1億7,220万ユーロの個人税を支払う。支払われる総税額は4億3,620万ユーロ（法人レベルでの2億6,400万ユーロと個人レベルでの1億7,220万ユーロの合計）となり、総税率は52.1%となる。

前述の例で示された税制上の不利にもかかわらず、企業形態は依然としていくつかの理由から魅力的である。企業の株主は配当に対して課税される一方で、個人事業主やパートナーは、配当として分配されるかどうかに関わらず、すべての利益に対して課税されることが多い（利益留保に対する例外も存在する）。この違いにより、例えば成長のための追加的な設備投資などに未分配利益を再投資することを想定する企業にとって、企業形態は魅力的となる。また、法人税率が個人所得税率よりも低い法域では、利益を企業内に「留保」することが有利となる場合がある。

  Question Set

1.

所有と経営の分離が、企業発行体の資本へのアクセスを拡大させる理由を説明せよ。

解答を確認

解答

所有と経営の責任を分離することにより、企業は経営に関与したくないが投資家として参加したいと考える個人および機関を含む、幅広い所有者を引き付けることができる。

4.

Check Answer

例えば負債証券や株式証券を発行することにより、金融市場で資金調達を行う有限責任会社または企業。

取締役の選任やその他株主投票に付される事項について、株主が投票を行う権限。

企業に対する所有持分。負債などのより優先度の高い請求権が満たされた後に残る資産に対する残余請求権。純資産（net assets）とも呼ばれる。

将来の特定時点において現金、株式、またはその他の資産を受け取る権利。債務者または借り手の観点からは、将来の特定時点に現金、株式、またはその他の資産を支払う義務。一般に、負債請求権は無条件であり、株式請求権よりも優先される。

企業から株主へ利益および／または純資産を分配すること。通常は現金で支払われるが、株式や不動産などの資産で支払われる場合もある。

企業に対する株式または負債の持分、あるいはデリバティブなどの関連する権利を示す証拠。多くの場合、証券取引所の要件に適合するよう標準化されている。

所得が二重に課税される。
