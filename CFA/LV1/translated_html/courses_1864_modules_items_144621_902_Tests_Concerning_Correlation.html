9.02 | Tests Concerning Correlation

モジュール9：独立性のパラメトリック検定およびノンパラメトリック検定（Parametric and Non-Parametric Tests of Independence）9.02｜相関に関する検定（Tests Concerning Correlation）

 Learning Outcome

The candidate should be able to:

  • 母相関係数（population correlation coefficient）がゼロに等しいという仮説に関するパラメトリック検定およびノンパラメトリック検定を説明し、与えられた有意水準においてその仮説が棄却されるかどうかを判断せよ。

相関に関する最も一般的な仮説は、母相関係数をゼロと比較するものである。なぜなら、関係が存在するかどうかを問うことが多く、その場合、帰無仮説は相関係数がゼロ（すなわち関係がない）であることを意味するからである。母相関係数に関する仮説は、他の検定で見たように、両側または片側となり得る。ρを母相関係数とすると、可能な仮説は次のとおりである。

Two sided: H0: ρ = 0 versus Ha : ρ ≠ 0
One sided (right side): H0: ρ ≤ 0 versus Ha : ρ > 0
One sided (left side): H0: ρ ≥ 0 versus Ha : ρ < 0

これらの仮説を母相関に関して検定するために、標本相関（sample correlation）を用いる。

相関のパラメトリック検定（Parametric Test of a Correlation）

パラメトリックなペアワイズ相関係数（pairwise correlation coefficient）は、しばしばピアソン相関（Pearson correlation）、二変量相関（bivariate correlation）、または単に相関（correlation）と呼ばれる。本節ではこの統計量の計算そのものではなく検定に焦点を当てるが、標本相関の式を見ることでノンパラメトリック相関との違いを理解する助けとなる。2つの変数XおよびYを考える。標本相関 rXY は次のとおりである。

rXY=sXYsXsY, 1

ここで sXY はXとYの標本共分散（sample covariance）、sX はXの標準偏差（standard deviation）、sY はYの標準偏差である。通常、添字を省略して単に r と表す。

この式から分かるように、各観測値はそれぞれの変数の平均と比較されており、共分散の影響により、各観測値がその平均からどれだけ乖離しているかが重要となる。なお、共分散が相関の符号を決定する。

2つの変数が正規分布（normally distributed）に従う場合、標本相関 r を用いて帰無仮説（H₀: ρ = 0）を棄却すべきかどうかを検定できる。t検定（t-test）の式は次のとおりである。

t=rn−21−r2. 2

この検定統計量は自由度 n − 2 のt分布（t-distribution）に従う。式2に関する実務的な観察として、帰無仮説を棄却するために必要な r の大きさは、標本サイズ n が増加するにつれて小さくなる。これは2つの理由による。第一に、n が増加すると自由度が増加し、t統計量の臨界値の絶対値が小さくなる。第二に、分子の絶対値が n の増加とともに大きくなり、計算されたt統計量の絶対値が大きくなるためである。例えば、標本サイズ n = 12 の場合、r = 0.35 は t統計量1.182となり、有意水準0.05ではゼロと有意に異ならない（tα/2 = ±2.228）。一方、標本サイズ n = 32 の場合、同じ標本相関 r = 0.35 は t統計量2.046となり、有意水準0.05でちょうど有意となる（tα/2 = ±2.042）。

別の見方をすると、同一の母集団から標本抽出する場合、標本サイズを増やすと、他の条件が同じであれば、誤った帰無仮説はより棄却されやすくなる（すなわち検定力（power of the test）が高まる）。これは観測数の増加により検定統計量の分子が大きくなるためである。図表1（Exhibit 1）では、3つの異なる標本相関係数について、対応するt統計量および5%水準（両側検定）での有意性を示している。標本サイズが増加するにつれて有意性が示されやすくなるが、その速度は標本相関係数に依存し、相関が高いほど、標本サイズの増加に伴いより早く有意となる。データセットが大規模化し標本サイズが増加すると、帰無仮説はほぼ常に棄却されるため、他のデータ分析手法を適用する必要がある。

ノンパラメトリックな相関の検定：スピアマン順位相関係数（Spearman Rank Correlation Coefficient）

対象とする母集団が正規性（normality）から大きく逸脱していると考えられる場合、スピアマン順位相関係数（Spearman rank correlation coefficient）rS に基づく検定を用いることができる。スピアマン順位相関係数は、本質的には前述のピアソン相関係数（Pearson correlation coefficient）と同等であるが、各標本における2つの変数（例えばXとY）の順位に基づいて計算される。rS の計算には次の手順が必要である：

  1. Xの観測値を大きい順に順位付けする。最大の値に1、2番目に大きい値に2、というように順位を割り当てる。同順位（ties）がある場合は、それらが占める順位の平均値を各観測に割り当てる。例えば、3番目と4番目に大きい値が同じであれば、両方に3.5（3と4の平均）を割り当てる。同様の手順をYの観測値にも適用する。
  2. XとYの各観測ペアについて順位の差 di を計算し、その後 di²（順位差の二乗）を計算する。

  3. 標本サイズをnとすると、スピアマン順位相関は次のように与えられる：

    rs=1−6∑i=1ndi2n(n2−1). 3

あるアナリスト（analyst）が、類似したリスクを持つ2つの投資ファンドAおよびBの35年間のリターンの関係を調査している。パラメトリック相関の仮定が満たされていない可能性を懸念し、スピアマン順位相関を検定することにした。仮説は H₀: rS = 0 および Hₐ: rS ≠ 0 である。彼女はリターンデータを収集し、各ファンドのリターンを順位付けし、順位差およびその二乗を計算した。部分的な表が図表4（Exhibit 4）に示されている。
