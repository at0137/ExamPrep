Real World Applications: Bloomberg's Security Ownership

モジュール2：投資家およびその他の利害関係者（Investors and Other Stakeholders）実務応用（Real World Applications）：ブルームバーグの証券保有（Bloomberg's Security Ownership）

ブルームバーグ（Bloomberg）は、金融専門家が業務全体を通じて使用する主要な市場データプラットフォームの一つである。この任意セクションでは、CFAプログラムのカリキュラムの実務応用をブルームバーグ上で探る。

{OWN} 証券保有（SECURITY OWNERSHIP）

ブルームバーグのOWN機能は、選択した企業の証券保有状況の概要を確認することを可能にする。

 

「OWN <GO> は、証券保有（Security Ownership (HDS)）機能のオーナーシップ概要（Ownership Summary）タブを表示し、特定時点と比較して対象証券の保有者構成がどのように変化したかを把握するために、保有者統計の概要を確認することができる。」—ブルームバーグ（Bloomberg）

 

この機能を利用するには、ブルームバーグのコマンドラインにOWNと入力し、Enterまたは<GO>を押す。次に、プロクター・アンド・ギャンブル（Procter & Gamble Company）の場合は「PG US EQUITY」のような証券コードを最初のアンバー色の入力欄に入力し、Enterまたは<GO>を押して読み込む。
