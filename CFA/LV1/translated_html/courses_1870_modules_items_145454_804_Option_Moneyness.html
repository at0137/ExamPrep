8.04 | Option Moneyness

モジュール8：オプションの価格付けと評価（Pricing and Valuation of Options）8.04｜オプションのマネーネス（Option Moneyness）

 Learning Outcome

The candidate should be able to:

  • オプションの行使価値（exercise value）、マネーネス（moneyness）、および時間価値（time value）を説明せよ。

任意の時点tにおけるオプションの行使価値は、その時点でのペイオフに等しいことが示されている。オプションの総価値と行使価格（exercise price, X）との関係は、マネーネス（moneyness）を表す。イン・ザ・マネー（in-the-money, ITM）の例としては、原資産のスポット価格が行使価格を上回るコールオプション、およびスポット価格が行使価格を下回るプットオプションがある。原資産価格が行使価格と等しい場合、コールまたはプットはアット・ザ・マネー（at-the-money, ATM）と呼ばれる。コールオプション（プットオプション）において、原資産価格が行使価格より低い（高い）場合、そのオプションは行使される可能性が低く、アウト・オブ・ザ・マネー（out-of-the-money, OTM）と呼ばれる。

また、オプションがどの程度イン・ザ・マネーまたはアウト・オブ・ザ・マネーであるかは、原資産価格の変化に対するオプション価格の感応度にも影響を与える。例えば、ディープ・イン・ザ・マネー（deep-in-the-money）オプション、すなわち行使される可能性が非常に高いオプションは、通常、オプション価格と原資産価格の変化の間にほぼ1対1の対応関係を示す。一方、ディープ・アウト・オブ・ザ・マネー（deep-out-of-the-money）オプションは、行使される可能性が非常に低く、原資産価格が変化してもオプション価格の感応度ははるかに低い。これに対し、アット・ザ・マネー（at-the-money）オプションでは、原資産価格の比較的小さな変化が行使されるかどうかを左右することが多い。マネーネスは、同一の原資産に対して異なる行使価格や満期を持つオプションを比較する際によく用いられる。図表1（Exhibit 1）は、満期時におけるコールオプションのマネーネスを示し、これらの関係をまとめている。

  Exhibit 1Call Option Moneyness at Maturity

Expand / Collapse Extended Description

このグラフには、原点から直線的に増加する緑色の破線が描かれており、これはオプションのペイオフを表す。また、x軸上にはゼロ利益水準を示す黒の水平線がある。さらに、x軸上のXとラベル付けされた黒の垂直線は、オプションの行使価格を示す。このグラフは、S_T が X を上回る場合にオプション保有者が正のペイオフを得て、S_T が X を下回る場合にはペイオフがゼロとなることを示している。

マネーネス（Moneyness） コールオプション（call option, cₜ） プットオプション（put option, pₜ）
イン・ザ・マネー（in-the-money, ITM） Sₜ > X Sₜ < X
アット・ザ・マネー（at-the-money, ATM） Sₜ = X Sₜ = X
アウト・オブ・ザ・マネー（out-of-the-money, OTM） Sₜ < X Sₜ > X

 

 

  Example 2Put Option Moneyness

例1を想起せよ。時点tにおいて、残存満期6か月のプットオプションは行使価格（X）が1,000ユーロ、原資産のスポット価格（Sₜ）が950ユーロであった。このときのプットオプションのマネーネスを説明せよ。

原資産のスポット価格が行使価格を下回っているため、このプットオプションは50ユーロ分イン・ザ・マネーである。

オプションの価値と行使価格との関係を、あらゆる可能な原資産価格の範囲にわたって表す概念。

行使される可能性が非常に高いオプション。

行使される可能性が非常に低いオプション。
