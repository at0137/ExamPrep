2.04 | Cash Flow Additivity

モジュール2：金融における貨幣の時間価値（Time Value of Money in Finance） 2.04｜キャッシュフロー加法性（Cash Flow Additivity）

 Learning Outcome

The candidate should be able to:

  • キャッシュフロー加法性の原則（cash flow additivity principle）を説明し、それが無裁定条件（no-arbitrage condition）において重要である理由、およびフォワード金利（forward interest rates）、フォワード為替レート（forward exchange rates）、オプション価値（option values）の算出への応用を説明する

現在と将来のキャッシュフローの間の貨幣の時間価値のトレードオフは、キャッシュフロー加法性の原則を用いることで、単一の金融商品のインプライド・リターンの計算や将来キャッシュフローの価格付けを超えて拡張できる。キャッシュフロー加法性のもとでは、同一時点に割り引かれた将来キャッシュフローの現在価値は、それぞれのキャッシュフローの現在価値の合計に等しい。この原則は、市場価格が無裁定条件を満たすこと、すなわち取引コストがない場合に無リスクで利益を得る機会が存在しないことを保証するうえで重要である。

解答

2つの戦略を比較するには、それぞれの現在価値を計算し、どちらが高いか、また少なくとも一方が正の現在価値を持つかを確認する必要がある。この目的のために、両戦略の現在価値を計算するか、あるいは各期間のキャッシュフローの差を取って新たなキャッシュフローを作成することができる。どちらの方法でも同じ数値結果が得られ、この数値的等価性がキャッシュフロー加法性の原則を表している。まず、図に示すように2つの戦略の差分キャッシュフローを作成する：

この純キャッシュフローの現在価値はゼロであり、両戦略が等価であることを示す：

PV= 0+ 151.10+ −51.102+ −12,651.103=0.

この分析から、2つの戦略は経済的に同等であるため、どちらを選んでも差はないと結論づけられる。また、これらの戦略が経済的に価値を持つかも確認する必要がある。そのため、それぞれの現在価値を個別に計算する。

戦略1の現在価値はGBP11.91である：

PV= −100+ 451.10+ 451.102+ 451.103=11.91.

戦略2の現在価値もGBP11.91である：

PV= −100+ 601.10+ 401.102+ 32.351.103=11.91.

両戦略とも現在価値が正であり価値がある。また、現在価値が等しいため、戦略2から戦略1の現在価値を引くとゼロとなる（すなわち、11.91 − 11.91）。

両戦略は価値があり現在価値も等しいため、どちらを選択してもよい。

以下の節では、キャッシュフロー加法性の原則を3つの異なる経済状況に適用し、それぞれにおいて経済的に同等な2つの戦略を比較することで無裁定の原則をさらに説明する。

キャッシュフロー加法性を用いたフォワード金利（forward rates）の導出

異なる満期を持つ2つの無リスク・ディスカウント債を考える：

  • 1年債：r₁ = 2.50%
  • 2年債：r₂ = 3.50%

2年間の投資期間でGBP100を運用したいリスク中立（risk-neutral）投資家には、2つの戦略がある：

  • 投資戦略1：年率3.50%で2年間投資する。式5を用いると、2年後の将来価値（FV₂）は次のように求められる：GBP100 = FV₂ / (1 + r₂)²；FV₂ = GBP107.12。
  • 投資戦略2：年率2.50%で1年間投資し、その後1年後に1年フォワード金利F₁,₁で再投資する。

これら2つの戦略は図表10にまとめられている：

 Exhibit 10Implied Forward Rate Example

戦略1：t=0でGBP100から始まり、t=1でGBP102.5となり、その後t=2では「GBP102.5 × (1 + フォワード金利)」となる。戦略2：t=0でGBP100から始まり、t=2では「GBP100 × (1 + 利回り)²」となる。

キャッシュフロー加法性の原則のもとでは、リスク中立投資家は次の条件のもとで戦略1と戦略2に無差別となる：

FV2 = PV0×(1+r2)2= PV0×(1 + r1)(1+F1,1),25
GBP107.12 = GBP100 (1.025)(1+F1,1), and
F1,1 = 4.51%.

式25を変形すると、一般的にF₁,₁は次のように求められる：

F1,1 = (1 + r2)2/(1 + r1) – 1.

1年後開始の1年フォワード金利が4.51%でなければならない理由を示すために、仮に投資家が1年後から1年間5%の利率を確定できるとする。この場合、以下の裁定戦略により無リスク利益が得られる：

  1. 年率3.50%で2年間GBP100を借り入れ、2年後にGBP107.12を返済する契約を結ぶ。
  2. GBP100を最初の1年間は2.50%、2年目は5%で運用し、2年後にGBP107.63を受け取る。
  3. 上記2つの戦略を組み合わせることで、初期投資ゼロで2年後にGBP0.51の無リスク利益が得られる。

これらの戦略は、金融市場において無リスクの裁定利益が得られないようにフォワード金利が設定されるべきであることを示している。これは、2つの経済的に同等な戦略（すなわち、2年物金利で2年間借りる場合と、異なる金利で1年ずつ2回借りる場合）を比較することで示される。フォワード金利F₁,₁は、1年後に開始する1年間の再投資における損益分岐金利として解釈できる。

無裁定を用いたフォワード為替レート（Forward Exchange Rates）

次に、キャッシュフロー加法性の原則を異なる通貨を含む経済状況に拡張する。

USD1,000を6か月間投資すると仮定する。米国または日本の6か月国債への無リスク投資を検討しているとする。現在の為替レートはJPY134.40 = USD1とする。6か月の円建て無リスク金利は0.05%、ドル建て無リスク金利は2.00%とする。本例では連続複利を仮定する。

  • 投資戦略1：USD1,000を6か月の米国短期国債に投資する。時点t=0：USD1,000を年率2.00%で6か月運用する。時点t=T（6か月後）：USD1,010.05（= 1,000e^(0.02 × 0.5)）を受け取る。
  • 投資戦略2：USD1,000を現在の為替レート134.40で円に交換し、日本の6か月国債に投資し、あらかじめ設定された6か月フォワード為替レート133.096で6か月後にドルに再交換する。時点t=0：USD1,000をJPY134,400に交換し、年率0.05%で6か月貸し付ける。時点t=T（6か月後）：JPY134,433.60（= 134,400e^(0.0005 × 0.5)）を受け取り、フォワードレート133.096でUSD1,010.05（= 134,433.60 / 133.096）に交換する。

これら2つの戦略は経済的に同等であり、いずれも時点 t = 0 に USD1,000 を投資し、6か月後に USD1,010.05 を受け取ることを意味する。これら2つの戦略を結び付ける要素は、6か月のフォワード為替レート（forward exchange rate）133.096 JPY/USD である。このフォワードレートが 133.096 より高くても低くても、日本円と米ドルの間で交換を行う投資家にとって裁定機会（arbitrage opportunity）が存在することになる。図表12はこれら2つの戦略の視覚的な構成を示している。

解答（Solution）

無裁定（no-arbitrage）USD/GBP フォワードレートは 1.2693 から 1.2742 に上昇する。

投資家が GBP1,000 を 1年間、英国ポンドまたは米ドルの無リスク割引債（risk-free discount bond）に投資すると仮定する。1年後に GBP1 を購入するために必要な USD の額は、割引された将来価格と等しくなければならない。

  • 5月31日時点：国内戦略（Domestic Strategy）：GBP1,000 を 1.291% の1年物英国ポンド無リスク金利で投資し、1年後に GBP1,012.99（= 1,000e (0.01291)）を受け取る。海外戦略（Foreign Strategy）：GBP1,000 を USD/GBP1.2602 で交換して USD1,260.20 を得て、これを 2.012% の1年物米ドル無リスク金利で運用すると、1年後に USD1,285.81（=1,260.20 e (0.02012)）となる。したがって、5月31日時点の無裁定 USD/GBP フォワードレートは 1.2693（= USD1,285.81/GBP1,012.99）である。
  • 6月15日時点：国内戦略（Domestic Strategy）：GBP1,000 を 1.562% の1年物英国ポンド無リスク金利で投資し、1年後に GBP1,015.74（= 1,000e (0.01562)）を受け取る。海外戦略（Foreign Strategy）：GBP1,000 を 1.2602 で交換して USD1,260.20 を得て、これを 2.667% の1年物米ドル無リスク金利で運用すると、1年後に USD1,294.26（=1,260.20 e (0.02667)）となる。したがって、6月15日時点の無裁定 USD/GBP フォワードレートは 1.2742（= USD1,294.26/GBP1,015.74）である。

6月中旬時点では、1年後に米ドルを英国ポンドに交換する契約を結ぶ投資家は、5月末に同様の契約を結んだ場合と比べて、GBP1 と引き換えにより多くの米ドル（1.2742 対 1.2693）を支払う必要がある。r USD の上昇幅が大きいため、金利差（r USD − r GBP）が拡大し、その結果、期間中に米ドルは英国ポンドに対してフォワードベースで減価する。言い換えると、米ドルが英国ポンドに対してフォワードで減価すると期待される場合、投資家を米ドルに引き付けるためには、より高い米ドル金利が必要となる。

キャッシュフロー加法性（cash flow additivity）を用いたオプション価格付け（Option Pricing）

ある資産の現在価格が 40 中国元（Chinese yuan, CNY40）であるとする。この資産はリスク資産であり、次の期間に価格が 40% 上昇して CNY56 になる可能性、または 20% 下落して CNY32 になる可能性がある。

投資家は、この資産に関する契約を売却したいと考えている。この契約では、買い手は義務ではなく権利として、次の期間の終了時に CNY50 で当該資産を購入することができる。この契約の無裁定価格（no-arbitrage pricing）を導くために、キャッシュフロー加法性の原則を適用する。

図表14の二項木（binomial tree）は、この資産の将来の2つの可能な結果を要約している。

 Exhibit 14One-Period Binomial Tree for the Asset’s Price

2つのシナリオにおける契約価値は以下の通りである（図表15に視覚的に示されている）。

  • 価格上昇（Price Increase）：40% の上昇により、契約価値は CNY6（=CNY56 − CNY50）となる。これは、契約保有者が CNY50 で資産を購入し、市場価格 CNY56 で売却できるためである。
  • 価格下落（Price Decrease）：20% の下落により、契約価値は 0（CNY0）となる。これは、市場価格が CNY32 に過ぎないため、契約保有者は CNY50 で資産を購入しない選択をするからである。

 Exhibit 15One-Period Binomial Tree for the Contract’s Price

次に、契約の売り手（contract seller）のポジションを考える：

  • t = 0 において、契約価値は c₀ である。
  • t = 1 において、契約価値は、基礎資産（underlying asset）が CNY56 に上昇した場合は CNY6、CNY32 に下落した場合は CNY0 のいずれかとなる。

初期契約価値 c₀ は未知であり、キャッシュフロー加法性（cash flow additivity）と無裁定価格（no-arbitrage pricing）を用いて求めることができる。すなわち、契約および基礎資産の各将来シナリオにおける価値を用いて、リスクフリー・ポートフォリオ（risk-free portfolio）、すなわち両シナリオで同一価値となるポートフォリオを構築できる。例えば、t = 0 において投資家が、契約を価格 c₀ で売却し、基礎資産を 0.25 単位購入するポートフォリオを構築するとする。このポートフォリオはレプリケーティング・ポートフォリオ（replicating portfolio）と呼ばれ、リスクフリー資産と同一の将来キャッシュフローを生成するように設計されている。ポートフォリオ価値を V とし、t = 0 および t = 1 の価格上昇・下落の両シナリオにおける値を表すと、次のようになる：

  • V₀ = 0.25 × 40 − c₀,
  • V₁ᵘ = 0.25 × 56 − 6 = 8,
  • V₁ᵈ = 0.25 × 32 − 0 = 8.

このように、レプリケーティング・ポートフォリオの価値は、資産価格が上昇しても下落しても常に CNY8 となる。したがって、このポートフォリオはリスクフリーであり、無リスク資産として割り引くことができる。これら2つのシナリオにおけるペイオフは図表16に示されている。

 Exhibit 16Call Option Replication

契約価格に対するコールオプションのレプリケーション（call option replication）。t = 0 におけるレプリケーティング・ポートフォリオの現在価値は、0.25 × S₀ − C₀ である。t = 1 における価格上昇時の価値は 0.25 × S₁ᵘ − C₁ᵘ、価格下落時の価値は 0.25 × S₁ᵈ − C₁ᵈ となる。

c₀ を求めるために、レプリケーティング・ポートフォリオの現在価値（0.25 × 40 − c₀）を、各シナリオにおけるリスクフリーペイオフ CNY8 の割引現在価値と等しくおく（式5）。金利 r を 5% と仮定する：

V0 = 0.25S0 − c0 ,
V0= V1u1+r= V1d1+r, and
0.25×40−c0= CNY81.05 .

c₀ を解くと CNY2.38 となる。したがって、CNY2.38 は契約の売り手が買い手から受け取るべき公正価格（fair price）である。

上記の例では、投資家がポートフォリオの一部として基礎資産を 0.25 単位購入することを仮定した。この基礎資産の割合は、オプション価格付けにおいてヘッジ比率（hedge ratio）と呼ばれる。実際、この例はコールオプションの価値を求めるための簡略化された手法である（この内容は後のカリキュラムでより詳しく扱われる）。これまでの例と同様に、キャッシュフロー加法性を用いて異なる戦略を比較し、金融商品の無裁定価格を求めている。

  Example 15Put Option

2つのシナリオにおけるプットオプション価値は次の通りである：

  • 価格上昇（Price Increase, p₁ᵘ）：40% の上昇により、プットオプション価値は CNY0 となる。これは、オプション保有者が行使価格で株式を売却しないことを選択するためである。
  • 価格下落（Price Decrease, p₁ᵈ）：20% の下落により、プットオプション価値は CNY18（CNY50 − CNY32）となる。これは、プットオプション保有者が市場価格 CNY32 の株式を行使価格 CNY50 で売却することを選択するためである。

コールオプションとは異なり、プットオプションは株価が下落すると価値が増加するため、リスクフリー・ポートフォリオは株式の購入とプットオプションの購入を組み合わせる。具体的には、t = 0 において価格 p₀ でプットオプションを購入し、基礎株式を 0.75 単位購入すると仮定する。このとき、t = 0 および t = 1 におけるポートフォリオ価値は次の通りである：

  • V₀ = 0.75 S₀ + p₀,
  • V₁ᵘ = 0.75 S₁ᵘ + p₁ᵘ,
  • V₁ᵈ = 0.75 S₁ᵈ + p₁ᵈ,

ここで V₁ᵘ = V₁ᵈ である。

1期間後のリスクフリー・ポートフォリオ価値（V₁ = 0.75 S₁ + p₁）は、2つのシナリオにおいて次のようになる：

  • 価格上昇（V₁ᵘ）：ポートフォリオ価値は CNY42（= (0.75 × CNY56) + CNY0）である。
  • 価格下落（V₁ᵈ）：ポートフォリオ価値は CNY42（= (0.75 × CNY32) + CNY18）である。

p₀ を求めるには、レプリケーティング・ポートフォリオの現在価値（0.75 S₀ + p₀）をペイオフの割引将来価値と等しくおく。ここで S₀ は CNY40、r は 5% とする：

V0 = 0.75S0 + p0,
V0= V1u1+r= V1d1+r, and
p0= CNY421.05−0.75CNY40.

p₀ を解くと CNY10 となる。プットオプション価格が高いのは、与えられた二項モデル（binomial model）のパラメータの下で、プットオプションのペイオフがコールオプションより大きいためである。オプション価格に影響を与える要因については、後のカリキュラムで詳細に扱う。

同一時点で評価された金額は加法的であるという原則。

デリバティブポジション（derivative position）に伴うリスクを相殺するための基礎資産ポジション（underlying investment position）の割合。
