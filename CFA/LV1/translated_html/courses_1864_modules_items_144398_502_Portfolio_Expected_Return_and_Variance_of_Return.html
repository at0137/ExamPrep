5.02 | Portfolio Expected Return and Variance of Return

モジュール5：ポートフォリオ数学（Portfolio Mathematics）5.02｜ポートフォリオ期待リターンとリターン分散（Portfolio Expected Return and Variance of Return）

Learning Outcome

The candidate should be able to:

  • ポートフォリオ・リターンの期待値（expected value）、分散（variance）、標準偏差（standard deviation）、共分散（covariances）、および相関（correlations）を計算し解釈する

ポートフォリオの期待リターン E(Rp) は、各構成証券の期待リターン（R₁からRₙ）を、それぞれのポートフォリオ内での比率（通貨単位ベースのウェイト w₁からwₙ）で加重平均したものである。

ERp=Ew1R1+w2R2+…+wnRn=w1ER1+w2ER2+…+wnERn.1

図表1に示される3資産ポートフォリオについて、各資産の期待リターンが推定されているとする。

図表1 サンプルポートフォリオのウェイトと期待リターン（Weights and Expected Returns of Sample Portfolio）

資産クラス（Asset Class） ウェイト（Weight） 期待リターン（%）（Expected Return (%)）
S&P 500 0.50 13
米国長期社債（US long-term corporate bonds） 0.25 6
MSCI EAFE 0.25 15

ポートフォリオの期待リターンは11.75%と計算される。

ERp=w1ER1+w2ER2+w3ER3=0.5013%+0.256%+0.2515%=11.75%.

ここでは投資リスクの尺度としてポートフォリオのリターン分散に関心がある。したがって、ポートフォリオ分散は次のとおりである。

σ2(Rp) = E[(Rp E(Rp))2].2

これは将来志向の意味での期待分散である。このポートフォリオ分散の定義を実装するには、ポートフォリオ内の個々の資産に関する情報に加えて、共分散（covariance）の概念が必要となる。記法を簡潔にするため、E(Rp) を ERp と表記する。

共分散（Covariance）

2つの確率変数 Ri と Rj が与えられたとき、それらの共分散は次のように定義される。

Cov(Ri ,Rj) = E[(Ri − ERi)(Rj − ERj)].3

別の表記として σ(Ri, Rj) や σij がある。式3は、2つの確率変数の共分散が、それぞれの期待値からの偏差の積の確率加重平均であることを示している。この測度は母集団共分散（population covariance）であり、将来志向である。過去データの標本（サイズ n）に基づく標本共分散（sample covariance）は次のとおりである。

Cov(Ri,Rj)=∑t=1T(Ri,t−R¯i)(Rj,t−R¯j)/(n−1).4

3資産ポートフォリオの分散の定義から出発し、それが3つの分散項と6つの共分散項に分解される様子を考える。導出過程は省略するが、結果は式5で与えられる。

σ 2 = E [ ( R p − E( R p ) ) 2 ]
= E [ ( w 1 R 1 + w 2 R 2 + w 3 R 3 − E ( w 1 R 1 + w 2 R 2 + w 3 R 3 ) ) 2 ]
= E [ ( w 1 R 1 + w 2 R 2 + w 3 R 3 − w 1 E R 1 − w 2 E R 2 − w 3 E R 3 ) 2 ] 5
=w12σ2R1+w1w2CovR1,R2+w1w3CovR1,R3+w1w2CovR1,R2+w22σ2R2+w2w3CovR2,R3+w1w3CovR1,R3+w2w3CovR2,R3+w32σ2R3.

共分散では変数の順序は関係ない（例えば Cov(R₂, R₁) = Cov(R₁, R₂)）。また、対角の分散項 σ²(R₁)、σ²(R₂)、σ²(R₃) はそれぞれ Cov(R₁, R₁)、Cov(R₂, R₂)、Cov(R₃, R₃) と表せる。このことを踏まえると、式5は次のように最も簡潔に表現できる。

σ2Rp=∑i=13∑j=13wiwjCovRi,Rj.

さらに、この表現は任意の資産数 n のポートフォリオに一般化できる。

σ2Rp=∑i=1n∑j=1nwiwjCovRi,Rj. 6

式6は、個々のリターン分散がポートフォリオ分散の一部ではあるが、それがすべてではないことを示している。3つの分散項に対して、対角外の共分散項は6つ存在する。資産が20個ある場合、分散項は20個であるのに対し、対角外の共分散項は20×20 − 20 = 380個となる。第一の観察として、他の条件が同じであれば、保有資産数が増えるほど共分散の重要性は増す。

共分散項は、リターンの共動（co-movements）がポートフォリオ全体の分散にどのように影響するかを捉える。共分散の定義から、2つの重要な性質が導かれる。

  1. 共分散の符号は次のように解釈できる。ある資産のリターンが期待値を上回るときに、もう一方の資産のリターンが期待値を下回る傾向がある場合（平均的に逆の関係）、共分散は負となる。資産間のリターンが無関係であれば共分散は0となる。両資産のリターンが同時に期待値より上または下にある傾向がある場合（平均的に正の関係）、共分散は正となる。確率変数が自分自身との共分散（own covariance）はその分散に等しい：Cov(R, R) = E{[R − E(R)][R − E(R)]} = E{[R − E(R)]²} = σ²(R)。

図表2は、ポートフォリオの期待リターン（パネルA）およびリターン分散（パネルB）の入力要素をまとめたものである。パネルBの共分散行列（covariance matrix）に示されるように、すべての共分散のリストが、ポートフォリオ分散を計算するために必要な統計情報を構成する。

図表2 ポートフォリオ期待リターンおよび分散の入力（Inputs to Portfolio Expected Return and Variance）

A. ポートフォリオ期待リターンの入力
Asset A B C
  E(RA) E(RB) E(RC)
B. 共分散行列：ポートフォリオ分散の入力
Asset A B C
A Cov(RA, RA) Cov(RA, RB) Cov(RA, RC)
B Cov(RB, RA) Cov(RB, RB) Cov(RB, RC)
C Cov(RC, RA) Cov(RC, RB) Cov(RC, RC)

資産が3つの場合、共分散行列は3² = 3 × 3 = 9個の要素を持つが、対角項である分散（図表2で太字）は、非対角項とは別に扱われる。したがって、分散を除くと9 − 3 = 6個の共分散がある。しかし、Cov(RB, RA) = Cov(RA, RB)、Cov(RC, RA) = Cov(RA, RC)、Cov(RC, RB) = Cov(RB, RC)である。対角線の下側の共分散行列は、上側の鏡像となるため、どちらか一方（上側または下側）のみを用いればよい。その結果、推定すべき異なる共分散は6/2 = 3個となる。一般に、n個の証券については、n(n − 1)/2個の異なる共分散とn個の分散を推定する必要がある。

図表3に示される共分散行列を考える。リターンはパーセンテージで表されており、表中の値はリターンの二乗（%²）として示されている。例えば、38%²および400%²はそれぞれ小数表記で0.0038および0.0400となる。パーセンテージと小数の適切な使用により、同一の結果が得られる。

  S&P 500 米国長期社債（US Long-Term Corporate Bonds） MSCI EAFE
S&P 500 400 45 189

式5を用いて分散項をまとめると、次のようになる。

σ2Rp=w12σ2R1+w22σ2R2+w32σ2R3+2w1w2CovR1,R2+2w1w3CovR1,R3+2w2w3CovR2,R3=0.502400+0.25281+0.252441+20.500.2545+20.500.25189+20.250.2538=100+5.0625+27.5625+11.25+47.25+4.75=195.875.

分散は195.875である。リターンの標準偏差は √195.875 = 14%である。まとめると、このポートフォリオの期待年率リターンは11.75%、標準偏差は14%である。

上記計算の最初の3項の合計（100 + 5.0625 + 27.5625）は132.625であり、これは個別分散がポートフォリオ分散に寄与する部分である。もし3資産のリターンが独立であれば、共分散は0となり、ポートフォリオの標準偏差は √132.625 = 11.52%となり、14%より低くなるため、より低リスクとなる。共分散が負であれば、132.625に負の値が加わるため、ポートフォリオ分散およびリスクはさらに低下する一方、期待リターンは変わらない。同じ期待リターンでリスクが低くなる。このリスク低減は分散投資効果（diversification benefit）であり、資産を組み合わせて保有することによるリスク低減の利益を意味する。分散投資効果は共分散が小さくなるほど大きくなる。この点は現代ポートフォリオ理論の重要な洞察である。この洞察は相関（correlation）の概念を用いるとより直感的に理解できる。

相関（Correlation）

2つの確率変数 Ri と Rj の相関は次のように定義される。

ρ(Ri,Rj) = Cov(Ri,Rj)/[σ(Ri)σ(Rj)].7
Alternative notations are Corr(Ri,Rj) and ρij.

上記の相関の定義は将来志向である。なぜなら、将来志向の共分散を将来志向の標準偏差の積で割っているためである。多くの場合、共分散は Cov(Ri, Rj) = ρ(Ri, Rj)σ(Ri)σ(Rj) の関係を用いて置き換えられる。共分散と同様に、相関係数は線形関係の尺度であるが、この定義では割り算を行うため、単位を持たない純粋な数となり、その取り得る値は+1から−1の範囲に制限される。

2つの変数が強い正の線形関係を持つ場合、相関は+1に近くなる。強い負の線形関係を持つ場合、相関は−1に近くなる。線形関係が弱い場合、相関は0に近くなる。この定義を用いると、共分散行列のデータのみから相関行列を構成することができる。図表4に相関行列を示す。

図表4 リターンの相関行列（Correlation Matrix of Returns）

  S&P 500 米国長期社債（US Long-Term Corporate Bonds） MSCI EAFE
S&P 500 1.00 0.25 0.45
米国長期社債（US long-term corporate bonds） 0.25 1.00 0.20
MSCI EAFE 0.45 0.20 1.00

例えば、図表3より長期社債とMSCI EAFEの共分散は38である。長期社債の標準偏差は √81 = 9%、MSCI EAFEの標準偏差は √441 = 21%である（図表3の対角項より）。したがって、相関係数 ρ(R long-term bonds, R EAFE) は 38 / [(9%)(21%)] = 0.201（約0.20）となる。また、S&P 500の自己相関は1である。これは共分散を標準偏差の二乗で割ることで得られる。

1.

2つの株式のリターン間の共分散を計算せよ。

解答を確認（Check Answer）

解答（Solution）

2つの株式リターンの相関は ρ(Ri, Rj) = Cov(Ri, Rj) / [σ(Ri) σ(Rj)] と表されるため、共分散は Cov(Ri, Rj) = ρ(Ri, Rj) σ(Ri) σ(Rj) となる。この2つの株式について、共分散は Cov(R1, R2) = ρ(R1, R2) σ(R1) σ(R2) = 0.30 × 6 × 15 = 27 である。
