8.05 | Option Time Value

モジュール8：オプションの価格付けと評価（Pricing and Valuation of Options）8.05｜オプションの時間価値（Option Time Value）

 Learning Outcome

The candidate should be able to:

  • オプションの行使価値（exercise value）、マネーネス（moneyness）、および時間価値（time value）を説明せよ。

オプションの行使価値は現在のペイオフを反映するが、オプション価値には満期までの残存時間に由来する追加的な要素も含まれる。ヨーロピアン・オプション（European options）は満期時にのみ行使可能であるが、満期前でも将来の期待ペイオフを反映した価格（コールの場合は cₜ、プットの場合は pₜ）で売買される。満期までの期間が長いほど、一定のボラティリティのもとで将来の原資産価格の分散が大きくなる可能性がある。同様に、一定の満期までの期間において、ボラティリティが上昇すると、同じ理由によりオプション価値は上昇する。これらの要因については後のレッスンでさらに検討する。

オプションの時間価値（time value）は、現在のオプション価格と現在のペイオフ（すなわち行使価値（exercise value））との差に等しい：

Call Option Time Value: ct−Max0, St−X1+r−T−t 3
or: ct=Max0, St−X1+r−T−t+Time Value
Put Option Time Value: pt−Max0, X1+r−T−t−St 4
or: pt=Max0, X1+r−T−t−St+Time Value

すなわち、現在のオプション価格は、行使価値と時間価値の合計に等しい。図表2（Exhibit 2）が示すように、オプションの時間価値は常に正であるが、満期時にはゼロに収束する。この過程は時間価値の減衰（time value decay）と呼ばれる。

原点からの直線は、時点tにおける縦線と交差し、その点は X(1 + r)^(−(T − t)) として示され、その後、時点Tの縦線上の点 X = F₀(T) に至る。上下に変動する曲線は原点から始まり時点tの縦線で終わる。上昇する直線と時点tでの曲線との間の差は行使価値（exercise value）として示されている。下部のグラフでは、時点tからTまでの区間が下に凸の曲線で表されている。

  Example 3Put Option Time Value

例1では、行使価格（X）が1,000ユーロの1年満期のプットオプションが、残存満期6か月時点でスポット価格（Sₜ）が950ユーロのとき、行使価値が45.04ユーロであることを示した。現在のプットオプション価格（pₜ）が50ユーロである場合、このオプションの時間価値はいくらか。

式4を用いて pₜ − Max（0, PV(X) − Sₜ）を求めよ。

pt−max0, X1+r−T−t−St
= EUR 4.96 (= EUR 50 − EUR 45.04)

 

  Question Set

原資産スポット価格に対するオプション価値（Option Value relative to Underlying Spot Price）

1.

行使価格（X）がフォワード価格（F₀(T)）と等しいロング・プットポジションの行使価値と、同一条件におけるショート・フォワードポジションの満期時ペイオフとの類似点および相違点を説明せよ。

解答を確認（Check Answer）

解答（Solution）

満期時における行使価格 X = F₀(T) のプットオプションの行使価値（Max（0, X − Sₜ））は、基礎資産価格が行使価格を下回る限り、ショート・フォワード契約の満期時ペイオフ（F₀(T) − Sₜ）と同一である。すなわち、F₀(T) = X の場合：

If X > ST: F0 (T) − ST = Max (0, X − ST )

Sₜ > X の場合、これらのペイオフは異なる。ショート・フォワードのペイオフでは売り手が買い手に支払いを行う必要があるのに対し、プットオプションの保有者はオプションを行使せずに満期を迎える。この比較は、時点t = 0で支払われるプットオプションのプレミアム（p₀）を考慮していない点に注意する。

2.

満期まで3か月のヨーロピアン・プットオプションについて考える。このオプションは追加的なキャッシュフローのない株式を原資産とし、行使価格（X）は50ポンド、無リスク金利（risk-free rate）は2%、現在の原資産価格（Sₜ）は55ポンドである。現在のプットオプション価格が5ポンドの場合、行使価値と時間価値を計算し、その結果を解釈せよ。

解答を確認（Check Answer）

解答（Solution）

オプションの価値は、行使価値と時間価値の合計から構成される。プットオプションの行使価値は Max（0, PV(X) − Sₜ）であり、以下のように計算できる。

Put Option Exercise Value=Max0, X1+r−T−t−St
Max(0, GBP 50(1.02)-0.25 − GBP 55)
= 0

行使価値はゼロである。これは現在の原資産価格が行使価格の現在価値を上回っているためである。時間価値はオプション価格と行使価値の差であり、満期時に有利な価格変動が生じる可能性を反映している。

Put Option Time Value=pt−Max0, X1+r−T−t−St
= GBP 5 (= GBP 5 − GBP 0)

時間価値は常に正であり、満期時（t = T）にはゼロに減少する。このプットオプションはアウト・オブ・ザ・マネーであるため、その価値はすべて時間価値から構成される。

3.

次の原資産価格と行使価格の関係を、それに対応するコールオプションと対応付けよ：

1. Sₜ = 60, X = 50 A. アット・ザ・マネー（at-the-money）のコールオプション
2. Sₜ = 50, X = 50 B. イン・ザ・マネー（in-the-money）のコールオプション
3. Sₜ = 40, X = 50 C. アウト・オブ・ザ・マネー（out-of-the-money）のコールオプション

解答を確認（Check Answer）

解答（Solution）

コールオプションは、Sₜ > X のときイン・ザ・マネー、Sₜ = X のときアット・ザ・マネー、Sₜ < X のときアウト・オブ・ザ・マネーである。したがって：

  1. Bが正しい。Sₜ > X であるため、イン・ザ・マネーのコールオプションである。
  2. Aが正しい。Sₜ = X = 50 であるため、アット・ザ・マネーのコールオプションである。
  3. Cが正しい。Sₜ < X であるため、アウト・オブ・ザ・マネーのコールオプションである。
4.

オプションのマネーネスが、原資産価格の変化に対してオプション価値がどのように変化するかに与える影響を説明せよ。

解答を確認（Check Answer）

解答（Solution）

オプションのマネーネスが高まるほど、その価値の原資産価格変化に対する感応度は高くなる。例えば、行使される可能性が非常に高いオプションは、原資産価格の変化に対してほぼ1対1で価値が変化する。一方、行使される可能性が低いオプションは、同じ価格変化に対して価値の変化は比較的小さい。

オプションの満期日に近づくにつれて、その時間価値（time value）がゼロへと減少していく過程。
