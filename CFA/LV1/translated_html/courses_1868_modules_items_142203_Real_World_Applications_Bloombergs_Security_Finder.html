Real World Applications: Bloomberg's Security Finder

モジュール1：市場の組織と構造（Market Organization and Structure）1.18｜実務応用：Bloombergのセキュリティファインダー（Security Finder）

Bloombergは、金融の専門家が業務全体で利用する主要な市場データプラットフォームの一つである。この任意セクションでは、Bloomberg上でのCFAプログラムのカリキュラムの実務的応用を探る。

{SECF} セキュリティファインダー（SECURITY FINDER）

BloombergのSECF機能は、グローバル金融市場における多様な証券タイプを表示することを可能にする。

 

「SECF <GO> を使用すると、Bloombergの証券および統計データのユニバースを検索でき、適切な金融商品をより迅速に見つけることができる。SECFは最大5,000件の検索結果を返し、資産クラス（asset class）、証券タイプ（security type）、キーワード（keyword）などの条件でフィルタリングできる。」―Bloomberg

 

この機能を試すには、BloombergのコマンドラインにSECFと入力し、Enterまたは<GO>を押す。アンバー色の「Category」ドロップダウンメニューをクリックして資産クラスを選択する。
