2.03 | Implied Return and Growth

モジュール2：金融における貨幣の時間価値（Time Value of Money in Finance） 2.03｜インプライド・リターンと成長（Implied Return and Growth）

 Learning Outcome

The candidate should be able to:

  • 現在価値（present value, PV）とキャッシュフローが与えられた場合に、債券（fixed-income）商品のインプライド・リターン（implied return）、および株式（equity）商品の要求収益率（required return）とインプライド成長率（implied growth）を計算し解釈する

レッスン1では、特定の債券および株式商品において、現在と将来のキャッシュフローの間の貨幣の時間価値のトレードオフを扱った。市場参加者はしばしば、現在価値と将来価値の両方が既知である状況に直面する。この場合、現在価格と将来キャッシュフローの特性から、インプライド・リターンや成長率を求めることが可能となる。この意味で、インプライド成長率やリターンを求めることは、資産価格に織り込まれている市場の期待を把握することにつながる。

債券（fixed-income instruments）のインプライド・リターン

債券は、契約に基づく利息および元本のキャッシュフローによって特徴付けられる。現在価値（または価格）が観測され、将来のキャッシュフローが約束通りに支払われると仮定すると、割引率（r）または満期利回り（yield-to-maturity, YTM）は、このキャッシュフローパターンに対するインプライド・リターンの指標となる。

ディスカウント債の場合、投資家は満期時に単一の元本キャッシュフロー（FV）を受け取り、（FV − PV）がインプライド・リターンを表す（図表7）。

 Exhibit 7Discount Bond Implied Return

式5を変形することで、商品の存続期間（t期間）にわたる各期のインプライド・リターンを次のように求めることができる：

r=FVtPVt−1 =FVtPV1t−1.18

  Example 8Discount Bond Implied Return

例1で示したように、2016年にドイツの10年国債は、元本EUR100あたりEUR100.50の価格と年率YTM −0.05%で発行された。すなわち、この債券は投資家に負のリターンをもたらすように価格付けされていた。その6年後、これらの債券は元本EUR100あたりEUR95.72で取引されていた。

1.

この債券を6年間保有した初期投資家のインプライド・リターンはいくらか。

解答を確認

解答

−0.81%

式18を用い、PV = 100.5、FV = 95.72、t = 6として、投資家の年率リターン（r）を求める：

r= −0.81%=95.72100.516−1.

発行時にディスカウント債を購入し、10年後にEUR100を受け取る投資家は、発行時のYTM −0.05%に等しいインプライド・リターンを期待する。しかし、最初の6年間でのEUR4.78の価格下落は、年率−0.81%のリターンに相当し、初期のYTM −0.05%を下回る。この負のリターンは、インフレ上昇に伴うディスカウント債の価格下落と整合的である。

2.

EUR95.72でこのディスカウント債を購入し、残り4年間保有した場合の期待リターンはいくらか。

解答を確認

解答

1.10%

この場合、式18において現在価格95.72がPV、元本100がFV、t = 4となる：

r= 1.10%=10095.7214−1.

r = 1.10%は、例1で求めた残存4年間のYTMと一致する。取引コストがないと仮定すれば、2人の投資家の累積リターンは初期のYTM −0.05%と一致する：

−0.05%=[1−0.00816×1+0.0114]110− 1.

これらの関係は次の図で要約される：

ディスカウント債とは異なり、定期利息を支払う債券は満期までの期間中にキャッシュフローを持つ。すべてのキャッシュフローに適用される単一の割引率（または内部収益率）がYTMであり、タイミングに関わらずすべてのキャッシュフローに対する市場のインプライド割引率を表す（図表8）。

 Exhibit 8Coupon Bond Implied Return

YTMは、投資家が満期までにすべてのキャッシュフローを受け取り、受け取ったキャッシュフローを同じYTMで再投資することを前提としている。クーポン債では定期的な利息のみが対象となるが、住宅ローンのような定額支払い型商品では、利息と元本返済の両方が同じ利率で再投資されると仮定される。他の内部収益率と同様に、YTMは解析的に解くことはできず、スプレッドシートや計算機を用いた反復計算によって求められる。これは式19においてrを解くことに相当する：

PV(Coupon Bond) = PMT1 / (1 + r)1 + PMT2 / (1 + r)2 + … + (PMTN + FVN ) / (1 + r) N ,19

ここでFVは債券の元本、Nは満期までの期間数である。

Microsoft ExcelまたはGoogle SheetsのYIELD関数は、定期利息と満期時の元本返済を伴う債券のYTMを計算するために使用できる：

= YIELD (settlement, maturity, rate, pr, redemption, frequency, [basis])

ここで：

  • settlement = DATE関数で入力された受渡日、
  • maturity = DATE関数で入力された満期日、
  • rate = 半年（または各期）のクーポン、
  • pr = 額面100あたりの価格、
  • redemption = 満期時の将来価値、
  • frequency = 年間のクーポン支払回数、
  • [basis] = 日数計算方式（通常、米国債では0：30/360方式）。

例9では、定期利息を伴う債券のインプライド・リターンを示す。

  Example 9Greek Coupon Bond Implied Return

例2より、2020年に発行された年率2.00%クーポンの7年物ギリシャ国債は、2年後に元本EUR100あたりEUR93.091であった。

1.

定期利息を元のYTM 2.00%で再投資できる投資家の2年間のインプライド・リターンはいくらか。

解答を確認

解答

−1.445%

ディスカウント債と同様に式18を用いるが、まず2年後の将来価値（FV）を、価格93.091および再投資されたすべてのキャッシュフローを含めて計算する必要がある：

FV2 = PMT 1(1 + r) + PMT2 + PV2
FV2 = 97.13 = 2 × (1.02) + 2 + 93.091
r= −1.445%=97.1310012−1.

この年率リターンが負であったのは、EUR6.91（=100 − 93.091）のキャピタルロスが、定期利息とその再投資収益EUR4.04（=2 × (1.02) + 2）を上回ったためである。2022年に債券価格がEUR93.091に下落したのは、期間中のユーロ圏のインフレ上昇の結果である。

2022年にEUR93.091でこの年率2%クーポン債を購入した投資家については、r（またはYTM）を3.532%と求めたことを思い出そう。この残存5年間のYTMの計算は、すべてのキャッシュフローが満期まで3.532%で再投資できると仮定している点に注意する。定期利息をどの利率で再投資できるかは、クーポン債のインプライド・リターンの計算において重要である。YTMは、債券が満期まで保有されることを前提として計算される。

株式（Equity Instruments）、インプライド・リターンおよびインプライド成長

株式投資の現在価値の計算で述べたように、株価は要求収益率だけでなくキャッシュフローの成長も反映する。式14の一定配当成長の仮定から出発すると、この式は次のように変形できる：

r−g= Dt1+gPVt=Dt+1PVt.20

式20の左辺は要求収益率と一定成長率の差を表し、右辺は次期の予想配当に基づく配当利回り（dividend yield）である。したがって、予想配当利回りと成長率が与えられた場合の株式のインプライド・リターンは式21で次のように表される：

r= Dt1+gPVt+g=Dt+1PVt+g.21

要するに、配当が無期限に一定成長すると仮定すると、株式のインプライド・リターンは、予想配当利回りと一定成長率の和に等しい。

逆に、株式のインプライド成長率を求めたい場合、この関係は式22で表される：

g= r*PVt−DtPVt+ Dt=r− Dt+1PVt.22

次期の予想配当が既知であれば、要求収益率から予想配当利回りを差し引くことでインプライド成長率を計算できる。

  Example 10Implied Return and Growth for a Stock

Coca-Cola Companyの株価はUSD63.00であり、来年の1株当たり予想配当はUSD1.76である。

1.

アナリストが配当が年率4%で無期限に成長すると予想する場合、投資家の要求収益率はいくらか。

解答を確認

解答

6.79%

式21を用いると、

r=1.7663+0.04=0.0679.

Coca-Cola株の要求収益率は、現在価格、予想配当、および配当成長率に基づき6.79%となる。将来の株式リターンに関する投資家の期待は、現在価格、将来のキャッシュフロー、およびその成長率の組み合わせから推測される。

要求収益率を推定する代わりに、投資家がインプライド配当成長率を求めたい場合もある。この場合、設問2と同様に将来の株式リターンを仮定する必要がある。

2.

アナリストがCoca-Cola株の投資家は7%のリターンを期待すべきと考える場合、インプライド配当成長率はいくらか。

解答を確認

解答

4.21%

式22を用いると、

g=0.07− 1.7663 = 0.0421.

Coca-Cola株のインプライド配当成長率は、期待リターン、価格、および予想配当に基づき4.21%となる。本設問では設問1よりも高い期待リターンを仮定しているため、USD63.00の株価を正当化するためには、より高いインプライド成長率が必要となる。

株価を通貨ベースで直接比較する代わりに、一般的には株価収益率（price-to-earnings ratio）を用いて比較する。

  Price-to-Earnings Ratio

株価収益率は、既知の価値ドライバーである1株当たり利益（earnings per share）および通貨の違いを調整することで比較可能性を高める相対評価指標である。不動産価格を1平方メートル当たりの価格で表すのと類似している。

株価収益率が20の株式は、株価が1株当たり利益の20倍であることを意味し、投資家が1株当たり利益の20倍を支払う意思があることを示す。これは、株価収益率が10の株式よりも割高である。

株価収益率は個別株だけでなく、S&P 500、FTSE 100、日経225などの株価指数の評価指標としても用いられる。この場合、指数の値を構成銘柄の1株当たり利益の加重平均で割る。本カリキュラムの後半で詳しく扱うが、株価（PV）と将来キャッシュフローの関係から、いくつかの有用な考察が可能である。まず式14を思い出そう：

PVt= Dt1+gr−g.

両辺を期tの1株当たり利益E_tで割ると次のようになる：

PVtEt= DtEt×1+gr− g.23

式23の左辺は株価収益率であり、右辺の分子の第1項は配当性向（dividend payout ratio）と呼ばれる、利益のうち株主に配当として分配される割合である。

株価収益率と配当性向が与えられれば、要求収益率またはインプライド成長率（もう一方を仮定すれば）を求めることができる。要求収益率は将来の投資家リターン期待の理解に有用であり、インプライド成長率は企業の予想成長率や過去の成長率との比較に有用である。例えば、インプライド成長率が10%であるにもかかわらず、アナリストが企業の成長率を5%と見積もる場合、その株式は割高と判断される可能性がある。

実務では、フォワード株価収益率、すなわち株価を次期（t+1）の予想1株当たり利益で割った指標が一般的に用いられる。この場合、先の式は次のように簡略化される：

PVtEt+1= Dt+1Et+1r− g.24

式24より、フォワード株価収益率は、期待配当性向および期待成長率が高いほど高くなり、要求収益率が高いほど低くなることが分かる。

  Example 11Implied Return and Growth from Price to Earnings Ratio

1.

Coca-Cola株のフォワード株価収益率が28、予想配当性向が70%、配当成長率が年率4%とする。このとき要求収益率はいくらか。

解答を確認

解答

6.50%

式24を用いると、

28= 0.7r−0.04.

この式をrについて解くと、Coca-Colaの要求収益率は6.50%となる。

上記の結果から、Coca-Colaの要求収益率を6.50%と仮定すれば、インプライド成長率は4%となることは明らかであり、この結果は設問2で確認されている。

2.

Coca-Cola株のフォワード株価収益率が28、予想配当性向が70%、要求収益率が6.50%とする。このときインプライド成長率はいくらか。

解答を確認

解答

4.00%

式24を用いると、

28= 0.700.065− g.

この式をgについて解くと、Coca-Colaのインプライド成長率は4.00%となる。具体的には、Coca-Colaのリターンはその配当成長率よりも2.5%高いと期待されている（すなわち、6.50% − 4.00%）。

前述の通り、同様の原則は株価指数の要求収益率や成長率の理解にも適用される。

3.

ある株価指数がフォワード株価収益率（price-to-earnings ratio）19で取引されているとする。指数の予想配当性向（dividend payout ratio）が60%、投資家が期待する指数のリターンが8%である場合、この指数のインプライド一定成長率はいくらか。

解答を確認

解答

4.84%

式24を用いると、

19= 0.600.08− g.

gを求めると、投資家は将来の配当成長率を4.84%と期待していることが分かる。あるいは、成長率を4.84%と仮定してrを求めれば、要求収益率8%が得られる。このように、フォワード株価収益率19と配当性向60%は、8%のリターンと4.84%の成長の期待を反映している。具体的には、要求収益率はインプライド成長率を3.16%上回っている（すなわち、8% − 4.84%）。

これまでの結果から重要な点は、株価は単なる価格であれ株価収益率であれ、将来のリターンと成長に関する期待の組み合わせを反映しているということである。リターンと成長の期待は、rとgの差によって結び付けられている。例えば、株価収益率28のCoca-Colaの例は、投資家が要求収益率が成長率より2.5%高いと考えている状況を示している。

4.

Coca-Cola株がフォワード株価収益率28、予想配当性向70%で取引されているとする。アナリストがCoca-Cola株の期待リターンを9%、配当成長率を年率4.50%と考えている場合、この株式に対する投資戦略を提案せよ。

解答を確認

解答

Coca-Cola株をショート（short position）する。

上記のパラメータを式24で評価すると、不等式が成立することが分かる。

28 > 0.700.09− 0.045 = 15.56.

Coca-Colaのフォワード株価収益率28は、式から求められる15.56よりも大幅に高い。これは、キャッシュフローの成長およびリターンに関する投資家の期待が、現在の株価収益率と整合していないことを意味する。具体的には、現在の株価収益率がファンダメンタルズから示唆される水準を大きく上回っているため、株価は下落する可能性があると考え、ショートポジションを検討すべきである。例の設問1および2の結果が示すように、配当性向70%のもとで株価収益率28を正当化するには、r − g = 2.50%である必要がある。

株価を1株当たり利益（earnings per share）で割った比率。

期間中の利益に対する配当支払いの比率。

将来の1株当たり利益（EPS）の予測に基づいて計算される株価収益率；現在の株価を翌期の予想利益で割ったもの。
