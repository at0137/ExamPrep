6.03 | MTM Valuation: Forwards versus Futures

モジュール6：先物契約（futures contracts）の価格決定（pricing）と評価（valuation） 6.03｜MTM評価（mark-to-market valuation）：フォワード対先物（forwards versus futures）

 Learning Outcome

The candidate should be able to:

  • フォワード契約（forward contracts）と先物契約（futures contracts）の価値（value）と価格（price）を比較する。

例1および例2は、契約開始時（inception）におけるフォワード価格と先物価格の類似性を示している。時間の経過とともに、異なるフォワード契約および先物契約の特徴により、同一の原資産（underlying assets）およびそれ以外の条件が同一であっても、契約のマーク・トゥ・マーケット値（MTM values）は異なるものとなる。例3では、損益の毎日決済（daily settlement）がこの差異をどのように生み出すかを示している。

  Example 3Procam Forward versus Futures Pricing and Valuation

先の例を拡張して、フォワードと先物の価格および評価を比較する。いずれの場合も、Procam Investments は、91日後に1オンスあたり$1,778.76の価格（f₀[T] = F₀[T]）で100オンスの金を購入する現金決済型フォワード・コミットメント（cash-settled forward commitment）を締結する。無リスク金利は2%、金の保管コストはゼロとする。

フォワード契約（Forward Contract）

フォワード価格は F₀(T) = 1オンスあたり$1,778.76である。契約開始時には現金の受け渡しや預託はなく、契約価値 V₀(T) はゼロである。

Expand / Collapse Extended Description

Procam Investments と金融仲介機関（financial intermediary）との間のフォワード契約（100オンスの金）。t = 0 では、フォワード価格 F₀(T) で契約が締結される。t = T では、金融仲介機関がスポット価格 Sₜ で100オンスの金をProcamに提供し、同時にProcamはフォワード価格 F₀(T) で100オンスを金融仲介機関に提供する。

時間の経過に伴い、フォワード価格 F₀(T) は変化せず、任意の時点におけるMTM値 Vₜ(T) は、現在のスポット価格 Sₜ とフォワード価格の現在価値 PVₜ[F₀(T)] の差に等しい（Procam＝フォワード買い手の視点）：

Vt(T) = St − F0(T)(1 + r)–(T – t).

例えば、71日が経過し、満期まで残り20日（T − t = 20/365 または 0.0548）とする。金のスポット価格 Sₜ が契約開始時から$50下落して1オンスあたり$1,720となった場合、Vₜ(T) は次のように求められる：

Vt (T) = $1,720 − $1,778.76(1.02)–0.0548
= –$56.83 per ounce, or a $5,683 MTM loss (= –$56.83 × 100 ounces).

フォワード契約の条件では、MTM金額の決済は満期まで行われない。このような契約価値のリセットが日々行われないため、先物契約が同様のMTM値に達する可能性は低い。

先物契約（Futures Contract）

先物価格 f₀(T) は1オンスあたり$1,778.76である。先物取引所（futures exchange）の毎日決済ルールに従い、契約の買い手と売り手は、金1契約（100オンス）あたり$4,950の初期証拠金（initial margin）を預託し、$4,500の維持証拠金（maintenance margin）を維持する必要がある。証拠金残高が$4,500を下回る場合、カウンターパーティ（counterparty）はマージンコール（margin call）を受け、直ちに$4,950まで補填しなければならない。

Expand / Collapse Extended Description

Procam Investments と金融仲介機関との間の先物契約（100オンスの金）。取引開始時 t = 0 において、Procamは$4,950の初期証拠金を金融仲介機関に提供し、金融仲介機関は同額をロンドン金属取引所（London Metal Exchange）に差し入れる。

取引初日を考える。このとき、金のスポット価格 S₀ は1オンスあたり$1,770、先物価格 f₀(T) は1オンスあたり$1,778.76である。

  • 初日の取引で、金先物価格 f₁(T) が$5下落して$1,773.76となり、スポット価格 S₁ が無裁定価格として$1,765.12（= $1,773.76(1.02) − 90/365）で終了したと仮定する。
  • Procamは$500のMTM損失（= 1オンスあたり$5 × 100オンス）を計上し、この金額が証拠金口座（margin account）から差し引かれ、残高は$4,450となる。
  • Procamの先物契約のMTM価値は、終値である$1,773.76においてゼロにリセットされる。
  • Procamの証拠金口座残高は維持証拠金$4,500を下回ったため、初期証拠金$4,950に戻すために$500を追加で預託しなければならない。

時間の経過に伴う先物とフォワードの価格および価値（Futures versus Forward Price and Value over Time）

同一の条件を用いて、2営業日にわたる先物とフォワードの価格および価値を比較する。2日目の取引は、1日目の終値（スポット価格および先物価格）から開始すると仮定する。以下の表はその比較を示す：

  Beginning of Day 2 Trading
契約タイプ（Contract Type） 契約価格（Contract Price） 契約のMTM（Contract MTM） 実現MTM（Realized MTM） 証拠金預託（Margin Deposit）
フォワード（Forward） F₀(T) = $177,876 –$498 $0 $0
先物（Futures） f₁(T) = $177,376 $0 –$500 $4,950

フォワード契約のMTM価値 Vₜ(T) は、現在のスポット価格 S₁ = $1,765.12 と、元のフォワード価格の現在価値 PVₜ[F₀(T)] の差に等しい。ここでは満期まで残り90日（T − t = 90/365 または 0.24657）である：

Vt(T) = $1,765.12 − $1,778.76(1.02)–0.24657
= $4.98 per ounce.

例をさらに拡張し、前日の終値時点である3日目の開始時点を考える。2日目に金先物価格 f₂(T) が1オンスあたり$4下落して$1,769.76となり、無裁定価格に対応するスポット価格 S₂ が$1,761.24（= $1,769.76(1.02) − 89/365）まで下落したと仮定する。以下の表は3日目の状況の要約である：

  Beginning of Day 3 Trading
契約タイプ（Contract Type） 契約価格（Contract Price） 契約のMTM（Contract MTM） 実現MTM（Realized MTM） 証拠金預託（Margin Deposit）
フォワード（Forward） F₀(T) = $177,876 –$895 $0 $0
先物（Futures） f₂(T) = $176,976 $0 –$400 $4,550

フォワード契約のMTM価値 Vₜ(T) は、現在のスポット価格 S₂ = $1,761.24 と、元のフォワード価格の現在価値 PVₜ[F₀(T)] の差に等しい。ここでは満期まで残り89日（T − t = 89/365 または 0.24384）である：

Vt(T) = $1,761.24 − $1,778.76(1.02)–0.24384
= $8.95 per ounce.

例3は、時間の経過に伴うフォワード契約と先物契約の価格および価値の主な違いを示している。フォワード契約価格 F₀(T) は、契約満期まで固定されたままである。フォワード契約のMTM価値の変化は、現在のスポット価格 Sₜ とフォワード価格の現在価値 PVₜ[F₀(T)] の差によって捉えられる。このフォワード契約のMTMは満期まで決済されないため、契約開始から満期または終了まで現金のやり取りがないことに起因して、時間の経過とともにカウンターパーティ信用リスク（counterparty credit risk）が生じる。一方、先物契約の価格は市場の変化に応じて日々変動する。毎日決済の仕組みにより、先物のMTMはゼロにリセットされ、差額は変動証拠金（variation margin）として決済されるため、カウンターパーティ信用リスクは低減される。先物契約における累積実現MTM損益（cumulative realized MTM gain or loss）は、同等のフォワード契約とほぼ同じである。これらの違いについては、次のレッスンで市場参照金利（market reference rates）に基づくフォワード契約および先物契約を扱う前に、さらに詳しく検討する。
